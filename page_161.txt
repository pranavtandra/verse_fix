evam api prabhedamātraṃ darśitaṃ na vikṣepasvarūpam iti pṛcchati - sa eṣa 
ṣaḍvidho vikṣepaḥ kiṃ lakṣaṇo veditavya ityata āha -
	vyutthānaṃ viṣaye sāraḥ /
iti vistaraḥ /
dhīmatā iti dhaīmān bodhisattvaḥ /
tatra vyutthānaṃ samādhitaḥ pañcabhir vijñānakāyair iti /
yadā hi samāhitasya punaḥ pañcānāṃ cakṣurādivijñānānām anyatamad 
ekam utpadyate tadā samādhito vyutthito bhavati /
ayañca prakṛtivikṣepaḥ bahirmukhapravṛttatvāt /
pañcānāṃ vijñānakāyānāṃ vikṣepātmakatvāt pañcabhir 
vijñānakāyaiḥ samādhito vyutthānaṃ prakṛtivikṣepa ucyate /
viṣaye visāro bahirdhāvikṣepa iti /
yadā samāhitasya samādhyālambanād akāmakasyālambanāntare 
manovijñānaṃ pravartate tadālambanāntarasañcārād bahirdhāvikṣepa 
ityeke /
samāhitasya samādhyālambanaṃ hitvā manovijñānasya 
viṣayāntarasañcāro bahirdhā vikṣepa ityapare /
samādher āsvānadāṃ layauddhatyaṃ cādhyātmavikṣepa iti /
yadi samāhitastameva samādhiṃ tṛṣṇādinā kleśenāsvādayati tataḥ 
śuddhakāt samādher vikṣepo bhavati kliṣṭasamādhipratyupasthānāt /
layauddhatyañca śamathavipaśyanāsamavāhinaḥ samādher vikṣepo bhavati /
asamāhitattvāc cāsvādānālayauddhatyānām 
āsvādanādayo'dhyātmavikṣepa ity ucyante /
sambhāvanābhisandhir nimittavikṣepaḥ /
dhyāyyayaṃ mahātmetyevaṃ sambhāvanātmanītyanena nimittena samādhau 
prayuktasyāśayavilayatvālabdhād alabdhād vā kuśalapakṣād vikṣepo 
nimittavikṣepa ityucyate /
atraiva kāraṇam āha - tan nimittaṃ kṛtvā prayogād iti /
jñātukāmatayā taṃ sambhāvanābhisandhiḥ nimittaṃ kṛtveti 
sambhāvanātmanīti /
samādhiprayuktatvād iti /
sāha:nkaramanaskāratā dauṣṭhulyavikṣepa iti /
āśayena samādhiguṇārthino'pi sāha:nkaramanaskāro'smimānasam 
prayukto'ham asmi dhyāyīti /
tataścātmotkarṣamanyatrāvajñāṃ karotīti /
kasmād eṣa dauṣṭhulyamanaskāraḥ ?
ityata āha - dauṣṭhulyavaśenāsmimānasamudācārād iti /
etad uktaṃ bhavati - dauṣṭhulyaprabhāvito vikṣepo dauṣṭhulyavikṣepa iti /
kiṃ punar dauṣṭhulyaṃ ?
asmimānavāsanāparipuṣṭistayā hyasmimānaḥ samudācaratīti 
tadvaśenāsmimānasamudācārāt /
satkāyadṛṣṭir dauṣṭhulyam ityanye /
hīnacittatvaṃ manasikāravikṣepa iti /
aniyataśrāvakādigotrasya
