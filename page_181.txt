yo'yam anukrānto vikalpaprabhedastasmād ayam aparaḥ saptavidho 
vikalpadvayāntaḥ /
vikalpaprabhedapradarśanārtham āha - tadyathā bhāve'pi vikalpo'ntaḥ 
abhāve'pi vikalpo'nta iti /
katham ityato bravīti - pudgalo'sti yasya vināśāya śūnyatā nairātmyam 
api vā nāstīti vikalpanāt /
yadi śūnyatayā dharmān prakṛtiśūnyatvān na śūnyān karoti tena tarhi 
pudgalo'sti yasya vināśāya śūnyatā /
nairātmyam api vā nāsti pudgalābhāvāt /
na hi pratipakṣo nirvipakṣo'sti /
tasmād avaśyaṃ pudgalāstitvaṃ vā nairātmyaṃ bhāvo vā pratipattavya iti 
/
tadbhāvābhāve vikalpo'ntaḥ /
etasya bhāvābhāvavikalpadvayāntasya parivarjanārtham iyaṃ madhyamā 
pratipat /
na khalu punaḥ kāśyapa pudgalavināśāya śūnyatā /
api tu śūnyataiva śūnyeti vistareṇa sūtrapāṭhaḥ /
tatra na sa:nkliṣṭa pudugalavināśāya na oyavadānpudgalothpādāya 
śūnyatā /senādivatprajñaptisatvāt pudgalasyotpādavināśābhāvād api 
tu śūnyataiva śūnyā na tu pudgalena satkāyadṛṣṭisamāropitena 
tasyātyantam asattvāt /
evañca kṛtvā pudgaladharmātmanā na bhāvaḥ śūunyatā tad rahitatvāt /
pariniṣpannena nābhāvaḥ tac ca śūnyatāyāḥ pariniṣpannātmakatvaṃ 
pratipādayann āha - pūrvāntaśūnyatā aparāntaśūnyatā 
pratyutpannaśūnyateti kālatraye'pi tathaivāstitvapratipādanāt /
athavā pṛthagjanāśaikṣaśaikṣāvasthāsu tathaivāstitvapratipādanāt /
tatra pṛthagjanāvasthā pūrvāntaḥ aśaikṣāvasthā'parāntaḥ 
śaikṣāvasthā pratyutpanna iti /
yasmāc caivaṃ bhāvābhāvasvabhāvaśūnyatā tasmād āha - śūnyatāṃ 
kāśyapa pratisarateti nirvikalpajñānaviṣayatvāt na pudgalam 
ityabhidhānamātratvād iti /
