lakṣaṇatattvānantaram aviparyāsatattvaṃ vaktavyaṃ tad anantaram 
uddiṣṭatvād ityata āha - aviparyāsatattvam iti vistaraḥ /
anityaduḥkhaśūnyā'nātmato hi 
nityasukhaśuddhātmaviparyāsapratipakṣatvād aviparyāsatattvam ity ucyate 
/
mūlatattve tad anyatattvāntarbhāvaḥ pūrvaṃ pratijñāta ityataḥ pṛcchati 
- mūlatattve katham iti vistaraḥ /
anityāditeti /
anityādayo ye'nityaduḥkhaśūnyā'nātmaprakārās te'nityādayaḥ /
tad bhāvo'nityāditā /
mūlatattve'viparyāsatattvasyāntarbhāvapradarśanārtham āha -
	asadartho hy anityārtha
iti vistaraḥ /
trividho hy anityārthaḥ - asadarthaḥ utpādavyārthaḥ samalāmalatārthaś 
ca /
yaḥ sarvadā tathāsti sa nityaḥ /
tad viparīto'nityaḥ /
teṣu yathākramam iti /
tatra sarvadā'sadarthā'nityārthena parikalpitasvabhāvo'nityaḥ /
asannityam anityam iti niruttinyāyena veditavyamityanye /
pratikṣaṇamutpādavyahalakṣaṇenā'nityārthenā'nityaḥ paratantrasvabhāvaḥ 
/
tasya pratyayotpādyatvād vināśe cā'hetukatvāt /
samalāmalatā'rthalakṣaṇenāgantukenā'nityārthenā'nityo'vikāradharmo'pi 
pariniṣpannasvabhāva iti /
	duḥkhamādānalakṣmākhyaṃ
iti vistaraḥ /
duḥkham api trividhaṃ - upādānaduḥkhaṃ lakṣaṇaduḥkhaṃ sambandhaduḥkhañca 
/
tatra kalpitasvabhāva upādānato duḥkhaṃ /
aparijñātenehā'mutra vā duḥkham upādīyata iti duḥkhaṃ /
yathā ca tenā'parijñātena duḥkham upādīyate tat pradarśayann āha -
