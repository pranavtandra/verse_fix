satyārthānantaraṃ yānatrayārthaṃ pratipādyata ityata āha -
	guṇadoṣā'vikalpena jñānena
iti vistaraḥ /
jñānaśabda ubhayatra sambadhyate /
guṇadoṣajñānenā'vikalpajñānena ceti /
athavā'ntare vikalpaśabdo lupto draṣṭavyaḥ /
tatra guṇadoṣajñānena parataḥ svayañceti ata evāha - vṛttau yathāyogam 
iti /
tatra nirvāṇasaṃsārayor guṇadoṣajñāneneti /
nirvāṇaśabdenātra mārganirodhau sa:ngṛhītau yathā praśāntaśabdaḥ /
tatra nirodhe śāntādyākāraiḥ mārge nairyāṇikādyākārair jñānaṃ 
nirvāṇe guṇajñānaṃ /
saṃsāro'pi duḥkhasamudayātmakaḥ /
tatrā'py anityaduḥkhādyākāraiḥ samudayapratyayādyākāraiś ca jñānaṃ 
saṃsāre doṣajñānaṃ /
parataḥ śrutveti buddhabodhisattvādibhyo nirvāṇasaṃsārayor 
guṇānyathokān doṣāṃś ca śrutveti /
niryāṇārtheneti saṃsāranirgamārthena /
śrāvakayānam iti paradghoṣasaṃniśrayeṇa niryāṇāt śrāvakāyanm iti 
pratyekabuddhayānād viśeṣaṃ darśayati /
tenaiveti nirvāṇasaṃsārayor guṇadoṣajñānena svayam aśrutvā parato 
buddhabodhasattvebhyo niryāṇārthena pratyekabuddhayānaṃ /
svayam aśrutveti śrāvakayānād viśeṣaṇaṃ /
avikalpena jñāneneti saṃsāranirvāṇayor anabhilāpyadharmatāprativedhād 
avikalpaṃ jñānaṃ /
svayam eva parato'śrutvā niryāṇārthena mahāyānaṃ /
atra hi jñāneneti pratyekabuddhayānād viśeṣaḥ /
anye tvāhuḥ - pudgalanairātmyaviṣayeṇa guṇadoṣā'vikalpena jñānena 
parataḥ śrutvā niryāṇaṃ śrāvakayānaṃ /
śravaṇanirapekṣasya svayaṃ niryāṇaṃ pratyekabuddhayānaṃ /
dharmanairātmyaviṣayeṇā'pratiṣṭhitanirvāṇaprāpakenā'vikalpena jñānena 
svayaṃ niryāṇaṃ mahāyānam ucyate /
etad vyākhyānaṃ bhāṣyeṇa virudhyate /
evaṃ yānatrayakuśalasya yogamātradarśanād ātmā yogīti grāho 
vyāvartate //
