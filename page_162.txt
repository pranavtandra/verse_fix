mahāyānamanasikāraprayuktasya samsaraduḥkhabhīrutayā 
sattvavipratipānatvenākalyāṇamitraniṣevaṇena ca tac 
chrāvakapratyekabuddhacittam utpadyate /
taccittaṃ hīnaṃ svārthamātraparāyaṇatvāt /
tacchrāvakapratyekabuddhacittaṃ mahāyānamanasikāravikṣepakatvān 
manasikāravikṣepaḥ kathyate /
atraiva kāraṇam āha - hīnayāna manasikārasamudācārād iti /
hīnayānaṃ śrāvakapratyekabuddhayānaṃ /
ātmaparinirvāṇādhikārapravṛttatvān manasikārasamudācāro 
mahāyānamanasikārebhyo vikṣipatīti hīnaṃ cittaṃ manasikāravikṣepa ity 
ucyate /
evam iyaṃ bodhisattvasya ṣaḍvidhavikṣepadoṣarahitā śamathābhāvanā 
veditavyā //

	{ aviparyāsapariṇatā }
evam avikṣiptena yathā'viparyāsapariṇatā'nudharmapratipattir 
bhāvayitavyā tathocyate /
aviparyāsātmanā pravṛttā'viparyāsapariṇatā /
tatra yad yathā vyavasthitaṃ vastu tathā tatra jñānam aviparyāsaḥ /
tad vastuta punar na jñāyate katividhe vastunītyata āha - 
tatrāviparyāso daśavidhe vastuni veditavya iti /
aviparyāsavastu pradarśanārthaṃ bravīti - yaduta
