vā caritāni dharmacaritāni /
tāni punar na jñāyanta ityata āha - katamad daśadhā dharmacaritam iti /
ata āha -
	lekhanā pūjanā dānaṃ
iti vistaraḥ /
tatra mahāyānasya svayaṃ pareṇa vā lekhanaṃ cirasthityarthaṃ 
śravaṇāvācanārthaṃ vā /
pūjanaṃ puṣpadhūpagandhādibhiḥ /
mahāyānasyeti prakṛtaṃ /
mahāyānapraviṣṭasya likhitapustakādīnāṃ parebhyaḥ satkṛtya dānaṃ /
pareṇa vācyamānasya satkṛtya śravaṇaṃ /
mahatā gauraveṇa svayaṃ vācanaṃ /
udgrahaṇaṃ mahatā prasādenātmasātkaraṇaṃ /
mahatā yatnena yogyebhyo granthastyārthasya vā āviṣkaraṇaṃ prakāśanaṃ /
svādhyāyanam uddhatasya punaḥ punar abhyasanaṃ /
cintanaṃ yuktidvāreṇa tadarthanirūpaṇaṃ /
bhāvamam abhyasanaṃ samāhitayā prajñayā /
ca śabdaḥ samuccaryārthaḥ /
tatra lekhanā pūjanā dānaṃ śravaṇam ityebhiś caturbhiḥ padaiḥ kāyakarma 
pradarśitaṃ /
vācanam udgrahaṇaṃ prakāśanā svādhyāyanam ityebhiś caturbhiḥ padair 
vākkarma pradarśitaṃ /
cintanā bhāvanā cetyābhyām api manaḥ karma pradarśitaṃ /
	ameyapuṇyaskandhaṃ hi caritaṃ tad daśātmakaṃ /
yadet daśaprakāraṃ dharmacaritam uktaṃ tata ekaikasmād dharmacaritādyaḥ 
puṇyaskandhodbhavo na sa punaḥ svayaṃ pareṇa vā pramātuṃ śakyate 
atibahutvād iti /kasmāt punar dānaśīladhyānādidharmanirdeśe tulye 
mahāyāna eva dharmacaritam atyarthaṃ mahāphalaṃ deśyate sūtreṣu na punaḥ 
śrāvakayāne ?
sūtreṣu dharmacaritaṃ mahāphalaṃ deśyata iti sambadhyate /
dvābhyāṃ kāraṇābhyām iti mahāyāne dharmacaritam atyarthaṃ mahāpalaṃ 
deśyata iti sambadhyate /
katam ābhyām ityata āha -
	viśeṣād akṣayatvāc ca
iti /
etad api na jñāyata ityataḥ pṛcchati - kathaṃ viśeṣāt kahtma akṣayatvād 
ityata āha -
	parānugrahato'śamāt /
iti /
mahāyāne hi bodhisattvānāṃ dānādipratipattiḥ 
sarvasattvārthakaraṇaikaṃ deśyate /
śrāvakayāne tvātmārthapradhānetyataḥ śrāvakāyānān mahāyānaṃ 
parānugrahapravṛttatvād viśiṣyate /
evaṃ tāvad hetusamākhyānataḥ śrāvakayānān mahāyānasya viśiṣṭatvaṃ /
acintyapuṇyajñānasambhāraphalasya trikāyasaṃgṛhītasya yāval lokas 
tāvat sattvahitakriyāpravṛttasya nirupadhiśeṣe'pi nirvāṇadhātāv 
anuparatasyāsravaparāvṛttilakṣaṇasya dyotanād akṣayatvaṃ /
na tvevaṃ śrāvakayāne /
śrāvakasambhāraphalaṃ hi kṣayatvam ākhyātaṃ nirupadhiśeṣanirvṛtānāṃ 
sarvātmanā samucchedāt /
evaṃ nirvāṇe'pyanuparataṃ phalaṃ dyotanāc chrāvakayānān mahāyānaṃ 
viśiṣyate /
