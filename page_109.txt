	{ dhātvarthaḥ }
skandhārthānantaraṃ dhātava uddiṣṭāḥ /
tasmāt skandhārthānanataraṃ dhātvartham āha -
	grāhakagrāhyatad grāhavījārthaścāparo mataḥ /
iti /
aparaśabdo'tra dhātāveva sambadhyate skandhānantaram uddiṣṭatvāt /
ata eva bhāṣyakāro'pyāha katamo'paraḥ /
dhātur iti /
bījārtho dhātvarthaḥ tadyathā svarṇo sati svarṇadhātur ucyate 
svarṇabījam iti gamyate /
sa ca phalabhedāt tridhā bhidyate - grāhakadhātuḥ grāhyadhātuḥ tad 
grāhadhātuṣca /
tatra grāhakabījārthaścakṣurdhātvādayaḥ /
cakṣurdhāturādir yeṣāṃ manaḥparyantās te cakṣur dhātvādayaḥ /
te ca rūpādiviṣayagrāhāśrayabhūtatvād grāhakāḥ /
sajātīyānāṃ cakṣurādīnāṃ hetutvād dhātava ucyante /
grāhyabījārtho rūpadhātvādayo dharmadhātuparyantāḥ /
cakṣurādivijñānaviṣayatvād grāhyāḥ /
te'pyevaṃ sabhāgānāṃ rūpādīnāṃ hetubhāvād dhātava ucyante /
tad grāhabījārthaścakṣurvijñānadhātvādayaḥ /
rūpādipratipattisvabhāvatvād rūpādigrāhāścakṣurivjñānādayo 
manovijñānaparyantāḥ /
te'pi sabhāgānām eva cakṣurādivijñānānāṃ hetubhūtatvād dhātava 
ucyante cakṣurvijñānadhātvādayo manovijñānadhātuparyantāḥ /
anye tvāhuḥ - cakṣurādyabhiniveśakarmavāsitam ālayavijñānaṃ 
yathākramaṃ cakṣurādīnāṃ bījaṃ /
tathā rūpādyabhiniveśakarmavāsitam ālayavijñānaṃ yathākramaṃ 
rūpādīnāṃ bījaṃ /
tathā cakṣurādivijñānābhiniveśakarmavāsitam ālayavijñānam eva 
cakṣurādivijñānānāṃ yathākramaṃ bījam iti /
evaṃ tvālayavijñānam eva dhātuśabdavācyaṃ na cakṣurādaya iti cintyam 
etat /
kartṛkriyākarmasammohavyāvartanārthaṃ trividhe bījārthe dhātur 
vyavasthāpitaḥ /
atra pūrvacakṣurādaya uttaracakṣurādīnām eva hetavo bhavanti na 
tatrānyaḥ kaścid ātmā'nyo vā hetuvidyata ityevaṃ dhātukauśalyād 
ātmani hetugrāhavyāvṛttiḥ //3-17//
