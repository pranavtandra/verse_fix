bhavati tadātmatvena siddheḥ /
utpadyamānasyāpi nirarthakam asattvāt /
notpadyamānasya nirarthakaṃ yasmād viṣayasadbhāve vijñānaṃ tan 
nirbhāsam ātmānaṃ pratilabhate /
yadi viṣayanirbhāso vijñānasya bāhyārthanimittaḥ evam 
atītānāgatasvapnādivijñānasya nirākāratvaṃ vartate /
tad arthābhāvād vā sattvaṃ /
na vikalaiḥ pratyayaiḥ kāryam utpadyate atiprasa:ngāt /
na cāsataḥ pratyayatvaṃ yujyate /
na ca bāhyārtha ālambanapratyayo vijñānasya tena vināpi bhāvāt /
tasmāt sarvam eva vijñānaṃ taimirikādīnām iva 
bāhyārthālambananirapekṣam evābhyupagantavyaṃ //

	{ pratītyasamutpādārthaḥ }
āyatanā'rthānantaraṃ pratītyasamutpādārtho vaktavya ityata āha - 
pratītyasamutpādārthaḥ /
	punar hetuphalāyāsā'nāropā'napavādataḥ //
nirīhānityasamarthapratyayotpādanārthaḥ /
yathoktaṃ sūtre'pi - etad asmin satīdambhavati /
asyotpādādidam utpadyate yad utāvidyāpratyayāḥ saṃskārā iti vistaraḥ 
/
ayaṃ ca pratītyasamutpādo hetupahalakriyāṇāṃ 
samāropāpavādapratiṣedheneṣyata iti pratipādayann āha - 
hetuphalakriyāṇām asamāropavādārthaḥ pratītyasamutpādārtha iti /
āyāsaśabdena tu kriyoktā /
tatra hetusamāropaḥ avidyādīn hitvā puruṣeśvarāṇupradhāṇādīnāṃ 
hetutvakalpanāt /
etad eva pratipādayann āha - saṃskārādīnāṃ jarāmaraṇaparyantānāṃ 
viṣamahetukalpnād iti /
saṃskārādīnāṃ hy anityānāṃ nityaḥ puruṣādiko hetur viṣamo visadṛśa 
ityarthaḥ /
athavā hetuḥ svarūpavikriyā'pekṣo bījādiko':nkurādīnāṃ dṛṣṭaḥ /
na ca nityasya svarūpavikriyā'stīti viṣamahetau hetutvakalpanād 
hetusamāropaḥ /
