paratantre samāropā'pavādadarśanaṃ na pravartate /
bhāvā'bhāva samāropāpavādadarśanam iti vistaraḥ /
kecidāhuḥ yadi śūnyatāṃ bhāva eveti samanupaśYati evaṃ 
bhāvasamāropadarśanaṃ bhavati /
athā'bhāva evetyevam abhāvadarśanaṃ /
etad ubhayam api bhāvābhāvadarśanaṃ /
yasya lakṣaṇasya jñānān na te bhāvābhāvadarśane pravartete tat 
pariniṣpannasvabhāve tattvalakṣaṇam iti /
tac ca bhāvā'bhāvāvimuktasvabhāvaṃ /
taj jñānāddhi pariniṣpanne svabhāve bhāvā'bhāvapradarśanaṃ na 
pravartata ityeke /
anye punar āhuḥ - dvayābhāvasvarūpaiva śūnyatetyabhāvasamāropāntaḥ 
bhāvā'bhāvavimuktātmakatve bhāvasvarūpasamāropaṇāt /
bhāvā'bhāvavimuktena dvayā'bhāvasvarūpeṇa 
nirvikalpajñānagocareṇā'sau nāstītyabhāvāpavādāntaḥ /
bhāvā'bhāvayoḥ pratyekaṃ samāropā'pavādadarśanasyābhisambandhāc 
cintyam etad vyākhyānādvayam ityaparaḥ /
bhāvo'tra dvayā'bhāvasya bhāvaḥ /
taṃ yadi bhāvam eva gṛhṇātyevaṃ bhāvaḥ samāropito bhavati /
yady abhāvam eva gṛhṇātyevam apodito bhavati /
bhāvo'pidvayābhāvasthaṃ yadi ucchedato gṛhṇātyevaṃ 
bhāvo'tyantābhāvatvena samāropito bhavati/
atha dvayata eva gṛhṇātyevam apodito bhavati yasmādetad dvayābhāvena 
prabhāvitaṃ /
athavā taṃ bhāvā'bhāvasvarūpeṇa gṛhṇātyevaṃ samāropito bhavati /
athā'bhāvam eva gṛhṇātyevaṃ saṃkleśo vyavadānañcā'poditam bhavatīti /
etad antadvayavidhātanaṃ pariniṣpannatve tattvalakṣaṇaṃ /
nanu pūrvam api trisvabhāvalakṣaṇam uktaṃ -
	arthād abhūtakalpāc ca dvayābhāvāc ca deśitaḥ //
iti /
tatra svabhāvo lakṣaṇam iha tu cihnam iti nāsti punaruktatā /
	yajjñānān na pravarteta taddhi tattvasya lakṣaṇaṃ /
iti pudgalādisamāropāpavādadarśanaṃ yaj jñānān na pravartate tan 
mūlatattvasya lakṣaṇaṃ //3-4//
