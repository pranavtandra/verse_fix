ananyatvāc caturvidhalakṣaṇatathatā /
vijñaptitattvaṃ vijñaptimātratā dharmāṇāṃ /
nirvikalpaṃ jñānaṃ vijñaptyāśrayabhāvād vijñptitattvam ity apare /
ananyathābhāvatvād vijñaptitattvasya vijñaptitathatocyate /
sanniveśatattvaṃ duḥkhasatyaṃ saṃskārāṇāṃ duḥkhasanniviṣṭatvāt /
saiva ca tathatā nityaṃ tathaiveti kṛtvā /
mithyāpratipattitattvaṃ samudayasatyaṃ /
sā punar yā teṣu saṃskāreṣu guṇadarśanena pravṛttiḥ /
viśuddhitattvaṃ kleśajñeyāvaraṇaviśuddhiḥ /
sā punas tathā nirodhasatyañca /
āśrayaparāvṛttyātmatā nirodhātmatā nirodhasatyam evetyapare /
samyakpratipattitattvaṃ mārgasatyaṃ /
vijñaptitattvaṃ yoniśaḥ prayogaḥ /
samyakpratipattitattvaṃ vijñaptitattvaṃ ca mārgasatyam evetyanye /
etāny api trīṇi tattvāni nityaṃ tathaiveti tathatāvacanam bhavati /
tatra -
	pravṛttitattvaṃ dvividhaṃ
mūlatattvaṃ veditavyaṃ parikalpitaparatantralakṣaṇaṃ /
tatra pravṛttir janmanāmanādimān prabandhaḥ /
sā cā'vidyamānatvāt parikalpitaḥ svabhāvaḥ /
yeṣāṃ tu sa prabandhaṣthe hetuprabhāvitatvāt paratantraḥ svabhāvaḥ /
evaṃ yeṣāṃ yā ca pravṛttis tat pravṛttitattvam iti nirdiṣṭaṃ bhavati /
athavā kleśakarmajanmanām anādimān utpādaḥ saṃsārapravṛttir ityucyate 
/
sā ca parikalpitasvabhāvābhiniveśaprabhāvetetyāha -
	pravṛttitattvaṃ dvividhaṃ
mūlatattvam ity uktaṃ /
tathā -
	sanniveśakupannatā /
iti /
mithyāpratipattiḥ kupannatā /
tathaiva dvividhaṃ mūlatattvam iti duḥkhasamudayayor 
hetupratyayaprabhāvitatvāt parikalpitaviṣayatvāc ca 
sanniveśamithyāpratipattitattvaṃ dvividhe saṃgṛhītaṃ /
	ekaṃ lakṣaṇavijñaptiśuddhisamyak prapannatā /
lakṣaṇatattvaṃ vijñaptitattvaṃ viśuddhitattvaṃ samyak pratipattitattvaṃ 
etāni catvāri tatvāny ekam eva mūlatattvaṃ yad uta pariniṣpannaḥ 
svabhāvaḥ /
tatra lakṣaṇatattvaviśuddhitattvayor ananyathāpariniṣpannatvāt 
pariniṣpannaḥ /
vijñaptitattvaṃ samyak pratipattitattvaṃ cā'viparyāsapariniṣpattyā 
pariniṣpannaḥ /
