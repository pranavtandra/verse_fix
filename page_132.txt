vistarīkṛtamālambanam iti cittasthāpanīyam avavādavastv abhilapanam 
ityartaḥ /
samprajanyaṃ smṛtyasampramoṣe sati layauddhatyāvabodha iti /
upasthitasmṛtir hi samprajanyavān bhavatyevetyata āha - smṛtyasampramoṣe 
satīti /
anubuddhyā layauddhatyaṃ śamathavipaśyane yuganaddhādaprayatnena 
pravartetetyarthaḥ /
upekṣeti viṣayāntarānabhisaṃskāra ityarthaḥ /
etad uktaṃ bhavati - sūpasthitasmṛtiravavāde līnaṃ cittam uddhtaṃ vā 
yathābhūtaṃ prajānāti /
layauddhatyavigamāya ca tad abhisaṃskaroti layauddhatyavigamāc copekṣāṃ 
karotīti /
evaṃ tasya cittapraśaṭhatā karmaṇyatā /
atra ca karmaṇyatāsthitir iti svārthe bhāvapratyayaḥ svapeṭatvādivat //

	{ pañcendriyāṇi }
ṛddhipādānantaraṃ pañcendriyāṇi śraddhādīnyuktānīti sambadhyate /
teṣāṃ kathaṃ vyavasthānam iti kenārthenetyabhiprāyaḥ /
ata āha -
	ropite mokṣabhāgīye chandayogādhipatyataḥ /
iti vistaraḥ /
ādhipatyata iti vartata ityanena cchandayogādhipatyata 
ityatrādhipatyaśabdaḥ prayukta uttaratrāpyanuvartata iti darśayati /
ṛddhipādaiḥ karmaṇyacittasyāropite mokṣabhāgīye kuśalamūle iti /
karmaṇyacittasya vimokṣabhāgīyaṃ kuśalamūlaṃ cittasandhāne pratiṣṭhāṃ 
pratilabhate chandādhipatyato yāvat pravicayādhipatyataḥ na tv 
akarmaṇyacittasya /
sā ca cittakarmaṇyatā ṛddhipādair bhavati /
tasmād ṛddhipādānantaraṃ vimokṣabhāgīyakuśalamūlaropaṇādhipatyataḥ 
pañca śraddhādīnīndriyāṇīti vyavasthāpyante /
chandādhipatyata iti kāraṇe kāryopacārāśchraddhaivātra 
chandaśabdenoktā /tadyathā dadhatrapuṣaṃ sadyojvara iti /
athavāstitvaguṇaśaktiṣu yathākramam abhisampratyayaprasādābhilāṣāḥ 
śraddhāyā lakṣaṇaṃ /
ataścandagrahaṇenātrābhilāṣalakṣaṇā śraddhaiva gṛhyate na tu cchanda iti 
/
prayogādhipatyata iti prayujyate 'neneti prayogaḥ /
prayogaśabdena vīryam abhipretaṃ /
ālambanāsampramoṣādhipatyata iti smṛtīndriyādhipatyataḥ /
ālmbanābhilapanalakṣaṇaṃ smṛtīndriyaṃ /
avisārādhipatyata iti samādhīndriyādhipatyataḥ /
avisāro hi samādhīndriyaṃ cittaikāgratālakṣaṇatvāt /
pravicayādhipatyataśceti prajñendriyādhipatyataḥ /
prajñendriyasya dharmapravicayātmakatvāt /
ca śabdo vimokṣabhāgīyakuśalamūlaropaṇādhipattyāt /
