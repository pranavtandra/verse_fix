{ śūnyatāpiṇḍdārthaḥ }
śūnyatāpiṇḍdārtho lakṣaṇatāṃ vyavasthānataśceti veditavyaḥ /
tatra lakṣaṇato 'bhāvalakṣaṇato bhāvalakṣaṇataś ceti dvayābhāvo 
hītivacanād abhāvalakṣaṇataḥ abhāvasya ca bhāva iti vacanād 
bhāvalakṣaṇataḥ /
bhāvalakṣaṇataś ca na bhāvo nā'pi cā'bhāva iti vacanāt /
bhāvābhāvavinirmuktalakṣaṇatas tattvānyatva vinirmuktalakṣaṇaś ceti /
etac ca śūnyatāyā lakṣaṇaṃ /
tasmād abhūtaparikalpān na pṛthaktvaikalakṣaṇam iti vacanāt /
evaṃ lakṣaṇataḥ piṇḍārthaḥ /
kathaṃ vyavasthānataḥ piṇḍdārtho veditavya ityta āha -
vyavasthānaṃ punaḥ paryāyādivyavasthānato veditavyaṃ /
paryāyatas tadarthas tadabhedas tat sādhanañcetyarthaḥ /
etayā catuḥprakāradeśanayā caturvidhopakleśapratipakṣeṇa svalakṣaṇaṃ 
karmalakṣaṇaṃ sa:nkleśavyavadānalakṣaṇaṃ yuktilakṣaṇañcodbhāvitaṃ bhavati 
/
tatra svalakṣaṇaṃ vikalpasya pratipakṣeṇa tasya hi 
bhāvā'bhāvobhayapṛthaktvaikatvagrāhātmakatvatvāt /
karmalakṣaṇam anadhimuktānāṃ śūnyatālakṣaṇāśratyātrāsasya 
pratipakṣeṇa /
abhrāntitathatā karmā'viparyāsakarma sarvanimittaprahāṇakarma 
sarvalokottarajñānaviṣayatvā'vasthānakarma 
pratilabdhāryadharmahetubhāvakarma ca /
evaṃ śūnyatāsvarūpakarmaśravaṇena prāptagrahāṇāṃ kusīdināṃ 
kausīdyāpananārthaṃ prabhedalakṣaṇaṃ /
kathaṃ sā sa:nkleśo bhavati vyavadānaṃ ceti saṃśayināṃ 
saṃśayāpanayanārthaṃ yukti lakṣaṇaṃ //
