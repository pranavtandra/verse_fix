iti darśanamārgaprāptiḥ /
abhyāsaphalaṃ pareṇa śaikṣāvasthāyām iti /
darśanamārgāt pareṇa śikṣāvasthāyāṃ pañcapudgalaprabhāvitāyāṃ 
sakṛdāgāmiphalam anāgāmiphalañca 
saprayogapṛṣṭhabhāvanāmārgātmakatvād abhyāsaphalaṃ /
samāptipahlam aśaikṣā dharmā iti /
te punaḥ samyag 
dṛṣṭisa:nkalpavākkarmāntājīvavyāyāma-smṛtisamādhivimukti-vimuktijñānadarśanākhyāḥ 
sarvaphalaparyantāvasthitatvāt samāptiphalam ity ucyate /
ānukṛtyaphalam upaniṣadbhāvenottarottaraphalam eva veditavyam iti /
upaniṣad bhāvo hetuḥ /
tatra cittotpādo gotrānukūlatvāc cittotpādaḥ phalam iti /
uttarottaraphalādīnām ānukūlyaphalādīnāñca pṛthag upadeśo 
yathākramam uddeśanirdeśato veditavya iti /
vipakṣaphalaṃ prahāṇamārgaḥ yadevādiphalaṃ /
vipakṣa eva phalam iti vipakṣaphalaṃ prahāṇamārgo yena kleśebhyo 
visaṃyujyate /
ādiphalaṃ yat pūrvam uktaṃ prathamato lokottaradharmapratilambha iti /
vipakṣaśabdo yadyapi pratipakṣavibandhanārthenāvaraṇam ucyate atra tu 
vipakṣaḥ pratipakṣo'bhipreta iti āvaraṇanirvāsapakṣo vipakṣa iti vigrahāt 
/
visaṃyogaphalam abhyāsaphalaṃ samāptiphalaṃ ca kleśavisaṃyogaḥ 
śaikṣāśaikṣāṇāṃ yathākramam iti /
abhyāsaphalaṃ visaṃyogaḥ sakṛdāgāmiphalapratipannādīnāṃ pañcānāṃ 
śaikṣāṇāṃ śiṣṭakleśatvāt /
samāptiphalaṃ visaṃyogaḥ /
aśaikṣāṇāṃ parihāṇidharmādīnāṃ ṣaṇṇām api 
traidhātukāśeṣakleśaprahīṇatvāt /
pūrvaṃ bhedenoktam idānīm abhedeneti viśeṣaḥ /
viśeṣaphalam abhijñādiko guṇaviśeṣa iti /
etad apyabhyāsasamāptiphalaviśeṣa eva veditavyaṃ /
kiṃ kāraṇaṃ ?
anāgāminām arhatāṃ pratyekabuddhānāṃ bodhisattvānāṃ 
tathāgatānāñca vaiśeṣikaguṇalābhasya vivakṣitatvāt /
uttaraphalaṃ bodhisattvabhūmayaḥ /
atra kāraṇam āha - tad anyayā nottaratvād iti /
bodhisattvabhūmayo hi yadyapi darśanabhāvanāmārgātmakatvād ādiphalam 
abhyāsaphalañca tathāpi śrāvakapratyekabuddhayānāpekṣayā 
bodhisattvabhūmīnām uttaraphalatvaṃ nirvikalpatvād buddhagotrakatvān 
mahābodhiphalatvāc ca veditavyaṃ /
anuttaraphalaṃ buddhabhūmir iti /
etad api samāptiphalam eva yataḥ sarvasavāsanākleśopakleśaprahāṇād 
akliṣṭājñānaprahāṇāc ca tad uttarabhūmyantarābhāvād anuttaraphalam 
ity ucyate /
etad anyat phalaṃ samāsanirdeśataḥ vyāsatastvaparimāṇam iti /
vyāsataḥ śrāvakapratyekabuddhabodhisattvasambuddhaphalāni 
pratipādyamānānyaparimāṇāni bhavantīti //

	{ pratipakṣabhāvanāpiṇḍārthaḥ }
