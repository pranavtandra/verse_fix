buddhabodhisattveṣvadhimuktyā svabhāvatrayamukhena dharmān pratipadyate /
tadanyasattvānāṃ duṣkaradānaśīlādiṣu buddhatvaniṣpādasambhāreṣu 
pravartata ityata iyam avasthādhimuktiprabhāvitattvād 
adhimukticaryetyucyate /
sā ca kalpāsaṃkhyeyam adhikaṃ vā /
praveśāvasthā prathamāyāṃ bhūmāv iti /
svabhāvatrayasyāvikalpena jñānena tatprathamata eva 
tattvasvasamvedanaprativedha iti praveśāvasthocyate /
eṣaiva bodhisattvasya darśanamārgaḥ niryāṇāvasthā tad uttarāsu ṣaṭsu 
bhūmiṣu iti /
bhāvanāprahātavyasya nimittanānātvābhiniveśasya prahāṇaniṣṭāgamanāt 
sābhogaprayoganiṣṭhāgamanāc caitā bhūmayo niryāṇāvasthety ucyate /
vyākaraṇāvasthā'ṣṭamyāṃ bhūmāv iti /
sarvathā sarvanimittābhogasaṃjñāvigamād aṣṭamyāṃ bhūmau niyatipatito 
bhavati buddhatva iti buddhair bhagavadbhiḥ sammukhavyākaraṇena vyākriyate 
/
kathakatvāvasthā navamyām iti /
navamyāṃ bhūmau śrāvakapratyekabuddhebhyaḥ prativiśiṣṭāścatasraḥ 
pratisamvidaḥ pratilabhata ity anuttaro mahādhārmakathiko bhavati /
abhiṣekāvasthā daśamyāṃ bhūmāv iti /
daśamyāṃ hi bhūmau bodhisatto mārganiṣṭhāgato 
daśadiglokadhātuvyavasthitānāṃ buddhānāṃ bhagavatāṃ raśmīṇām 
abhiṣekeṇābhiṣicyate /
prāptyavasthā buddhānāṃ dharmakāya iti /
sarvāvaraṇaprahāṇāt tat pratipakṣānāsravadharmabījapracayāc 
cāśrayaparāvṛttyātmakaḥ sarvadharmavartyānālaya iti buddhānāṃ 
dharmakāyāḥ /
tena dharmaprativedhaniṣṭhādhigamanāt prāptyavasthety ucyate /
anye tu niḥśeṣāgantukamalāpagamāt suviśuddho dharmadhātur eva 
dharmatākāyo dharmakāya iti varṇayanti /
sarvasminn api jñeye'saktāpratihatapravṛttiviśiṣṭaṃ jñānaṃ dharmakāya 
ityapare /
anuśaṃsā'vasthā sāmbhogikaḥ kāya iti /
svābhāviko yasmin kāye vyavasthite'bhisambudhyate yena kāyena 
niṣṭhāgatair bodhisattvaiḥ saha parṣanmaṇḍalena dharmasaṃgītim anubhavati 
sa sāmbhogikaḥ kāyaḥ /
kṛtyanuṣṭhānāvasthā nirmāṇakāya iti /
svābhāvikakāyāśritaḥ sattvāśayālocano vineyārthaṃ 
prakriyāpravṛttāvanantaprabhedo nairmāṇikaḥ kāyaḥ /
sarvāpyeṣā bahuvidhā'vasthā'bhisamasya veditavyā -
	dharmadhātau tridhā punaḥ //
iti /
katham ityata āha -
	aśuddhaśuddhaśuddhā ca viśuddhā ca yathārhataḥ /
iti /
tatrāśuddhāvasthā hetvavasthām upādāya yāvat prayogād iti
