yadyanirjitavipakṣāṇīndriyāṇi tat kathaṃ kauśalyatattve 
laukikaviśuddhyādhipatyena śraddhādīnām indriyatvam uktaṃ /
tatra hy abhedenendriyāṇy abhipretāni atra tu nirvedhabhāgīyasya 
kuśalamūlasya bhedapradarśanārthaṃ /
tasmān nāsti virodhaḥ /
atra evātrāha etānyevendriyāṇi balāny ucyanta iti /
śraddadhāno hi hetuphale vīryam ārabhate /
ayaṃ heturasyaiva phalasya idañca phalam asyaiva hetor ityevaṃ 
śraddadhānaḥ phalaśraddhayā taddhetau vīryam ārabhate /
ārabhdavīryasya tat paratvāt smṛtir upatiṣṭhate /
upasthitasmṛter vipakṣepābhāvāc cittaṃ samādhīyate ekāgraṃ bhavati /
samāhitacitto yathābhūtaṃ prajānātīti samādhyanugṛhītā prajñā 
sphuṭā bhavati /
avaropitamokṣa bhāgīyasyendriyāṇy uktāni /
mokṣabhāgīyānantaraṃ nirvedhabhāgīyānīti indriyānantarañca 
balānyato vaktavyaṃ nirvedhabhāgīyāni kim indriyāvasthāyāṃ 
veditavyānyāhosvid balāvasthāyāṃ ?
tathā kiṃ nirvedhabhāgīyānīnidriyāṇi balasvabhāvānyāhosvit tad 
vyatiriktasvabhāvānītyata āha -
	dvau dvau nirvedhabhāgīyāvīndriyāṇi balāni ca /
iti /
athavā yathā moksabhāgīyāni pratipakṣabhāvanāprabhāvitāni 
śraddhādiprabhāvitāni vā bhavanti tathā nirvedhabhāgīyāny api /
ata āha - ropitamokṣabhāgīyasyendriyāṇy uktānīti vistaraḥ /
ata āha -
	dvau dvau nirvedhabhāgīyau
iti sarvaṃ /
uṣmagatāni mūrdhāścendriyāṇīti /
mṛdutvād ayuktatvād eṣāṃ kuśalānām eṣu balānām indriyāṇyeva /
kṣāntayo laukikāścāgradharmā balānīti /
adhimātratvād eṣām indiryāṇām asambhavāt kṣāntayo'gradharmāś ca 
balānyeva /
mṛdumadhyādhimātrabhedena hi trividhāni śraddhādīni /
tatra mṛdūni madhyāni cendriyāṇi /
adhimātrāṇi balāni /
yāni mṛdūni tānyuṣmagatāni /
madhyāni mūrdhānaḥ /
adhimātrāṇi punaścaturdhā kriyante /
tatra mṛdumadhyādhimātrāṇi yathākramaṃ mṛdumadhyādhimātrāḥ kṣāntayaḥ 
/
adhimātrādhimātrāṇi laukikā agradharmā iti /
etāni ca nirvedhabhāgīyāni sarvāṇi
