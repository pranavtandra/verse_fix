yatra duḥkhaṃ nāsti tac ca nirvāṇaṃ tasyākaraṇam aprāptir nirvāṇaṃ na 
prāpnotītyarthaḥ /
athavā duḥkhasyānto yasmād duḥkhādanyaduḥkhaṃ na sandhīyate tat 
tādṛśaṃ duḥkhaṃ na karotītyapratisandhikaṃ duḥkham na karotītyarthaḥ /
pañca nivaraṇaprahāṇapratibaddhatā viśuddhau pāratantryaṃ /
samotpattau pāratantryaṃ dvayor apūrvācaramayor iti sahitayor ityarthaḥ /
tathāgatayoścakravartinoścaikasmil lokadhātāvanutpādād iti /
tathāgatayos trisāhasramahāsāhasro lokadhātur abhipretaḥ /
cakravartinos tu cātur dvīpaka iti kecit /
tathāgatayor api cātur dvīpaka evetyapare /
buddhā hi bhagavanto dharmakāyādhipatyāt trisāhasramahāsāhasre 
lokadhātau pratyekaṃ cātur dvīpakeṣu nirmāṇakāyenābhisambodhiṃ 
parinirvāṇaṃ ca darśayanti /
tathā hi yadekasmiñcāturmahādvīpake tuṣiteṣu manuṣyeṣu ca 
cyutijanmaparigraho bodhisattvānām prayojanaṃ /
taditareṣv api cātur dvīpakeṣu samānaṃ /
tasmād akaniṣṭhabhavana evābhisambuddho bhagavān nirmāṇakāyena 
trisāhasramahāsāhasre vantargateṣu sarvacāturdvīpakeṣu 
tuṣiteṣūpapattiṃ manuṣyeṣu ca janmaparigrahādikaṃ sarvaṃ darśayatīti 
manyante /
tatra ca samasaṃvartanīyakarmāyattatā buddhacarkravartyutpatteḥ 
sahabhāve pāratantryaṃ /
ādhipatye pāratantryaṃ striyā cakravartitvādyakaraṇād iti 
cakravartisaṃvartanīyasya karmaṇaḥ pumātmabhāva eva 
vipākadānasāmarthyāt /
na hi strīḥ stryādiratnopabhogasamarthā /
ādiśabdena śakratvādr gṛhyate /
samprāptau pāratantryaṃ stryāpratyekānuttarabodhyanabhisambodhāt /
prāptir buddha-pratyekabuddhatvādhigamaḥ /
tatra pāratantryaṃ puruṣasya prāptir na tu striyaḥ /
tasyā hi saṃsargābhāvāt khaḍgavat 
tribhūmiśāstrītvāsambhavādyathākramaṃ /
sā punaḥ sarvadhūrtajanagamyā alpaprajñatvād 
anācāryakābhisambodhāyogyā ceti /
samudācāre pāratantryaṃ dṛṣṭisampannasya vadhādyasamudācārāt 
pṛthagjanasya ca samudācārād iti /
dṛṣṭisampanno dṛṣṭasatyaḥ /
vadhaḥ prāṇātipātaḥ /
