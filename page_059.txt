kim etad evāvaraṇaṃ /
etat tu sāmānyaṃ bodhisattvaśrāvakādīnāṃ /
bodhisattvānāntvāvaraṇaṃ /
	śubhādau daśadhā'paraṃ /
āvaraṇaṃ /
ke ca śubhādaya iti na jñāyate /
ata āha -
	aprayogo'nāyatane 'yogavihitaś ca yaḥ /
ityevamādyāvaraṇaṃ /
śubhādayaḥ punaḥ /
	śubhaṃ bodhiḥ samādānaṃ
ityādi daśārthāḥ /
tatra śubhaṃ hi bodhihetubhūtaṃ sarvakuśalamūlaṃ /
bodhis tu tat phalaṃ /
tatra śubhaṃ sāmānyata uddiṣṭaṃ prabhedataḥ svabhāvataś ca na vijñāyata 
ity atas tat pratipādanārtham ārabhyate -
	samādānaṃ dhīmattā'bhrāntyanāvṛtī /
	nānyatrāso'matsaritvaṃ vaśitvaṃ ca
ityetac chubham iti /
bodhir api vacanamātreṇa svabhāvato na vijñāyata ityasyā vibhutvaṃ 
darśayati /
śubhasya bodheś ca samādānādīnām iva svātantryeṇāvaraṇanirdeśāt 
svātantryam evā'tra vijñāyate na tad deśanirdeśabhāva ityeke /
utpattyāvaraṇaṃ hi śūbhe vyavasthāpyate tac ca bodhicittādīnāṃ 
sāmānyaṃ tasyāḥ śubhotpādyatvāt /
dhṛtyādyāvaraṇantu samādānādyāvaraṇaṃ samādānādiṣu 
vṛttyādyarthabhedāt bodhirapyekopyā prāptavyā cetyatas tasyāḥ 
sthityāvaraṇaṃ prāptyāvaraṇaṃ ca vyavasthāpyate /
tasmān nāsty atra svātantryaprasa:nga iti /
kramaś ca teṣāṃ samāsataḥ - kuśalamūlaṃ niravaśeṣaṃ niṣpādya bodhiḥ 
prāptavyā /
vyāsataḥ punaḥ prathamata eva bodhicittam utpādayitavyaṃ 
svaparārthasampadādhārabhūtatvāt /
tataścittotpādanā'nurūpā dānādiṣu pratipattiḥ yayā bodhisattva 
ityākhyāṃ labhate /
tato'saṃkhyeyakalpenābhyasyataḥ puṇyajñānasambhārasañcayād 
āśayaviśodhanārthaṃ pudgaladharmabhrāntipratipakṣeṇā'bhrāntyātmako 
darśanamārga utpādayitavyaḥ /
tataścāvaraṇavigamaprabhāvito bhāvanāmārgaḥ prakarṣaṃ netavyo 
viśuddhiviśeṣayogārthaṃ /
