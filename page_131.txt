prahāṇasaṃskārāḥ prahāṇāya saṃskārāḥ /
kasya prahāṇāya ?
anantaroktānāṃ pañcānāṃ doṣāṇāṃ prahāṇāya /
yadyevaṃ tena vaktavyam eṣāṃ prahāṇāya katham aṣṭau prahāṇasaṃskārā 
vyavasthāpyanta iti /
tatra kausīdyaprahāṇāya catvāraś chandavyāyāmaśraddhāprasrabdhayaḥ /
tatra cchando'bhilāṣaḥ /
vyāyāmo vīryaṃ /
śraddhā'bhisampratyayaḥ /
prasrabdhiḥ karmaṇyatā /
kathaṃ punar ete kausīdyaprahāṇāya vyavasthāpyanta ityata āha - te 
punar yathākramaṃ veditavyāḥ /
	āśrayo'rthāśritas tasya nimittaṃ phalam eva ca /
iti /
ato'nyonyaprabhāvitatvād ete kausīdyaprahāṇāya samvartanta iti 
darśayati - āśrayaśchando vyāyāmasya āśrito vyāyāmaḥ /
abhilāṣasya vyāyāmapūrvakatvāt /
tasyāśrayasya chandasya nimittaṃ śraddhā /
katham ityato bravīti - sampratyaye saty abhilāṣād iti /
śraddadhānasya hi hetuphale kriyābhilāṣo bhavati /
tasyāśritasya vyāyāmasya phalaṃ prasrabdhiḥ /
katham ityata āha - prārabdhavīryasya samādhiviśeṣā'dhigamād iti /
samādhiviśeṣagamanaṃ hi prasrabdhinimittaṃ /
ata ārabdhavīryasya samādhiviśeṣagamanāt prasrabdhir vyāyāmaphalam 
ityuktaṃ /
tathā hy ārabdhavīryasya dauṣṭhulyavitarkādyapakṣālavigamāt pramoda 
utpadyate /
manaḥpramodāt kāyacittayoḥ prasrabdhiḥ karmaṇyatālakṣaṇatā /
tataḥ samādhiviśeṣagamanam iti /
atra ca vīryaṃ sahetukaṃ saphalaṃ ca kausīdyapratipakṣeṇoktaṃ /
caturṇāṃ doṣāṇāṃ yathāsaṃkhyaṃ pratipakṣa iti /
tatra smṛtir avavādasammoṣasya pratipakṣaḥ /
samprayojanyaṃ layauddhatyasya /
cetanā'nabhisaṃskārasya /
abhisaṃskārasyopekṣeti saṃskāropekṣā'tra veditavyā /
smṛtyādīnāṃ punar lakṣaṇavivakṣayedam āha - te punaḥ smṛtyādayo 
veditavyā yathākramam iti /
smṛtir ālambane'sammoṣa iti /
