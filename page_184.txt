vikalpadvayāntasya parivarjanārthaṃ māyākāradṛṣṭāntaḥ /t
tadyathā kāśyapa māyākāraḥ puruṣo māyākṛtaṃ nirmimīti /
atha sa māyānirmitas tameva māyākāraṃ khādet /
evam eva kāśyapa yogācāro bhikṣur yadyadevārambaṇaṃ manaskaroti tat 
sarvam asya riktam eva khyātīti vistaraḥ /
katham anenaivaṃ vidhaṃ vikalpadvayāntaṃ parivarjayantīti ?
ata āha - vijñaptimātrajñānakṛtaṃ hy arthābhāvajñānaṃ /
tac cārthābhāvajñānaṃ tadeva vijñaptimātrajñānaṃ nivartayati /
arthābhāve vijñaptyasambhavād ityetad atra sādharmyam iti /
māyākārasthānīyaṃ vijñaptimātratājñānaṃ /
nirmitayakṣasthānīyam arthābhāvajñānaṃ /
vijñaptimātrajñānaprabhāvitatvād arthābhāvajñānasya /
yathā nirmito nirmātāraṃ bhakṣyedevam arthābhāvajñānaṃ tadeva 
vijñaptimātrajñānaṃ bhakṣyati /
asatyarthe vijñaptimātrasyāsaṃbhavād arthopalabdhir vijñaptir iti 
kṛtvā /
samyaktve'pi vikalpo'nto mithyātve'pi vikalpo'nta ityadhikṛtaṃ /
nirvedhabhāgīyāvasthāyāṃ bhūtapratyavekṣāṃ samyaktvena vā 
vikalpayato mithyātvena vā 'ntadvayaṃ /
kathaṃ ?
samyaktvamanāsravā dharmā mithyātvaṃ sāsravāḥ /
tatra yadi bhūtapratyavekṣāṃ samyaktvenaivādhyavastyatyevaṃ 
kuśalasāsravā na bhūtapratyavekṣā syāt /
atha punar asamyaktvalakṣaṇam eva evam api mithyātvam evopapadyate 
traidhātukapratisaṃyuktakuśalavad iti /
etasyāntadvayasya parivarjanārthaṃ kāṣṭhadvayāgnidṛṣṭāntaḥ /
tadyathā kāśYapa kāṣṭhadvayaṃ pratītyagnir jāyata iti jātaś ca 
samānastadeva kāṣṭhadvayaṃ dahati /
evam eva kāśyapa bhūtapratyavekṣāṃ pratītyāryaṃ prajñendriyaṃ jāyate 
jātaṃ ca tām eva bhūtapratyavekṣāṃ dahatīti /
amamātrārthaḥ - yathā kāṣṭhadvayād anagnilakṣaṇād agnir jāyate 
jātaśca kāṣṭhadvayaṃ dahatīti /
evam evāsamyaktvalakṣaṇāyā bhūtapratyavekṣāyāḥ samyaktvalakṣaṇam 
āryaṃ prajñendriyaṃ jāyate /
jātaś ca tām eva bhūtaparatyavekṣāṃ dahati /
vibhāvayatītyarthaḥ /
sā hi vibhāvanād eva dagdheva dagdhety ucyate vikṛtidarśanāt /
iyañca bhūtapratyavekṣā sāsravāpi satī samyaktvānukūlyād 
bhūtapratyavekṣety ucyata iti /
vyāpṛtāv api vikalpo'ntaḥ avyāpṛtāv api /
vikalpo'nta ity adhikṛtaṃ /
katham ityata āha - jñānasya buddhipūrvāṃ kriyāṃ niḥsāmarthyaṃ vā 
kalpayata iti /
