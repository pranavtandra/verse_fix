upalabdher anupalabdhisvabhāvatvam iti pradarśayann āha -
	upalabdhes tataḥ siddhā nopalabdhisvabhāvatā /
iti /
atha vā yat pūrvaṃ pratijñātam abhūtaparikalpe dvayaṃ nāstīti tad 
anenākāreṇa pratyātmavedyatvāt siddham iti pradarśayann āha -
	upalabdhes tataḥ siddhā nopalabhisvabhāvatā /
iti /
tata iti -
upalabhyārthābhāve upālabdhyabhāvāt /
upalabdhir iti bhāvakartṛkaraṇānām anyatamat parikalpyate /
etat tritayam apy abhāvān na yujyate /
	upalabdhes tataḥ siddhā nopalabdhisvabhāvatā /
ata eva vṛttikāra āha -
	upalabhyārthābhāva upalabdhyayogād iti /
	tasmāc ca samatā jñeyā nopalambhopalambhayoḥ /
yasmād upalabdhir anupalabdhisvabhāvā tasmāt samatā tulyatā jñeyā /
arthānupalambhasya vijñaptimātratopalambhasya cāsattvād aviśeṣataḥ /
upalabdhyanupalabdhiśabdayoḥ parasparaviruddhatvāpanaya nārthamāha -
abhūtārthapratibhāsatayā tūpalabdhir ity ucyate /
na tu tayā kiñcid upalabhyte'rthābhāvād iti /
ataḥ paramārthato 'nupalabdhisvabhāvā'pi satīti nāsti virodhaḥ /
apara āha -
yaś ca bālānām asatyārthe'rthopalambho yaścāryāṇām arthānupalambhas 
tayor dvayos tulyalakṣaṇatvāt samatā jñeyā sarpabhrānty anupalabdhivat 
/
ata eva bravīti - abhūtārthapratibhāsatayā tūpalabdhir ity 
ucyate'nupalabdhisvabhāvā'pi satīti sarpabhrānty anupalabdhivyapadeśavad 
iti /
anyaḥ punar āha -
	yaś ca bālānāṃ grāhakopalambho yaścāryāṇām arthābhāvād 
grāhakānupalambhas tayor dvayor upalambhānupalambhayor ubahyato'pi 
samatā jñeyā grāhye'sati grāhakābhāvāviśeṣāt /
ata evāha - abhūtārthapratibhāsatayeti vistaraḥ /
kecitu manyante samāropā'pavādapratipakṣeṇāha -
	tasmāc ca samtā jñeyā nopalambhopalambhayoḥ /
iti /
arthābhāvād upalabdher upalabdhisvarūpatvenā'bhāvāt svarūpeṇa tad 
abhāva ucyate /
upalambha upalabdhisvabhāvo nāpanīyate nāpyanupalabdhisvabhāvaḥ 
prakṣipyate /
kiṃ tarhi samam etad ubhayaṃ nirvikalpatayā /
tasmānnopalambhopalambhayoḥ samāropāpavādanirapekṣeṇa pratyātmasamatā 
jñeyeti /
āha ca -
	nāpaneyam ataḥ kiñcit prakṣeptavyaṃ na kiñcana /
	draṣṭavyaṃ bhūtato bhūtaṃ bhūtadarśī vimucyate //
