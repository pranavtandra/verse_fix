atha vā /
upabhogo vedaneti yathāsvamālambanānubhāvāt 
sukhādivedanīyakarmānubhavanāc ca /
evaṃ viṣayakarmaṇor upabhuktir vedanety upabhogaḥ /
viṣayacihnaṃ vyavahāranimittaṃ paricchinnaṃ karotīti paricchedaḥ saṃjñā 
/
vijñānāntarālambana ābhogaḥ saṃskārāḥ /
tataścetanādyādhipatyād vijñānam ālambanānantaraṃ bhajate /
ādiśabdena chandhādayo gṛhyante /
uktaṃ yato yādṛśo yad arthā ca pravṛttir iti //1-10//

tatra chādanād avidyayā yathābhūtadarśanāvabandhanāt /
ropaṇāt saṃskārair vijñāne karmavāsanāyāḥ pratiṣṭhāpanāt /
nayanād vijñānenopapattisthānasaṃprāpaṇāt /
samparigrahaṇān nāmarūpeṇātmabhāvasya /
pūraṇāt ṣaḍāyatanena /
triparicchedāt sparśena /
upabhogād vedanayā /
karṣaṇāt tṛṣṇayā karmākṣiptasya punar bhavasya /
nibandhanād upādānair vijñānasyotpatyanukūleṣu kāmādiṣu /
ābhimukhyād bhavena kṛtasya karmaṇaḥ puarnbhave 
vipākadānāyābhimukhīkaraṇāt /
duḥKhanā jantyā jarāmaraṇena ca parikliśyate jagat /
sa:nkleśalakṣaṇañca khyāpayatīti kleśakarmajanmasa:nkleśā yathā 
pravartamānā jagataḥ parikleśāya bhavanti tat sa:nkleśalakṣaṇaṃ /
tathācā'satyapyātmaninabhūtaparikalpamātrāt saṃsāraḥ prajāyate iti 
pradarśanāthaṃ khyāpyate /
chādanādropaṇāc caiva nayanāt samparigrahāt /
iti vistaraḥ /
so'yaṃ pravṛttipakṣamadhi dvādaśā:ngaḥ pratītyasamutpādo darśitaḥ /
tatra chādanāt /
kliśyate jagad ityantavacanaṃ sarveṇa sambadhyate /
kena chādanāt kathaṃ vā chādanāt kliśyata iti ?
ata āha -
avidyayā yathābhūtadarśanavibandhād iti /
avidyayā hy adarśanātmakatvād bhūtadarśanaviṣaye pracchādite 
bhūtadarśanaṃ notpadyate'to bhūtadarśanotpattivibandhanād avidyā 
bhūtadarśanavibandhaḥ /
bhūtadarśanaṃ punaḥ prādhānyena lokottaraprajñā tat pṛṣṭhodbhavatvāt 
tat pṛṣṭhalābhāt tad avagamāc ca tat prayoga iti 
śrutacintābhāvanāmayī bhūtadarśanam ity ucyate /
bhūtadarśanavibandhād avidyā saṃskārāṇāṃ pratyaya ityevaṃ chādanād 
avidyayā kliśyate jagad iti /
