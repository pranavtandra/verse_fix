niyamāvakrāntipāṭhe'pi niyama eva niyamaḥ /
tathā hy utpannadarśanamārgo niyato bhavati sugatau nirvāṇe ceti /
kuśalamūlanirūḍhir dīrghakālaparicayāt sattvaparipākasamudāgama iti /
labdhadarśanamārgāṇāṃ tat pṛṣṭhalabdhasya bhāvanāmārgasya 
dīrghakālaparicayād abhyāsāt kuśalamūlavṛddhiḥ kuśalamūlanirūḍhiḥ /
tayā sattvaparipācanasamartho bhavati /
ataḥ kuśalamūlanirūḍhiḥ sattvaparipākasamudāgamaḥ /
anye tvāhuḥ - evam asya sattvaparipākaḥ samudāgato bhavatiḥ yadyanena 
dīrghakālābhyāsamāgamya sattvasantāneṣu kuśalamūlaṃ dṛḍhīkṛtaṃ /
cittakarmañatā kṣetrapariśuddhisamudāgama iti /
evaṃ bhāvanayā vivṛddhakuśalamūlānām ekāyatanamārgavipakṣayor 
nimittābhogayor vigamāc cittasya karmañatā /
yat karmañatādhipatyena yathecchaṃ sphaṭikādimayaṃ buddhakṣetraṃ 
darśayatītyaścittakarmaṇyatā kṣetrapariśruddhisamudāgamaḥ //

	{ yānānuttaryapiṇḍārthaḥ }
