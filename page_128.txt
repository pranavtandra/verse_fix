{ catvāri samyakprahāṇāni }
   smṛtyupasthānabhāvanānantaraṃ samyakprahāṇabhāvanoktā /
kiṃ kāraṇam ata āha yasmāt -
	parijñāte vipakṣe ca
iti vistaraḥ /
smṛtyupasthānabhāvanayeti /
aviśeṣābhidhāne'pi dharmasmṛtyupasthānabhāvanayaiveti veditavyaṃ /
dhātubhūmyavakāśaiḥ sāmānyalakṣaṇādyākārais tathā 
sāsravānāsravanadhātubhūmi-śaikṣāśaikṣādyākārais tathā 
sāsravānāsravadhātubhūmi-śaikṣāśaikṣādibhedena ca vipakṣe pratipakṣe 
ca sarvaprakāra parijñāte /
vipakṣāgamāyeti /
vipakṣāpagamamo'trotpannānāṃ prabandhacchedo'nutpannānāṃ cānutpādaḥ /
pratipakṣopagamāya ceti /
pratipakṣāgama utpannānāṃ prabandhānuvṛttir anutpannānāñcotpādaḥ /
vīryaṃ caturdhā pravartata iti vipakṣapratipakṣayor utpanānutpannabhedena 
caturdhā vīryasya pravṛttiḥ /
utpannānāṃ pāpakānām akuśalānāṃ dharmāṇāṃ prahāṇāyeti vistara 
itīdam āgamena vīryasyaiva caturdhā pravṛttiṃ darśayati /
vistaravacanena śeṣagranthābhisambandhaḥ /
tatrāyāṃ śeṣo granghaḥ - prahāṇāya chandaṃ janayati vyāyacchate vīryam 
ārabhate cittaṃ pragṛhṇāti samyakpradadhātīti /
evam anutpannānāṃ pāpakānām akuśalānāṃ dharmāṇām anutpādāya 
chandañjanayatīti vistaraḥ /
evam anutpannānāṃ kuśalānāṃ dharmāṇām utpādaya chandañjanayatīti 
vistaraḥ /
utpannānāṃ kuśalānāṃ dharmāṇāṃ sthitaye bhūyobhāvatāyai 
asampramoṣāya paripūraṇāya chandañjanayati vyāyacchate vīryam 
ārabhate cittaṃ pragṛhṇāti /
samyak pradadhāti /
utpannānāṃ sthitir aparihāṇiḥ /
bhūyobhāvaḥ prabandhenotpādaḥ /
paripūriḥ prakarṣagamanaṃ /
chandañjanayatīti sagauravodyaktavīryasamṛddhiṃ darśayati /
vyāyacchata iti kausīdyaprahāṇāya kāyacitte'bhimukhīkarotītyarthaḥ /
vīryam ārabhata iti layauddhatyāpakarṣaṇārthaṃ /
katham ārabheta ?
līnaṃ cittaṃ pragṛhṇāti samvejanīyapramodānīyadharmamanaskāraiḥ /
uddhataṃ cittaṃ tasminnevālambane samyak pradadhāti 
samvejanīyapramodanīyadharmamanaskārair dhārayatītyarthaḥ /
evañca
