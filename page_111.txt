upālambhaḥ /
yasmāt parikalpitarūpasyaivātra pratiṣedhaḥ kriyate na vikalpatārūpasya 
/
tatra cakṣurādiprapañcaparibhāvitakarmākṣiptatvenālayavijñānaṃ 
pariṇāmaviśeṣaprāptamādhyātmikam āyatanaṃ /
tan nimittasya pravṛttivijñānasya rūpādipratibhāsaṃ bāhyāyatanaṃ /
ya ālayavijñānasya sādhāraṇārthapratibhāso yaś ca sattvapratibhāsaḥ 
so'pi grāhyagrāhakapratibhāsasya 
pravṛttivijñānasyotpattāvadhipatipratyayatvād bāhyam āyatanam ucyate 
na tu tad viṣayatvād iti /
nanvevaṃ dvayam āśritya cakṣur vijñānādaya utpadyanta iti sūtravacanena 
virudhyate /
rūpādipratibhāso vijñānād bhinnaḥ .
na ca tadeva tasyaivotpattipratyayo yuktaḥ /
utpannānutpannāvasthayor bhāvād abhāvāc ca svātmani vṛttivirodhāt /
tasmād avaśyaṃ vijñānanirbhāsād bhinnā rūpādayo'bhyupagantavyā iti 
/
naiṣo'pyupālambhaḥ /
yasmād pādinirbhāsavijñānṃ sabhāgavijñānotpattaye pratyayavijñāne 
vāsanām ādhatte /
vasanāpariṇāmāc ca rūpādyābhāsaṃ punar vijñānaṃ jāyate /
evaṃ vijñānanirbhāsātmakatvena vāsanāhito vijñānapratyaya ukta iti 
na sūtravirodhaḥ /
evaṃ rūpādinirbhāsasyālambanārtho'py upapannaḥ svanirbhāsākārasya 
vijñānasya pratyayatvāt /
athavā yathā bāhyārthavādibhir vijñānena sahaja ālambanapratyayo 
vyavasthāpyate janyajanakavad abhāve'pi tad bhāvābhāvābhyāṃ 
vijñānabhāvābhāvāvadhikṛtya evam ihāpi rūpādinirbhāsasya 
vijñānasya vā'satyapi janyajanakabhāve tad bhāvābhāvayor 
vijñānabhāvābhāvavyabhicārāt pratyayapratyayitvaṃ vyavasthāpyate /
rūpanirbhāsavijñānayor abhinnatvād dṛṣṭāntena mukhyato'sādṛśyam iti 
na bhedopacārāt /
tadyathā grāhyagrāhakavikalpādhiṣṭḥānatvāt 
svapnamāyāgandharvanagarādivijñānaṃ paramārthato'abhinnam api 
grāhyagrāhakākāreṇa bhinnam iva vyavasthāpyata 
ālambyālambakabhāvatvena tathā'nyatrāpi nāsti virodhaḥ /
api cālambyālambakabhāvaḥ sarvatropacaryata eva na mukhyo'sti /
nahi vijñānam utpannam anutpannaṃ vā viṣayam ālambate /
utpattyanantaravināśitvāt kriyākāle'navasthānād asattvāc ca /
viṣayaścotpanno'nutpanno vobhayathā'pyālambanatvenāyogyaḥ /
utpannasyā'navasthānād anutpannasya cāsattvāt /
nahi nirādhārālambanakriyā yujyate /
yasyāpi sattāmātreṇa kāryakāraṇabhāva evālambyālambakabhāvas 
tasyāpi vartamānasya vijñānasyālambanaṃ nirarthakaṃ
