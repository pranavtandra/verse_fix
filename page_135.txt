samādhiprajñāsvabhāvāni sasaṃprayogāṇi /
tatroṣmaprakāra uṣmagataṃ /
yathā'raṇinirmathanāt tad dahanasamarthasyāgner utpatīcihnaṃ pūrvam 
ūṣmotpadyate evaṃ sarvakleśendhanadahanasamarthasyāryamārgāgneḥ 
pūrvarūpatvāt tad uṣmagatam ity ucyate /
calakuśalamūlebhyo mūrdhopagamanatvān mūrdhā /
kṣāntiḥ satyārthe kṣamāvattvāt /
tad anyasāsravadharmaśreṣṭhatvāc ca laukikā agradharmā ity ucyante /
teṣāṃ tv agratā tad anantaraṃ darśanamārgotpatteḥ /
te ca kṣaṇikāḥ śeṣāṇi nirvedhabhāgīyāni santānavanti /
tatra mokṣabhāgīyaṃ kuśalamūlaṃ saṃsāraprabhāvanaśaktighātayogena 
veditavyaṃ /
nirvedhabhāgīyam anāsravadharmotpattiśaktyādhānayogeneti //

	{ sapta bodhya:ngāni }
agradharmānantaraṃ darśanamārgotpattes tasya bodhya:ngaprabhāvitatvād 
āha - balānantaraṃ bodhya:ngānīti /
teṣāṃ kathaṃ vyavasthānam iti /
kim arthaṃ vyavasthānam ityabhiprāyaḥ /
ata āha -
	āśrayā:ngaṃ svabhāvā:ngaṃ
iti vistaraḥ /
darśanamārge bodhāva:ngāni bodhya:ngānīti /
adṛṣṭapūrvasya tattvasya tat prathamata ālocanād darśanaṃ 
āryabhūmiprāpakatvān mārga ityato darśanamārga ity ucyate /
nirvedhabhāgīyāvasthāyāṃ hi pelavavastrāntaritānīvopalakṣyante 
satyāni /
darśanamārgāvasthāyāṃ tadapanīta iveti /
tatra tattvābhisambodho bodhinirvikalpaṃ jñānaṃ tatra ṣaṭ smṛṭyādīni 
bodhyanukūlāt tada:ngāni /
dharmapravicayastv avabodhātmakatvād bodhiḥ itareṣāṃ ca sahāyabhāvād 
a:ngatvaṃ /
anyaḥ punar āha - ānantaryamārgagṛhīto dharmapravicayo bodhya:ngaṃ 
vimuktimārgasa:ngṛhītas tu bodhir iti /
tatra bodher āśrā:ngaṃ smṛtiḥ smṛtbalena tasminnālambane'vikṣepaṇāt /
svabhāvā:ngaṃ dharmapravicayaḥ jñānasvabhāvatvād bodheḥ /
niryāṇā:ngaṃ vīryaṃ vīryeṇa
