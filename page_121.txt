[ saṃskṛtā'saṃskṛtārthaḥ }
      idānīṃ saṃskṛtāsaṃskṛtārthaṃ pratipādayann āha -
	saprajñaptisahetukāt /
	nimittātpraśamāt sārthāt paścimaṃ samudāhṛtaṃ //
iti /
kim idaṃ paścimaṃ samudāhṛtaṃ ?
ityata āha - saṃskṛtā'saṃskṛtam iti sarvapaścād udiṣṭatvāt /
tatra saṃskṛtaṃ saprajñaptikāt sahetukāc ca nimittāt asaṃskṛtaṃ punaḥ 
sārthāt praśamād vyavasthāpitaṃ /
prajñaptihetunimittaṃ punar ajñānāt /
tat pradarśayann āha - tatra saprajñaptir nāmakāyādaya iti /
prajñapyete svabhāvaviśeṣau bhāvānāṃ nāmapadakāyābhyām iti 
nāmapadakayau prajñaptiḥ /
vyañjanakāyas tu tad ubhayābhivyañjakatvāt prajñaptiḥ /
hetuḥ punar bījasa:ngṛhītam ālayavijñānam iti /
tac ca sarvasāsravadharmavāsanānvitam adattaphalaṃ 
samudayasatyasa:ngṛhītaṃ /
ata eva dattaphalād vyavacchedārthaṃ bījasa:ngṛhītam ityāha /
nimittaṃ pratiṣṭhādehabhogasa:ngṛhītam ityālayavijñānam iti prakṛtaṃ /
pravṛttivijñānasa:ngṛhītaśca manadugrahavikalpa iti /
tatra pratiṣṭhā bhājanalokaḥ /
tadābhāsam ālayavijñānaṃ /
pratiṣṭhātmanā sa:ngṛhītam iti pratiṣṭhāsa:ngṛhītaṃ /
dehaḥ sendriyaṃ śarīraṃ tadābhāsam ālayavijñānaṃ dehasa:ngṛhītaṃ 
dehātmanopagatam ityarthaḥ /
