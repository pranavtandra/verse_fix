darśitaṃ /
lakṣaṇavijñaptitattvābhyāṃ dharmasāmyaṃ /
viśuddhitattvena bodhisāmyaṃ /

	sattvasāmyaṃ tribhiḥ proktaṃ dharmasāmyaṃ dvayena tu /
	bodhiprajñāsāmyam evam ekaikena pradarśitaṃ //

	{ kauśalyatattvaṃ }
prabhedatattvānantaraṃ kauśalyatattvam uddiṣṭaṃ /
kuśalyatattvaṃ darśanapratipakṣeṇaety uktaṃ /
tatpunaḥ skandhādiṣu kathaṃ daśavidham ātmadarśanam iti na jñāyata 
ityataḥ pṛcchati - katham eṣu skandhādiṣu daśavidham ātmadarśanam iti ?
ata āha -
	ekahetutvabhoktṛtve kartṛtvavaśavartate /
	ādhipatyārthanityatve kleśaśuddhyāśraye'pi ca //
	yigotvā'muktamuktatve ātmadarśnam eṣu hi /
iti /
skandhādiṣv iti sambadhyate /
