vipakṣaprahāṇārthaḥ pakṣaḥ pratipakṣaḥ /
sa sārgaḥ /
tadabhyāso bhāvanā /
avasthā tasyaiva santānenotpattiviśeṣaḥ /
sā punar ekānnaviṃśatiprakārā gotrāvasthādikā /
phalasya lābhaḥ phalaprāptiḥ /
tat punaḥ pañcadaśaprakāraṃ vipākaphalādi /
yānānuttartham iti /
gamyate'neneti yānaṃ /
yānac ca tadānuttaryañceti yānānuttaryaṃ /
tat punastrividhaṃ pratipattyānuttaryādi /
saptamo'rtha iti niyamārtham anukramārthañcāha /
etāvanta evārthā nirdiśyante nāto'nya iti /
anukramaḥ punar ayaṃ lokottarajñānānukūlyārthaḥ /
tathā hi bodhisattvenādhimukticaryābhūmisthitena śīlapratiṣṭhitena 
prathamaṃ sa:nkleśavyavadānakuśalena bhavitavyaṃ /
tataḥ kuśalānāṃ dharmāṇāṃ yasya yad āvaraṇaṃ tasya taj jñeyaṃ /
tasyāprahāṇād vimukter asambhavād avijñānañca prahātuṃ na śakyate 
doṣadarśanāt /
tatas tadāvaraṇād yenālambanena cittaṃ vimucyate tatttvaṃ veditavyaṃ /
tatastenālambhanenatadāvaraṇasya yena prayogena kṣayo bhavati sā 
pratipakṣabhāvanā jñeyā/
tatas tasyāṃ pratipakṣabhāvāyāṃ vipakṣahānyā pratipakṣavṛddhyā 
cā'vasthā jñeyā gotrāvasthādikā /
tato lokottaradharmābhimukhīkaraṇaṃ phalāni srota apatti phalādīni 
veditavyāni /
sarvam apyetat srottaratvācchrāvakādibhiḥ sādhāraṇaṃ bodhisattvānāṃ /
yathoktaṃ sūtre -
sa pravrajitaḥ śrāvakaśikṣācāragocarasamudācāram api śikṣate 
pratyekabuddhaśikṣācāragocarasamudācāram api śikṣate 
bodhisattvaśikṣācāragocarasamudācāram api śikṣate /
bodhisattvānāṃ tvānuttaryam asādhāraṇaṃ /
saptamo'rtha ānuttaryaṃ /
anyastvāha /
sa:nkleśavyavadānalakṣaṇe kauśalotpādanārtham ādau laksaṇaṃ /
tatra yaḥ sa:nkleśas tadāvaraṇaṃ /
yad vyavadānaṃ tat tattvaṃ /
āvaraṇaprahāṇaṃ ca tattvādhigamād bhavatītyata āvaraṇatattvaṃ /
tat prahāṇopāyapradarśanārthaṃ tataḥ saparivāraḥ pratipakṣo mārgaḥ /
tasya mārgasyādimadhyānteṣu mṛdumadhyādhimātraprabhedapradarśanārthaṃ 
tatrāvasthā /
avasthānurūpaṃ phalamāvahatīti tad anantaraṃ phalaṃ /
etac ca sarvaṃ bodhisattvasya śrāvakādisādhāraṇam ity 
asādhāraṇa-mahāyānanayaprakāśanārthaṃ yānānuttaryam iti /
apare punar āhuḥ /
sadasal lakṣaṇajñāpanārthaṃ prathamato lakṣaṇam ākhyātaṃ /
lakṣaṇaṃ jñātvāvaraṇaṃ prahātavyaṃ tattvaṃ ca sākṣāt kartavyam iti tad 
anantaram āvaraṇatattvaṃ /
