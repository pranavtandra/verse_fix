tat punaretad āvaraṇam aprayogādikaṃ /
śubhādau vaśitvaparyantadaśavidhe vastuni /
yatrā'rthe daśa kāraṇāni bhavantīti - utpattyādau prāptiparyante 
yasminn arthe tadāvaraṇaviparyayeṇa prayogādīni 
samādhiparikarmitatvā'vasānāni daśakāraṇāni bhavanti /
atra cotpattyādikāraṇatvā'viśeṣād bhinnasyā'pi pratyekam apy 
abhedanirdeśaḥ /
tad arthādhikāreṇety utpattyyādyarthādhipatyenetyarthaḥ teṣāṃ hi 
śubhādīnām utpattyādikāraṇavibandhaṃ vartamānam utpattyādyāvaraṇam 
ucyate tadyathā kuḍyādikam ālokamāvṛnvat kāryasya cakṣur 
vijñānasyāvaraṇam ucyate /
evaṃ kṛtvaiṣāṃ daśānāṃ dharmāṇāṃ triṃśad āvaraṇaviparyayeṇa triṃśat 
kāraṇaṃ veditavyaṃ /

tatra śubhotpattikāraṇaṃ trīṇi - prayoga āyatanaprayogo yoniśaḥ prayogaś 
ca /
tad yathā cakṣur ādaya ityatraiva dṛṣṭāntaḥ /
cakṣurādayo vijñānasyotpattikāraṇam eva sthityādikāraṇaṃ /
tatrā'prayogādayaḥ prayogādīnāṃ sākṣād vibandhe vartamānāḥ 
paramparayā śubhotpatter api pratibandhakāḥ /
evam utpattyāvaraṇaṃ śubhe draṣṭavyam ityāha tasyotpādanīyatvād iti 
śubhe sthityādyasambhavād utpattyāvaraṇam evaitad iti darśayati /
sthitikāraṇaṃ bodhau /
āśrayaparāvṛttir bodhiḥ āśrayo nirmalatathatā /
atra kāraṇam āha - akopyatvād iti /
yāval lokas tāvad avasthānād akopyatvaṃ, nā'nyathā bhavati vinivartate 
vā /
śrāvakādīnān tu bodhiḥ svasantāna eva kṛtārthatvād vyāvartate /
bodhisattvā hi sarvasattvahitasukhādhyāśayena sambhāreṣu pravartamānā
