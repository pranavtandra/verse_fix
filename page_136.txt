pṛthagjanabhūmisamatikramāt /
anuśaṃsā:ngaṃ prītiḥ tathā kāyacittānugrahāt /
asa:nkleśā:ngaṃ tridhā - prasrabdhisamādhyupekṣāḥ tāsāṃ 
sa:nkleśapratipakṣabhūtatvāt /
athavā'sa:nkleśanidānāśrayasvabhāvabhedāt /
kim arthaṃ punar asa:nkleśā:ngaṃ tridhā deśitam iti /
kenārthena tridhā deśaitam ityabhiprāyaḥ /
ata āha -
	nidānenāśrayeṇeha svabhāvena ca deśitaṃ /
iti /
asaṃkleśasya nidānaṃ prasrabdhiḥ /
asa:nkleśo viśuddhaḥ /
nidānaṃ kāraṇaṃ /
kathaṃ nidānam ityato bravīti - dauṣṭhulyahetukatvāt sa:nkleśasya tasyāś 
ca tat pratipakṣatvād iti /
etad uktaṃ bhavati - sa:nkleśanidānapratipakṣabhūtatvād 
asa:nkleśanidānam iti /
kāyacittākarmaṇyatā dauṣṭhulyaṃ /
tasmin sati cittaikāgratā'sambhavāc cittavikṣepaḥ /
vikṣiptacittasya ca rāgādisa:nkleśaḥ pravartate /
evaṃ dauṣṭhulyaṃ sa:nkleśasya hetuḥ /
tasya dauṣṭhulyasya pratipakṣaḥ prasrabdhiḥ /
katham iti ?
prasrabdhyā kāyacittayoḥ karmaṇyatāpādanāt /
evaṃ prasrabdhirasa:nkleśanidānaṃ vyavasthāpyate /
āśrayaḥ samādhiḥ prasrabdhyanugṛhītakāyacittasya cittaṃ samādhīyate /
samāhite citte yathābhūtaṃ prajānāti /
yathābhūtadarśanāt kleśaḥ prahīyante /
evam asa:nkleśasya samādhirāśrayaḥ /
svabhāva upekṣeti 
sa:nkliṣṭavihārānukūlāyāścittasamatāyāścittāpraśaṭhatāyāścittābhogāvasthitatāyāś 
ca pratipakṣatvāt /
evam upekṣā'sa:nkleśasvabhāvaḥ /
anye tv āhuḥ - prahāṇair mārgair darśanaheye saṃkleśe prahīṇe tat 
sa:nkleśānubandhavimuktā svabhāvaviśuddhā vimuktimārgasaṃgṛhītā 
jñeye suniścitā yā prajñā sātra prajñopekṣety ucyate kṛtakṛtyatvāt 
kṣaye kṣaya ity upekṣākaraṇād iti //4-9//
