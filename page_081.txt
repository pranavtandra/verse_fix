karmavaśitāśrayatvaṃ daśamyā pratividhyatīti sambadhyate /
kartam etad gamyate ?
ata āha - yathecchaṃ nirmāṇaiḥ sattvārthakaraṇād iti /
niścayenā'nena daśamyām bhūmau dharmadhātoḥ karmavaśitāśrayatvaṃ 
pratividhyati tena tathāgatakarma svādhipatyaṃ pratilabhate /
tathā hi sugatatvād daśadiganantalokadhātusthitānāṃ sattvānāṃ 
nirmāṇakāyair yatheṣṭaṃ yasya yathā yasmin kāle yat kartavyaṃ tasya 
tathā tasmin kāle tat karoti /
tasmād asyām bhūmau jñānapāramitā'tiriktatarā bhavati /
dharmaparibhogasattvaparipākaviśeṣayogād iti /
evam āsāṃ yathoktānāṃ bodhisattvabhūmīnāṃ 
daśavidhadharmadhātuprativedhasarvākārapariṣodhanavibhandhabūtaṃ yad 
akliṣṭam ajñānam ekaikāyām api bhūmyām āvaraṇaṃ tat pratipakṣā 
bodhisattvānāṃ daśa bhūmayo vyavasthitāḥ navamyā daśamyāśca bhūmeḥ 
phalena viśeṣaḥ paridīpito nāvaraṇabheda prativedhaviśeṣeṇa tad vihārasya 
kāmapyanirūpyā'vasthāṃ gatatvāt /
sarvatragādyarthaprativedhaprayojanaṃ daśabhūmiṣu daśānām pāramitānāṃ 
viśeṣagatir yathākramaṃ /
tasmād etad evoktaṃ - bhūmyāvaraṇaṃ pāramitāviśeṣagatyāvaraṇam iti /
athavā sūtroktam eva samādhisamāpattyādikam uttarottarabhūmiviśiṣṭam 
sarvākāraṃ nānā'vasānaṃ phalaṃ sarvatragādyarthaprativedhaprayojanam 
iti //2-16//

	{ āvaraṇasamāsaḥ }
prāg uktāni bahuvidhāni sarvāvaraṇāni dvayāvaraṇāntar gatānīti 
pradarśanārtham āha - samāsena punaḥ -
	kleśāvaraṇam ākhyātaṃ jñeyāvaraṇam eva ca /
katham etad gamyate ?
samāsato dvayam eva na punas tṛtīyam apy astīti /
ata āha -
	sarvāñ āvaraṇānīha
iti /
atrāntargatānīti vākyaśeṣaḥ /
sarvāvaraṇāni dvayāvaraṇāntargatāni nānyat tṛtīyam astīti gamyate /
yataś ca sarvāvaraṇānyatraivāntargatānyatasyaiva 
sarvāvaraṇasa:ngrāhakasya kleśajñeyāvaraṇadvayasya kṣayāt 
sarvāvaraṇebhyo mukhtiriṣyate sarvāvaraṇānā prahāṇa mityarthaḥ /
buddhabūmyā varaṇe'nākhyāte katham atraivāntar bhāvaḥ ?
na hy abhidhānād antarbhāvaḥ kin tarhi svabhāvāt /
api ca vyāpyāvaraṇavacanāt tad apy uktam eva /
anyathā ca pradeśāvaraṇam eva bhavatīti /
athavā buddhabhūmyāvaraṇasa:ngrahārtham āha -
	sarvāṇyāvaraṇānīha yat kṣayān muktir iṣyate /
