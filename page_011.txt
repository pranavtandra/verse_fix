abhūtaparikalpo'sti
iti /
na sarvam asat nāpi sadātmakaṃ vijñānapariṇāmāatmanā'stīti kṛtvā /
ye tu manyante yathaiva rūpādayaḥ prakhyāyante tathiavābhūtaparikalpāt 
svabhāvataḥ pṛthag vidyanta iti tān pratyāha 
abhūtasamāropapratiṣedhārthaṃ -
	dvayaṃ tatra na vidyate /
iti /
abhūtaparikalpamātraṃ vidyata ityabhiprāyaḥ /
kenacit dvayor abhāvo vo vnadhyāputravat cheda rūpo gṛhyate /
anyair antarvyāpārapuruṣarahitatā dharmāṇāṃ śūnyate'tyucyate /
ataḥ śūnyatāpavādapratiṣedhārthaṃ bhūtanairātmyakhyāpanārthañcāha 
-
	śūnyatā vidyate tvatra
iti /
yadyabhūtaparikalpe śūnyatāstyevaṃ sati sarvaprāṇinām aprayatnena 
mokṣaprasa:ngaḥ /
nāsyetad yasmāt -
	tasyām api sa vidyate /
na hyaviśodhitāyāṃ śūnyatāyāṃ mokṣo'sti sa:nkliṣṭā ca mahatā yatnena 
viśodhyata iti nāsty ayatnena mokṣaḥ /
atha vā lakṣaṇaṃ sa:nkleśavyavadānābhidhānād anyan nāstītyataḥ 
sa:nkleśavyavadānalakṣaṇapradarśanārthamāha -
	abhūtaparikalpo'sti
iti vistaraḥ /
abhūtaparikalpasvabhāvaḥ saṃkleśo bhrāntilakṣaṇatvāt /
katham etaj jñātavyaṃ bhrāntilakṣaṇam iti ?
yena -
	dvayaṃ tatra na vidyate
svātmanyavidhyamānena grāhyagrāhakākāreṇa pratyānād 
bhrāntisvarūpeṇa jñayate/
edāniṃ vyavadānasvarūpapradarṣana arthamāḥ -
	śūnyatā vidyate tvatra
iti/
śūnyatāsvabhāvo hi vyavadānaṃ dvayābhāvasvabhāvatvāt /
atra ca śūnyatāprabhāvitatvāt mārganirodhayor api grahaṇamṃ veditavyaṃ 
/
sa:nkleśapakṣād eva vyavadānapakṣo mārgayitavyo na punaḥ pṛthaktvam 
asyāstīti pradarśanārtham āha atreti /
yadi dvayaṃ nāsti kathaṃ tasyāṃ vidyamānāyāṃ loko bhrānta ityāha -
	tasyām api sa vidyate
iti /
grāhyagrāhakavikalpo hastyādyākāraśūnyamāyāyām iva 
hastyākārādayaḥ /
abhūtam asmin dvayaṃ parikalpyate'nena vetyabhūtaparikalpaḥ /
abhūtavacanena ca yathā'yaṃ parikalpyate grāhyagrāhakatvena tathā 
nāstīti pradarśayati /
parikalpavacanena tvartho yathā parikalpyate tathārtho na vidhyate iti 
pradarṣayati/
evam asya grāhyagrāhakavinirmuktaṃ lakṣaṇaṃ paridīpitaṃ bhavati /
kaḥ punar asau ?
atītānāgatavartamānā hetuphalabhūtās traidhātukā anādikālikā 
nirvāṇaparyavasānāḥ saṃsārānurūpāścittacaittā 
aviśeṣeṇābhūtaparikalpaḥ /
viśeṣeṇa
