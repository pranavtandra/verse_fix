jñānam idam ajñānaṃ prajahāmītyavadhārya tat prahāṇāya pravartata 
ityayaṃ vyāpṛtau vikalpo'ntaḥ /
jñānasya kṣaṇikatvān na prahāṇavyāpārakāle saṃbhavaḥ /
ato'jñānaprahāṇasya niḥsāmarthyam tiyayam avyāpṛtau vikalpo'ntaḥ /
etasya vikalpadvayāntasya parivarjanārthaṃ tailapradyota dṛṣṭāntaḥ /
tadyathā kāśyapa tailapradyote kṛte tamo'ndhakāraṃ vigacchati na khalu 
punaḥ kāśyapa tailapradyotasyaivaṃ bhavaty ahaṃ sarvam andhakāraṃ 
vidhamāmīti /
api tu khalu tailapradyotaṃ pratītya tamo'ndhakāraṃ vigacchatīti /
evam eva kāśyapa jñāna utpanne'jñānaṃ vigacchati na khalu punaḥ 
kāśyapa jñānasyaivaṃ bhavaty aham ajñānaṃ vidhamāmīti /
api tu khalu jñānaṃ pratītyājñānaṃ vigacchatīti /
etad uktaṃ bhavati - yathā pradīpo vyāpāraśūnyas 
tamo'ndhakāravyudāsābhiprāyarahitaś ca punar yasmin pradeśe kriyate 
tasmāt pradeśāt pradīpasattāmātreṇa sarvatamo'ndhakāraṃ vigacchatīti 
/
evam eva yasmin santāne nirvikalpaṃ jñānam utpadyate tat 
sattāmātrasadbhāvād eva tasmāt santānād ajñānaṃ vigacchati 
jñānasya nirvikalpatvāt kriyā'bhāvārtheti /
evaṃ jñānasya vyāpārābhāvāt sāmarthyābhāvāt sāmarrthyābhāvaḥ 
pradīpavat sattāmātreṇa pratipakṣanirākaraṇād iti /
ajanmatve'pi vikalpo'ntaḥ samakālatve'pi /
vikalpo'nta ityanuvartate /
katham ityata āha - yadi pratipakṣasyānutpattiṃ vā kalpayati yatheyatā 
kālena notpannapūrvaḥ pratipakṣas tathedānīm api notpatsyata iti ayam 
ajanmatve vikalpo'ntaḥ /
sa:nkleśasya vā dīrghakālatvam iti /
na hy ajñāne jñānapratiyoginyavasthita eva jñānam utpattuṃ samarthaṃ /
athāpi kathañcid utpadyeta tathāpi cirakālaviruddhatvād ajñānasya na 
jñānena tat kṣaṇikena vyudāsaḥ śakyate kartuṃ /
tataśca sa:nkleśavyavadānayoḥ samānakālatvam iti /
etasya vikalpadvayāntasya parivarjanārthaṃ dvitīyas 
tailapradyotadṛṣṭānta iti /
tadyathā kāśyapa gṛhe vā layane vā'vacarake vā varṣasahasrasyātyayān 
na kenacit tailapradyotaḥ kuto bhavati /
atha tatra kaścid eva puruṣas tailapradyotaṃ kuryāt /
tat kiṃ manyase kāśyapāpi tu tamo'ndhākārasyaivaṃ bhaved 
varṣasahasrasañcito'haṃ nāham ato vigamiṣyāmīti /
āha - no bhagavan na hi tamo'ndhakārasya śaktir asti tailapradyote kṛte na 
vigantum avaśyaṃ tena vigantavyaṃ /
bhagavān āha - evam eva kāśyapa kalpaśatasahasrasañcito'pi karmakleśa 
ekena yoniśomanaskāreṇa vigacchati /
tailapradyota iti kāśyapa
