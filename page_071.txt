kuśalamūlam abhisaṃskaroti tena mokṣalābhaniścayāt mokṣabhāgīyam ucyate 
tasmiścāropite śraddhādaya indriyākhyāṃ pratilabhante nānyathetyato 
mokṣabhāgīyānām aropaṇam indriyeṣv āvaraṇam uktaṃ /
kim atrāvaraṇaṃ ?
yenopakleśena mokṣabhāgīyaṃ na rocyate /
sā punar bhavasaktir nirvāṇabhītiś ca /
baleṣu teṣām evendriyāṇāndaurbalayaṃ /
āvaraṇam iti prakṛtaṃ /
katham punar indriyāṇān daurbalyam ityata āha - vipakṣavyavakiraṇād iti 
/
āśraddyakausīdyam uṣitasmṛtivikṣepaduṣprajñātmakena 
vipakṣeṇābhibhūtatvād ityarthaḥ /
indriyāṇi hy uṣmagata mūrdhānābhyāṃ prabhāvitāni /
etac ca dvividham api nirvedhabhāgīyān durbalam 
aśraddhyādivipakṣābhibhavāt /
ata eva tasmāt parihāṇisambhavaḥ /
nirjitavipakṣatvāt tu śraddhādīni kṣāntyagradharmāvasthāyāṃ 
balākhyāmpratilabhante /
tasmād yat teṣām eva śraddhādīnām uṣmagatamūrdhānāvasthāyāṃ 
vipakṣā'bhibhavād daurbalyantad balānām āvaraṇaṃ /
tasmin sati balā'vasthā'sambhavād iti /
kim atrāvaraṇaṃ ?
sa evāśraddhyādiko vipakṣaḥ /
bodhya:ngeṣu dṛṣṭidoṣaḥ /
āvaraṇam iti prakṛtaṃ /
bodhir hyatra darśanamārgo'bhipretaḥ /
tasyāstāni 
smṛti-dharmapravicaya-vīrya-prīty-praśrabdhi-samādhy-upekṣātmakāni 
sapta bodhya:ngāni darśanaheyakleśaprahāṇakāla utpadyanta itya:ngānīty 
ucyate /
yadi tu bodheranukūlatvād a:ngāni syur evaṃ smṛtyupasthānādīnām api 
bodhya:ngatvamprasajyeta /
katham idaṃ bodhya:ngeṣu dṛṣṭidoṣa āvaraṇam ityata āha - teṣān 
darśanamārgaprabhāvitatvād iti darśanamārgavyavasthāpitatvād ity 
arthaḥ /
kaḥ punar anyadṛṣṭidoṣaḥ ?
parikalpitāḥ pañca dṛṣṭayo vicikitsā'vidayā tadālambanāṣca rāgādayaḥ 
sasamprayogāḥ kleṣopakleṣāḥ /
nanu vicikitsā'dayopya attrāvaraṇaṃ /
āvaraṇambhavatu /
iha tu pradhānavivakṣayā dṛṣṭidoṣopadeśa iti na teṣām anupadeśo 
virudhyate /
teṣāñca dṛṣṭyādīnām ālayavijñānasanniviṣṭaṃ bījam eva 
darśanamārgapraheyatvāt tad āvaraṇam ucyate /
mārgā:ngeṣu samyag 
dṛṣṭisaṃkalpakarmāntājīvavyāyāmasmṛtisamādhyākhyeṣū dauṣṭhulyadoṣa 
āvaraṇam iti prastutaṃ /
atraiva kāraṇaṃ āha - teṣām mārgā:ngānāṃ 
bhāvanāmārgaprabhāvitatvād iti bhāvanāmārgavyavasthāpitatvād /
kaḥ sa dauṣṭhulyadoṣaḥ ?
vyāsataḥ satkāyāntagrāhadṛṣṭī sahaje sasamprayoge tadālambanā 
viṣayālambanāś ca rāgādayaḥ kleśopakleśāḥ sasamprayogāḥ /
teṣām ālayavijñānasanniviṣṭaṃ yad bījantad dauṣṭhulyaṃ /
sa:nkṣepatas tu darśanaprahātavyād anyā sāsravā tacchaktir 
ālayavijñānasanniviṣṭā dauṣṭhulyaṃ /
tac ca bhāvanāmārgapraheyatvān mārgā:ngeṣv āvaraṇatvena 
vyavasthāpyate /
uktaṃ bodhipakṣeṣvāvaraṇaṃ //2-11//
