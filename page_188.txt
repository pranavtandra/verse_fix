dhāraṇāt pradhāraṇā prathamabhūmipraveśaprāyogikaṃ bhāvanāmayaṃ 
jñānaṃ /
tat punaḥ smṛtyupasthānānyārabhya yāvad balāni /
tasyālambanaṃ pradhāraṇālambanaṃ /
aṣṭamaṃ prathamāyāṃ bhūmau darśanamārgasyeti /
pratyātmam advayasya dharmadhātoḥ sarvatragatvaprativedhaḥ /
athavā darśanaheyānāṃ kleśānām akliṣṭasya cājñānasya 
sarvatragārthaviṣayasya pratyanīko vedhaḥ prativedhaḥ /
sa punar darśanamārgaḥ /
tasyālambanaṃ prativedhālambanaṃ /
navamaṃ bhāvanāmārgasya yāvat saptamyāṃ bhūmāv iti /
tasyaiva dharmadhātuprativedhasyābhyāsād uttarottaraprakarṣaṃ gatiḥ 
pratānatā bhāvanāmārgo yāvat saptamyāṃ bhūmau /
tasya bhāvanāmārgasyālambanaṃ pratānatālambanaṃ /
daśamaṃ tatraiva laukikalokottarasya mārgasyālambanam iti prakṛtaṃ /
prakāraśo dharmādhigamād iti /
tasyām eva bhūmau 
sarvasūtrādidharmanimittaprakārādyabhiniveśavigamamukhena prakāreṇa 
dharmādhigamāt pragama iti /
sa eva bhāvanāmārga ucyate saptamyām eva bhūmau /
pragamasyālambanaṃ pragamālambanaṃ /
ekādaśamaṣṭamyāṃ bhūmau /
aṣṭamyāṃ hi bhūmau sarvanimittābhogamanasikāravigatasya svarasena 
praśaṭhabvahanāt sa eva bhāvanāmārgaḥ /
praśaṭhasyālambanaṃ praśaṭhālambanaṃ /
dvādaśaṃ navamyādibhūmitraye navamyāṃ pratisaṃvijjñānaprakarṣalābhāt 
/
daśamyāṃ karmavaśitāprakarṣalābhāt /
buddhabhūmāv aśeṣakleśajñayāvaraṇaviśuddhiprakarṣālābhāt /
evaṃ navamyādibhūmitraye jñānakarmaviśuddhiprakarṣagamanāt tad 
bhūmivyavasthitasya yadālambanaṃ tat prakarṣālambanam ucyate /
anyaḥ punar āha - mārgaprakarṣasyaiva mṛdumadhyādhimātrabhedaṃ kṛtvā 
prakarṣālambanaṃ tisṛṣu bhūmiṣūktam iti /
tadeva hi prathamadvayaṃ tasyāṃ tasyām avasthāyāṃ tat tadālambananāma 
pratilabhata iti /
etac cālambanaṃ yad uta dharmavyavasthānālambanaṃ 
dharmadhātvālambanañca /
taccaitad ālambanadvayaṃ tasyāṃ tasyāṃ jñānāvasthāyāṃ tat tan nāma 
labhate /
śrutamayajñānāvasthāyāṃ dhāraṇālambanaṃ dhāraṇājñānaviṣayatvāc 
chrutasya /
tena ca dhāraṇājñānaṃ dhāraṇety ucyate /
evaṃ yāvat prakarṣālambanaṃ vaktavyaṃ /
navamyāṃ daśamyāṃ buddhabhūmau ca prakarṣāprāptaṃ lokottaraṃ jñānaṃ 
tat pṛṣṭhalabdhaṃ ca laukikaṃ /
tasyālambanaṃ prakarṣālambanaṃ /
uktam ālambanānuttaryaṃ //

	{ samudāgamānuttaryaṃ }
