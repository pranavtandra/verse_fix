ādiśabdenādattādānādayaḥ saṃgṛhyante /
te vadhādayaḥ kāyavāk-kriyāsvabhāvā dṛṣṭisampannasya na samudācaranti 
darśanamārgeṇa vadhādi-duścaritasamutthāpakakleśaprahāṇāt pṛthagjanasya 
samudācaranti tad aprahāṇāt /
tad aprahāṇaṃ punas tat pratipakṣeṇa mārgānutpannatvāt /
evam aniṣṭādipāratantryānugatam asthānakauśalyaṃ puruṣeśvarādau 
svātantryāgrāhapratipakṣaḥ /
yadi hi svatantraḥ kaścit syān na duścaritena durgatiṃ gacchet /
evam anyatrāpi vācyaṃ //3-19//

	{ indriyārthaḥ }
sthānāsthānārthanirdeśānantaram indriyārtho vaktavya ityata āha - 
indriyaṃ dvāviṃśatividham iti cakṣurindriyaṃ yāvad 
ājñātāvindriyāvasānaṃ /
atra punar ādhipātyarthaṃ indriyārthaḥ /
kasya punaḥ kvādhipatyam ityata āha -
	grahaṇasthānasandhānabhogaśuddhidvayārthataḥ /
iti /
atra ca dvayādhipattyārthata iti yuktau kārikānukūlārtham 
ādhipatyaśabdo lupto draṣṭavyaḥ /
tasmād bhāṣyakāra āha - teṣu tadādhipatyād iti /
grahaṇārtheneti /
arthaśabdo'tra prayojanavācī /
grahaṇaprayojanaṃ nimittaṃ tatrendriyam iti darśayati /
evam anyatrāpi yathāyogaṃ vaktavyaṃ /
teṣvityarthagrahaṇādiṣu ādhipatyam ityadhikaprabhutvam ityādhipatyaṃ 
hetutvam ityarthaḥ /
rūpādayo dharmaparyantā rūpādiviṣayāḥ /
teṣāṃ grahaṇaṃ cakṣurādivijñānaṃ /
tatra cakṣurādīnāṃ ṣaṇṇām ādhipatyaṃ /
cakṣuṣo niravaśeṣarūpaprabhedavijñānāśrayatvād rūpādīnām 
adhikaprabhutvam itīndriyam ucyate /
na tu nīlarūpaṃ pītarūpādivijñānasyālambanaṃ bhavati /
evaṃ śrotrādiṣv api yathāsambhavaṃ vistareṇa vācyaṃ /
sthāne jīvitendriyasyeti /
jīvitendriyaṃ punaḥ karmavāsanāvidhaṃ vipākasantāne'sati 
vicchittipratyaye pūrvpūrvakṣāṇasarūpeṣūttarottarakṣaṇeṣu 
pratisandhisāmarthyaṃ /
sthāne jīvitendriyasyādhipatyam ityatraiva kāraṇam āha - tad 
ādhipatyenā'maraṇād iti /
amaraṇaṃ pūrvakarmākṣiptanikāyasabhāgānuvṛttiḥ /
satyapi sattvasthitikāraṇatve nāhārādīnām indriyatvaṃ /
tat sadbhāve'pyāyuṣaḥ kṣaye'vasthānā'sadbhāvāt /
