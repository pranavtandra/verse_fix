tasya traidhātukacittacaittasvarūpasya tathāprakhyāyamānasya 
bhrāntimātrāstitvān nāpi nāsti /
ādiśabdena marīcisvapnodakacandrādayo dṛṣṭāntā yathāyogaṃ veditavyā 
iti /
nahi marīcisvapnodakacandrādayo yathā strīpumātmanā prakhyāyante 
tathā santi na ca santyeva bhrāntimātrāstitvāt evam artho'pīti 
vistareṇa pratyekaṃ yojayitavyaṃ /
atra cādiśabdena gandharvanagarapratiśrutkādayo veditavyāḥ /
evaṃ yan māyādyupamārthadarśanāt sattvāsattvapakṣayor avisāraṃ cetasaḥ 
avikṣiptatvaṃ paśyati so'visāre'viparyāsaḥ /
kaḥ punar avisāraḥ ?
madhyamā pratipat samādhir ityanye /
yadyevaṃ katham avisāre'viparyāsa ityata āha - tena 
bhāvābhāvayościttasyāvisaraṇād iti /
anenāviparyāsena bodhisattvo 
bhāvābhāvavikalparahitāchūnyatā'lambane'vikṣepaḥ /
tasmād avisāre viparyāsa ucyate /
ayam api saṃvṛttyairvāviparyāso na paramārthataḥ tasya nirvikalpakatvāt 
/
idānīṃ svalakṣaṇe'viparyāsapradarśanārtham āha -
	sarvasya nāmamātratvaṃ sarvakalpāpravṛttaye /
iti vistaraḥ /
sarvam idaṃ nāmamātraṃ yadidaṃ cakṣūrūpaṃ yāvan manodharmā iti /
ato dvādaśāyatanātmakaṃ jñeyaṃ sarvaṃ nāmamātraṃ /
tatra nāmasvabhāvaviśeṣaprajñaptiḥ /
na ca prajñaptyasyārthasya prajñapter vā svabhāvo'sti /
tenātmanā parikalpitatvāt parikalpitasya cāsattvāt /
anyathā nāmanirapekṣayā cakṣurāditvena grahaṇaṃ syāt /
na cāstyavyutpannasa:nketasya grahaṇaṃ /
tasmāc cakṣūrupam ityādyasyātat svabhāvatvān nāmamātram eva 
cakṣūrūpaṃ yāvan manodharmā ityevaṃ yaj jñānaṃ sarvavikalpānāṃ 
pratipakṣeṇa ayaṃ svalakṣaṇe'viparyāsa iti parikalpitadharmamukhena 
sarvadharmeṣu sarvavikalpapravṛttiḥ /
ato nāmamātraprativedhajñānaṃ sarvavikalpapratipakṣaḥ 
svalakṣaṇe'viparyāsa ity uktaṃ /
tatra na jñāyate kiṃ paramārthe svalakṣaṇe āhosvit saṃvṛtāv ityataḥ 
pṛcchati - katam asmin svalakṣaṇa iti /
ata āha -
	paramārthe svalakṣaṇe /
ayam aviparyāsaḥ /
paramārthasya svalakṣaṇasyānabhilāpyatvān na saṃvṛteḥ /
kiṃ kāraṇaṃ ?
saṃvṛttyā yato nedaṃ nāmamātram iti gṛhyate abhilāpadvāreṇa 
dharmasvarūpe sa:nkalpapravṛtteḥ ayam api saṃvṛtyaiva na paramārthato 
nāmamātram iti vikalpanāt /
anena cāviparyāsena bodhisattvaḥ saṃvṛtiparamārthakuśalo bhavati /
