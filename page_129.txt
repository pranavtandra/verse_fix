samyakpradhānāni vīryātmakānīti jñāpitam bhavati /
samyag ebhiḥ kāyavā:nmanāṃsi dhārayantīti samyakpradhānāni //4-2//

	{ catvāraḥ ṛddhipādāḥ }
	karmañatā sthites tatra
iti vistaraḥ /
tatreti tasyāṃ vipakṣapratipakṣāyāpagamavīryabhāvanāyāṃ /
karmaṇyatā cittasthiteḥ sarvārthasamṛddhiyogyatā /
sā ca karmaṇyatā chandavīryacittamīmāṃsādhipatyaprabhavetyata āha - 
karmaṇyatā catvāra ṛddhipādā iti /
sarvasyābhijñādikasyārthasya samṛddhi ṛddhiḥ /
tasyāḥ pratiṣṭhārthena ṛddhipādā ṛddhihetava ityarthaḥ /
ata evāha - sarvārthasamṛddhihetutvād iti /
te punaścandasamādhiḥ prahāṇasaṃskārasamanvāgata ṛddhipādaḥ 
vīryasamādhiḥ prahāṇasaṃskārasamanvāgata ṛddhipādā cittasamādhiḥ 
prahāṇasaṃskārasamanavāgata ṛddhipādaḥ mīmāṃsāsamādhiḥ 
prahāṇasaṃskārasamanvāgata ṛddhipāda iti /
tatra cchandsamādhir yadā chandādhipatyād vīryabhāvanāyāṃ sagauravaṃ 
pravartamānaḥ spṛśati cittasyakāgratāṃ /
vīryasamādhir yadā vīryam ārabhamāṇaḥ sātatyaprayogāt spṛśati 
cittasyaikāgratāṃ /
cittasamādhir yadā pūrvasamādhibījam āgamya citta eva cittaṃ dhārayan 
spṛśati cittasyaikāgratāṃ /
mīmāmsāsamādhir yadālambanapravicito'nuspṛśati cittasyaikāgratām iti 
/
sthitiratra cittasthitiḥ samādhir veditavya iti samādhigrahaṇaṃ 
saṃskṛtalakṣaṇād vyavacchedārthaṃ /
yata ārabdhavīryasya cittakarmaṇyatā catvāra ṛddhipādā bhavanti ataḥ 
samyak prahāṇānantaraṃ ṛddhipādā nirdiṣṭāḥ /
sā punaḥ karmaṇyatā
