kim prayojanā kiṃ hetukā cetyata āha - 
pañcadoṣaprahāṇāyāṣṭaprahāṇasaṃskārabhāvanānvayā videtavyeti /
anvayo hetuḥ //4-3//

	{ pañca doṣāḥ }
te ca pañcadoṣā na vijñāyanta ityataḥ pṛcchati katame pañca doṣā iti /
tat pradarśayann āha -
	kausīdyam avavādasya sammoṣo laya udbhavaḥ /
iti vistaraḥ /
ete hi ṣaḍdoṣā ityata āha - tatra layauddhatyam eko doṣaḥ kriyata iti 
pañca bhavanti /
tatra kausīdyaṃ prayogakāle doṣas tenāprayogāt /
udyuktasyāvavādasammoṣo doṣas tena cittasamādhānabhāvāt /
samāhitasya layauddhatyaṃ doṣastena cittakarmaṇyatā'bhāvāt /
anabhisaṃskāro layauddhatyapraśamanakāle doṣaḥ /
anabhisaṃskāra upekṣā tayā tad dvayāśamanāt /
abhisaṃskāraḥ praśāntau doṣa iti /
abhisaṃskāraścetanā /
layauddhatyavigamāddhi samatāprāptatvāt praśāntam ity ucyate //4-4//

	{ aṣṭau prahāṇasaṃskārāḥ }
