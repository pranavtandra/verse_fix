pṛthyvyeveyam ityādivaṃśādidarśanatulyateti /
etal lokaprasiddhaṃ tattvaṃ /
lokaprasiddhyā tat tathaiveti kṛtvā /
	trayādyuktiprasiddhakaṃ /
iti /
trividham api hi svabhāvam adhikṛtya yuktiḥ kriyata iti /
ataḥ svabhāvatrayeṇa yuktiprasiddhatvaṃ vyavasthāpyate /
tac ca na vijñāyata ityataḥ pradarśayann āha - yat satāṃ 
yuktārthapaṇḍitānām ityādi /
tatra santo ye yuktārthapaṇḍitāḥ /
tārkikā ye mīmāṃsakāḥ /
pūrvasya vyākhyānāśayasya paścān nirdeśoktiḥ /
athavā santo'nveṣakāḥ kiṃ kuśalam iti /
yuktā'rthapaṇḍitā 
apekṣākāryakāraṇopapattidharmatāyukticatuṣṭayā'viruddhārthakṛtakauśalāḥ 
/
tārkikās tuhapohasamarthās tathāgatapravacanasamāśrayeṇā 
viparītaśāstrapraṇetāraḥ /
mīmāṃsakāḥ svaparaśāstrapaṇḍitāḥ śreyodoṣaparīkṣakāḥ /
kecit mīmāṃsakā iti na paṭhanti /
ta evam āhuḥ - satām ityukte ke punaḥ santaḥ ityata āha - 
yuktārthapaṇḍitā iti /
te ca tarkabhūmau bhāvābhūmau ca vyavasthitatvād dvividhāḥ /
tasmāt tārkikā ity ucyanta iti /
teṣāṃ ca kaścil lokaprasiddhānusārītyata āha pramāṇatrayaṃ niśrityeti 
pramāṇatrayāvirodhena /
pramāṇatrayaṃ punaḥ pratyakṣam anumānam āgamaś ca /
tatra pratyakṣaṃ pañcendriyajaṃ mānasaṃ ca sukhaduḥkhādisamvedanaṃ /
trirūpali:ngenānumeuyārthajñānam anumānaṃ /
āptavāgāgamaḥ /
āptāḥ punar anṛtahetuvimuktāḥ /
athavā pratyakṣānumānāgamaprasiddhena hetunā dṛṣṭāntena ca tat 
prasiddhasyaiva sādhyasya paramprati sādhyatvāt pramāṇatrayaṃ niśrityeti 
/
upapattisādhanayuktyā prasiddhaṃ vastu iti /
upapattis trirūpaṃ li:ngaṃ sādhyapratipādanopāyatvāt /
upapatteḥ sādhanam upapattisādhanaṃ trirūpali:ngākhyānaṃ tena tat 
pratipādanāt /
yuktir upāyo yena prayoganiyamena li:ngasya trirūpyam ākhyāyate /
athavopapātir ananyathāsambhavaḥ /
sādhanyuktir avayavasambandhaḥ /
upapatteḥ sādhanayuktir iti vigrahaḥ /
athavopapattir eva sādhanayuktir iti vigrahaḥ /
anyasādhanayuktibhyo vyavacchedārtham upapattigrahaṇaṃ /
evaṃ pramāṇatrayaṃ niśrityopapattisādhana yuktyā yat prasiddhaṃ vastu 
trayāṇāṃ bhāvanām anyatamat tad yuktiprasiddham ucyate /
uktaṃ prasiddhatattvaṃ //

	{ viśuddhigocaratattvaṃ }
