yathākramaṃ pañca śraddhādīnīndriyāṇi veditavyānīti darśayati /
śraddhadhāno hi saṃsāracārakān niḥsaraṇārthaṃ vīryam ārabhate /
ārabdhavīryasya triśikṣāniṣpattiḥ smṛtisamādhiprajñendriyāṇi 
yathākramaṃ /
evaṃ darśanamārgaprayogabhūmāv indriyāṇāṃ prādhānyam iṣyate /
anye tvāhuḥ - na mokṣabhāgīyakuśalamūlaropaṇādhipatyataḥ 
śraddhādīnām indriyatvaṃ /
kiṃ tarhi ropite mokṣabhāgīyakuśalamūle chandādīnām ādhipatyata 
etānīndriyāṇy ucyante /
etad eva pradarśayann āha - chandādhipatyataḥ śraddhāyā indriyatvam iti 
sambadhyate /
prayogo vyāpāraḥ /
asammoṣa ālambanavipraṇāśaḥ /
avisaraṇam avisāraḥ /
pravicayanaṃ pravicayaḥ /
teṣāṃ prayogasammoṣāvisārapravicayānām ādhipatyād 
vīryasmṛtisamādhiprajñākhyaścaittā indriyākhyaṃ bhavantīti /
atra tvetad vaktavyaṃ - ropite mokṣabhāgīye kuśalamūle ropitāvasthātaś 
candādyādhipatyasya śraddhādīnāñca kaścid viśiṣṭasvabhāvo nāsti 
yasmāt tadavasthāyāṃ tad ādhipatyāt teṣām indriyatvam avasthāpyata 
iti //

	{ pañca balāni }
tānyeva śraddhādīni balavanti balānyucyante indriyānantaram uktānīti 
/

	vipakṣasya {hi} saṃlikhāt
iti /
vipakṣasya saṃlikhitatvāt /
etad eva pradarśayann āha - yadā tānīti śraddhādīndriyāṇi /
āśraddhyādibhir vipakṣair iti /
āśraddhyakausīdyavismṛtivikṣepāsamprajanyaiḥ śraddhādivipakṣair na

vyāmiśryanta ityarthaḥ tadā balānyucyante /
