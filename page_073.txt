tad upabhogavaśitvaṃ /
asyobhayasyāvaraṇam āvaraṇaṃ dānapāramitāyās tat phalatvāt /
kim punas tat ?
mātsaryaṃ /
sugatyāvaraṇam iti /
śobhanā gatiḥ sugatiḥ manuṣyo devaś ca /
tasyāvaraṇam āvaraṇaṃ śīlapāramitāyās tat phalatvāt /
tat punar dauśīlyaṃ kutsitaṃ kāya-vāk-karma /
sattvā'parityāga iti /
upakārivad apakārisattvopādānaṃ /
etac ca kṣānti phalaṃ tadāvaraṇaṃ krodhaḥ /
doṣaguṇahānibuddhī iti /
svaparā'pakāraprayuktānām utpannānāṃ prahāṇaṃ 
anutpannānāñcā'nutpādo doṣāṇāṃ hāniḥ /
svaparā'nugrahapravṛttānām utpannānāṃ 
bhūyobhāvo'nutpannānāñcotpādo vṛddhiḥ /
kiṃ punar atrāvaraṇaṃ ?
kausīdyaṃ /
vineyāvatāraṇāvaraṇam iti /
vineyānāṃ śāsane'vatāraṇaṃ 
svargā'pavargalābhakāraṇatvenā'bhyupagamāt /
upaśāntyā gaurava utpadyamāne sattvāḥ sañjalparddhyāpryatnena 
śāsanampratipadyante /
atra ca vikṣepa āvaraṇaṃ /
vimocanāvaraṇam iti /
vimocanaṃ kleśaprahāṇaṃ /
taccā'nuśāsanyā bhavati anuśāsanī ca prajñā nā'nyathetyataḥ 
prajñāpāramitāphalaṃ vimocanaṃ /
tadāvaraṇaṃ kliṣṭaprajñā moho duṣprajñā vā /
dānādyakṣayatvāvaraṇam iti /
dānaśīlādīnām akṣayatve yadāvaraṇam tad upāyapāramitāyā āvaraṇaṃ 
/
tat punar dānādyakṣayatvaṃ kathambhavatītyata āha - bodhipariṇāmanayā 
tad akṣayatvād iti /
mahābodhipariṇataṃ dānādi kuśalamūlaṃ na kṣīyate buddhatvam iva /
tadāvaraṇaṃ mahābodhipariṇāmanā lakṣaṇā'nupāyajñatetyeke /
anupāyakauśalyaṃ trimaṇḍalagrāhas tadāvaraṇam ity anye /
sarvajanmasu kuśalanairantarya pravṛttyāvaraṇam iti /
kuśalanairantaryaṃ pratidinaṃpratikṣaṇaṃ vā /
tac ca praṇidhānapāramitāyāḥ phalaṃ /
katham ityata āha - praṇidhānavaśena tad anukūlopapattiparigrahād iti /
praṇidhāvaśena hi dānādīnāṃ nairantaryapravṛttyā 'nuguṇām upapattiṃ 
parigṛhṇāti /
tasyāvaraṇaṃ dānādyanukūlopapattāv apraṇidhānaṃ /
tasyaiva kuśalasya niyatīkaraṇāvaraṇam iti /
pratidinaṃ pratikṣaṇaṃ vā niyatīkaraṇaṃ niścayaḥ /
atraiva kāraṇam āha - pratisaṃkhyānabhāvanābalābhyāṃ 
vipakṣā'nabhibhavād iti /
atra pratisaṃkhyānabalaṃ dhairyasahitā prajñā /
dhairyaṃ punaḥ prajñaptidharmaḥ /
sahaje vīryasamādhiprajñātraye pratisaṃkhyānatvāt /
bhāvanābalam abhyāsasahitenā'prayatnena
