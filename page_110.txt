{ āyatanārthaḥ }
dhātvarthānantaram āyatanānām artho vaktavyaḥ /
ata āha -
	veditārthaparicchedabhogāyadvārato'paraṃ /
iti /
dhātor aparāṇāṃ hyāyatanapratītyasamutpādādīnāṃ bahutvād 
aparaśabdasya ca sāmānyatvāt praśnaḥ kim aparam iti /
dhātubhyo'nantaram uddiṣṭatvād arthata āyataneṣvevāyam 
aparaśabdo'vatiṣṭhata iti jñāpanārtham āha - āyatanam iti /
tatra veditopabhogāyadvārārthena ṣaḍādhyātmikānyāyatanāni /
arthaparicchedopabhogāyadvārārthena ṣaḍ bāhyānīti /
āyadvārārthaṃ āyatanārthaḥ /
āya āgamaḥ /
taṃ veditārthaparicchedopabhogayos tanvantītyāyatanāni /
veditasaṃjñopabhogāyadvārebhedanādhyātmikabāhyaprabhedaḥ /
veditaṃ hi sukhaduḥkhā'duḥkhā'sukhaṃ /
tadevopabhujyata ity upabhogaḥ /
anubhūyata ityarthaḥ /
tasya vedanopabhogasyāyadvāram ādhyātimkānyevāyatanāny uktāni /
satyapi viṣayāṇāṃ tadāyadvārabhāve cakṣurādīnām āśrayāṇām 
anugrahopaghātatvena tat pravīṇatvena cāsādhāraṇatvenāśrayatvāt sati 
cakṣurādau taddvāreṇa vedanopabhogān nāstīti /
arthaparicchedaḥ saṃjñā arthanimittodgrahaṇātmakatvāt /
saivārthapratipattyatmakatvād viṣayam upabhu:nkte /
tadāyadvārabhāvena bāhyānyāyatanānyuktāni na tvādhyātmikāni 
satyapi tadāyadvārabhāve yasmād 
viṣayābhāve'rthaparicchedopabhogābhāvaḥ satyapy ādhyātmike 
cakṣurādyāyatane /
veditārthaparicchedayoḥ sasamprayogayor ayadvārabhāvatvād 
etānyāyatanāni veditavyāni /
pradhānaṃ hi dvayagrahaṇaṃ /
grahaṇaprādhānyaṃ punaḥ karmaphalopabhogād arthaparicchedātmakatvāc ca 
/
veditārthaparicchedopabhogamukhena bhoktṛvikalpaḥ pravartata ityato 
dvividhenārthenāyatane vyavasthānaṃ kṛtaṃ /
evam āyatanakauśalyaṃ bhoktā''dhyātmikānyāyatanāni bhojanam 
āyatanāni bāhyāni /
nānyo bhoktetyātmani bhoktṛtvagrāho vyāvartate /
yad uktaṃ cakṣuāradyāyatanānām anugrahopaghātāt tad adhiṣṭhāna eva 
bhoktṛtvavyavahāra iti tad ayuktaṃ /
rūpapratiṣedhena cakṣurādīnām āyatanānām apy asattvādātmavad iti
