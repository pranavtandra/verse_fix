vimukti-paripācinyāḥ prajñāyāḥ paripakvabhāvaḥ /
tadyathā agnyādayaḥ pākyādīnām iti /
pākyo hyodanādikas taṇḍulādivilakṣaṇas tat santānāntarbhūtaḥ /
tat kāraṇaṃ cāgnyādi /
viśleṣakāraṇam anāvaraṇe /
darśanamārgapraheyādāvaraṇāc cheṣam āvaraṇam anena prahīyata 
ityanāvaraṇaṃ bhāvanāmārgaḥ /
tasya trīṇi kāraṇāni - sahajadauṣṭhulyopaghāto vīryam apramādaś ca /
teṣāñca viśleṣakāraṇapratipakṣatvāt /
ata evāha - viśleṣāvaraṇam anāvaraṇe /
etad api viśleṣakāraṇapratipakṣabhāvenaivāvaraṇaṃ /
atraiva kāraṇam āha - tasyāvaraṇaviṣaṃyogatvād iti /
visaṃyujyate'neneti viśleṣaḥ /
pariṇatikāraṇaṃ natau /
tasya mahābodhau kuśalamūlaprayojakalakṣaṇatvāt /
tat punar bhavabhogavaimukhyaṃ līnacittatā'bhāvaś ca /
tataś ca pariṇatyāvaraṇaṃ pariṇatikāraṇavibandhena draṣṭavyaṃ /
kiṃ kāraṇaṃ ?
nater mahābodhau cittapariṇatilakṣaṇatvād iti /
sampratyayakāraṇam atrāse /
tac ca pudgalasambhāvanā dharmā'dhimuktir ābhiprāyikārthā'dhigamaś ca 
/
ebhir dharmaiḥ sampratyayo bhavatītyataś ca sampratyayakāraṇavibandhena 
sampratyayāvaraṇamatrāse /
atraiva kāraṇam āha - asampratyayena trasanād iti /
asampratyayena hi dharmasya trasyati /
sampratyāyanakaraṇamamatsaritve /
tat punaḥ saddharme gauravaṃ lābhasatkārapūjādinirapekṣatāa sattveṣu ca 
kāruṇyaṃ /
sampratyayāyanakāraṇaviruddhatvāc ca sampratyāyanāvaraṇam amatsaritve 
'vasthāpyate /
atraiva kāraṇam āha - dharmā'matsaritvena parasampratyāyanāditi /
saddharme'gauravādito dharme mātsarye karoti /
dharmamātsaryeṇa ca pareṣāṃ dharmaṃ na deśayati /
ataḥ sampratyāyanāvaraṇam amatsaritva iti /
vaśitvasya trīṇi kāraṇāni - dharmavyasanasaṃvartanīyakarmā'bhāvo 
bahuśrutatvaṃ samādhiparikarmitatvañca /
tadviruddhatvāc cu tavyasanādikaṃ vaśitva āvaraṇaṃ vyavasthāpyate /
atraiva kāraṇam āha - tasya vaśitvasya vibhutvaprāptilakṣaṇatvād iti /
kecit tu śubhādīnām evam anukrame kāraṇam āhuḥ /
kuśaladharmopaṣṭabdhasatatir bodhiḥ prāptavyā /
tata āditaḥ śubham uktaṃ /
tad anantaraṃ bodhir uktā /
bodhicittañcāgamyaitad ubhayasampādane samartho bhavati ā bodher 
bodhicittasya sarvakuśaladharmādhārabhūtatvād atas tad anantaraṃ 
bodhicittam uktaṃ /
utpāditabodhicittasya ṣaṭpāramitāsu prayogaḥ kujanaduṣṭajanābhyāñca 
parihāraḥ /
evam asya bodhisattvabhāvaḥ prakāśībhavati /
