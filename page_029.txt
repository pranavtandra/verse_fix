ropaṇāt kliśyate jagad iti sambadhyate /
kena ropaṇāt kva kasya vetyata āha saṃskārair vijñāne karmavāsanāyā 
iti /
tatra saṃskārā yat kāyikaṃ karma vācikaṃ mānasaṃ ca 
puṇyāpuṇyāniñjyasvarūpaṃ tat punar bhavam abhisaṃskarotīti saṃskāraḥ /
apraṇihitaṃ rohayatītyarthaḥ /
tadeva ca saṃskāraśabdenocyate na tu sarvaṃ /
taccābidyādhipatyāt punar bhavam ākṣeptuṃ samarthaṃ bhavati na 
sattāmātreṇetyavidyāpratyayāḥ saṃskārā ity ucyate /
tathāhi tāny anutpannavidyasya punarbhavaṃ saṃskurvanti notpannavidyasya /
na cādhipatyamātreṇaivāvidyā saṃskārāṇāṃ pratyaya ucyate sāmānyena 
sā samutthānapratyayopi /
avidhyā hi samprayuktvādhyathā sarvakleṣānāṃ sāmānyena 
samutyānapratyayastayā tatsamuttyā nāṃ cetanānām api /
api saṃskārāṇāṃ bhavabhogabhedaprārthanānvayānāṃ samutthāne tat 
sahabhūtā'vidyā sāmānyapratyayaḥ /
āniñjyānām api tad bhūminiḥsaranadṛṣṭisamutthānānāṃ tat 
sahabhūtā'vidyā sāmānyapratyaya iti saṃskārāṇāṃ pratyaya uktaḥ /
vijñāna iti atrālayavijñānam abhipretaṃ na tu pravṛttivijñānaṃ /
vyuparatapravṛttivijñānasyotpattisthānanayanā'sambhavāt kuśalakliṣṭayoś 
ca samavadhānābhāvāt saṃskārais tatra karmavāsanāyāḥ 
pratiṣṭhāpanābhāvāt /
karmavāsanāyā iti /
kā punar iyaṃ karmavāsanā nāma ?
anāgatasya janmano bījaṃ hetubhāvaḥ /
tad yathā brīhiphalaṃ bhūmibhasmapurīṣādipratyayaviśeṣam apekṣya 
pariṇamad a:nkurotpattau hetubhūtaṃ bījam iti /
pratiṣṭhāpanād iti ropaṇam eva paryāyāntareṇācaṣṭe /
pratiṣṭhāpanaṃ punaratra vijñānasantāne tat prathamatas tad bījasyāpy 
āyanaṃ /
nahi kasyacit sāsravasya dharmasyānāsravasyevāpūrvaṃ bījam ādhīyate 
/
evaṃ saṃskārair vijñāne punarbhāvabījādhānāj jagat kliśyate /
nayanāj jagat kliśyata iti /
kena nayanāt kva kasya veti na vijñāyata ityāha vijñānenotpatti 
sthānasaṃpreṣaṇād iti /
karmaparibhāvitena vijñānena santānavṛttyā cyutisthānād 
upapattisthāne punar bhavabījabhūtāyā vāsanāyāḥ prāpaṇaṃ nayanaṃ /
tathā ca sati na pratisandhivijñānaṃ saṃskārapratyayaṃ 
pūrvakālabhavavijñānam eva
