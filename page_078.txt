śrutasyā'gratāṃ viditveti vistaraḥ /
teneti pratividhhadharmadhātunā /
śrutasyeti śrutasūtrādidharmasya tasyā'gratā viditveti nirvikalpena 
jñānena tathā pratyātmaṃ jñātvetyarthaḥ /
tad artham iti buddhavacanaśravaṇārthaṃ /
trisāhasramahāsāhasrapramāṇāyām apy agnikaṇḍāyām ātmānaṃ prakṣiped 
iti /
uktaṃ hi sūtre - sa ced idaṃ kaścidevaṃ brūyāt - evam ahaṃ tubhyam idaṃ 
dharmapadaṃ samyak sambuddhopanītaṃ bodhisattvacaryāpariśodhanaṃ 
saṃśrāvayeyaṃ sa cet tvaṃ mahatyām agnikhaḍāyāṃ samprajvalitāyām 
ekajvālībhūtāyām ātmānaṃ prapātayeḥ mahāntaṃ ca 
duḥkhavedanopakramaṃ svaśarīreṇopādadyāḥ /
tasyaivaṃ bhavati - utsahe'ham ekasyāpi dharmapadasya 
samyaksambuddhopanītasya bodhisattvacaryāpariśodhanasyārthāya 
trisāhasramahāsāhasralokadhātāv agniparipūrṇe brahmalokād ātmānam 
utsraṣṭuṃ /
kim punaḥ prākṛtāyām agnikhaḍāyāṃ /
api tu khalu punaḥ sarvair nirayāpāyaduḥkhasaṃvāsair apy asmābhis 
sarvabuddhadharmāḥ paryeṣitavyāḥ kiṃ punar manuṣyaduḥkhasaṃvāsair iti /
ata evāsyāmbhūmau kṣāntipāramitā'tiriktatarā bhavati śrutārthaṃ 
sarvakleśasahanādibhir apy abhedyatvāt /
caturthyā niṣparigrahārthaṃ dharmadhātoḥ pratividhyatīti prakṛtaṃ /
caturthyam bhūmau samatvād anātmīyatvaṃ dharmadhātoḥ pratividhyati 
yasmān na dharmadhātoḥ kasyacit samatāstīti /
athavā sarvaviparyāsā'viṣayatvān na kasyacit parigrahaḥ /
svayam aviṣayatvān nā'tra kaścit parigrahaḥ /
yasmād dharmatṛṣṇā'pi vyāvartata ityāgamavacanaṃ /
evam arciṣmatyāṃ niṣprapañcena jñānena dharmadhātvadhigamād 
bodhipakṣadharmaprātpiḥ /
bodhisattvasya yā'pi sūtrādidharmatṛṣṇā pūrvam āsīt sā'pi 
vyāvartate /
dhyānasamādhisamāpattir dharmatṛṣṇetyanye /
tasmāt tasyābhūmau vīryapāramitā'tiriktatarā bhavati /
abhīkṣṇaṃ vā:nmanojalpasamatikrānte bodhipakṣair viharaṇāt /
pañcamyā santānābhedārthaṃ /
dharmadhātoḥ pratividhyatīty adhikṛtaṃ /
atītā'nāgatavartamānānāṃ sarvabuddhānāṃ paramparato 
bodhisattvānāñca santānair ātmanaḥ
