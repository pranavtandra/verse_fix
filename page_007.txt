vā /
ādita iti prathamata iti /
śāstraṃ vivaraṇaṃ /
taccharīraṃ saṃkṣepaḥ piṇḍārtho vāśrayārthena vā śarīraṃ /
yathā hi bāhyādhyātmikāyatanāśrayaḥ kāyaḥ śarīram ity ucyate evaṃ 
yānarthān niścitya śāstraṃ pravartate te'rthās tasya śarīraṃ /
te ca saptyārthā lakṣaṇādayaḥ /
vyavasthāpyata iti prajñāpyate ākhyāyata ityarthaḥ /
nanu śāstrāvabhodhenaiva taccharīraṃ vijñāsyata iti nirarthakam ādau 
tad vyavasthāpanaṃ /
na nirarthakaṃ śiṣyā'nugrahārthatvāt /
gatārtho hi śiṣyaḥ sukhaṃ vistaram ucyamānaṃ pratipādyate /
dṛṣṭā bhūmir niḥśa:nkam aśvavāhaneneva nānyathā /
ityete saptārthā hyasmin śāstra upadiśyante /
itiḥ śāstraśarīrasamāptyarthaḥ /
eta iti ye lakṣaṇādaya uddiṣṭāḥ /
sapteti saṃkhyāparyantādhigamārtham upādānaṃ /
arthyanta ityarthā adhigamyanta iti yāvat /
asmin śāstra iti madhyāntavibhāgasaṃjñake /
upadiśyanta ityākhyāyante viniściyante vā /
yad uta iti tathā'rthapratipādananipātaḥ /
lakṣaṇam iti /
lakṣyante'neneti lakṣaṇaṃ /
tac ca dvividhaṃ sa:nkleśalakṣaṇaṃ vyavadānalakṣaṇāñca /
tatra sa:nkleśalakṣaṇaṃ navavidhaṃ /
	abhūtaparikalpo'sti
ityārabhya yāvat
	saptadhā'bhūtakalpanāt
iti /
śeṣeṇārdhena vyavadānalakṣaṇam ākhyātaṃ /
yadi lakṣyante'neneti laksaṇam ityevaṃ sati lakṣaṇasya 
sa:nkleśavyavadānabhedo bhaved iti naitad evaṃ /
svabhāva eva hi bhāvānāṃ lakṣaṇaṃ /
tad yathā pṛthivīdhātuḥ kharalakṣaṇo na ca kharatvāt pṛthivīdhātuḥ 
pṛthagastīti /
atha vā lakṣyate tad iti lakṣaṇaṃ /
evaṃ hi sa:nkleśo vyavadānañca sa:nkleśavyavadānātmatvena lakṣyata iti 
lakṣaṇaṃ /
atha vā dvayoḥ sa:nkleśavyavadānayor lakṣaṇaṃ dvividhaṃ svalakṣaṇañca 
sāmānyalakṣaṇañca /
āvaraṇam iti /
āvṛṇoti kuśalān dharmānāvriyante'nena kuśaladharmā utpattivibandhāt 
ityāvaraṇaṃ /
tat punas tripañcāśat prakāraṃ /
tattvam iti /
tadevedaṃ tat tasya bhāvas tattvam aviparyāsa ityarthaḥ /
tac ca daśa prakāraṃ /
