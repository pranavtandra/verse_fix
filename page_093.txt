anye tu tathatāprativedhena kleśaparikalpitamaladvayaśāntir iti 
malaśāntistathatety ucyate dvayanirodhaḥ prakṛtinirodhaśceti vacanāt /
prakṛtir iti tathatā prakṛtiprabhāsvaratvāc cittasyeti /
atas tathatālambanādyo nirodhaḥ sa prakṛtinirodha ity ucyate /
niruddhamātraṃ pratisaṃkhyānirodhaḥ /
nirodhyate'syām iti nirodhastathatetyapare /
nirodhasya punar anutpādātmakatvāt sarve svabhāvānutpattyādikaṃ 
saṃkṣipya nirodhasatyam ity ucyate /
ukto nirodhasatyasya mūlatattve'ntarbhāvaḥ /
tad anantaraṃ tu mārgasatyam antarbhāvayitavyam ityataḥ pṛcchati - 
mārgasatyaṃ trividhamūlatattve kathaṃ vyavasthāpyata ityata -
	parijñāyāṃ prahāṇe ca
ityādi /
parikalpitasya parijñāna iti /
parikalpitato'tyantam asanneveti tasya parijñānam eva na prahāṇaṃ /
na hyasataḥ prahāṇaṃ yujyate /
paratantrasya parijñāne prahāṇe ca /
paratantro hi yathā khyāti tathā'sattvaṃ vijñeyaṃ na tu 
sarvātmatvenāsattvaṃ kalpitavat /
karmakleśayor bhāvātmatvāt prahātavyaś ca /
pariniṣpannasya praijñāne prāptisākṣātkaraṇe ceti /
pariniṣpanno hi bhāvābhāvavinirmuktalakṣaṇam ataḥ parijñeyaḥ 
āśrayaparāvṛttitāyāś ca /
āśayaparāvṛttitā vinirmuktadharmakāyatvena sākṣātkartavyeti śūnyatā 
parijñeyā /
nirodhaḥ sākṣātkartavyaḥ prāptisākṣātkriyayā /
ubhayaṃ cobhathetyanye /
dvividhā hi parijñā - jñānaparijñā prahāṇaparijñā ca /
tatra jñānaparijñāyaparijñeyaṃ prathamaṃ tṛtīyañca /
dvitīyaṃ tūbhayaparijñeyaṃ /
sākṣātkriyāpi dvidhā - jñānasākṣātkriyā prāptisākṣātkriyā ceti 
/
tatra jñānasākṣātkriyayā trayam api sākṣātkartavyaṃ /
prāptisākṣātkriyayā tu tṛtīyam eva /
evam atra mūlatattve parijñā prahāṇasākṣātkriyāyāṃ mārgasatya 
vyavasthānaṃ veditavyaṃ na tu tatrāntarbhāvāt /
atra ca satyacatuṣṭayena yaḥ sa:nkleśo yataś ca tayoś ca viśuddhir yā 
yataśca tat paridīpitaṃ /
anukramas tveṣāṃ yathā'bhisamayam iti /
	{ audārikasūkṣmatattvaṃ }
