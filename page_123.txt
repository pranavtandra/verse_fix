anekatvābhisaṃkṣepaparicchedārtha āditaḥ /
iti vistareṇa sarvatra vācyaṃ //3-22//

	{ tattvapiṇḍārthaḥ }
tattvasya piṇḍārthaḥ /
samāsato dvividhaṃ tattvaṃ /
vistareṇa daśavidham uktaṃ /
ataḥ samāsato dvividham ityāha - ādarśatattvaṃ dṛśyatattvañca /
tatra mūlatattvam ādarśatattvaṃ /
tat punas trayaḥ svabhāvāḥ /
kim arthaṃ punas tad ādarśatattvam ityucyate ?
śeṣāṇāṃ lakṣaṇatattvādīnāṃ tatra darśanāt /
dṛśyatattvaṃ yat lakṣaṇatattvādikaṃ pūrvanirdiṣṭaṃ navavidhaṃ /
kimarthaṃ tad dṛṣyatattvam ity ucyate ?
tasya mūlatattve darśanāt nirabhimānadṛśyatattvaṃ lakṣaṇatattvaṃ /
tena hi nirabhimāno bhavati /
pudgaladharmayor grāhyagrāhakayor bhāvābhāvayoś 
cā'samāropānapavādaviparyāsa dṛśyatattvam aviparyāsalakṣaṇaṃ /
taddhi nityādiviparyāsapratipakṣabhūtaṃ /
śrāvakayānaniryāṇadṛśyatattvaṃ phalahetumayaṃ tattvaṃ /
caturāryasatya prativedhabhāvanayā śrāvakaniryāṇaṃ /
mahāyānaniryāṇadrśyatattvaṃ sūkṣmaudārikatattvaṃ /
audārikena saṃvṛtisatyena sattvaparipācanāt sūkṣmeṇa paramārthasatyena 
vimocanāt mahāyānena niryāti /
paravādinigrahadṛśyatattvaṃ prasiddhatattvaṃ /
katham ityata āha - dṛṣṭānasanniśrayeṇa yuktyā nigrahād iti /
nigrahe lokaprasiddho dṛṣṭāntaḥ /
yuktiprasiddhaśabdāśrayād yuktiḥ /
mahāyānābhidyotanadṛśyatattvaṃ viśuddhijñānagocaratattvaṃ /
