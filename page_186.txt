āryasyaitat prajñendriyasyādhivacanaṃ /
tamo'ndhakāra iti śabalasyaitat karmakleśasyādhivacanam iti /
nidarśanena pratipakṣasyānutpattiṃ pariharati pratyayapratibuddhatvāt tad 
utpatteḥ sa:nkleśatulyakālatāñca /
kṛṣṇapakṣasya durbalatvāt balavattvāñca tat pratipakṣasya 
tamo'ndhakārapradīpavad iti /
uktāntadvayavarjane pratipattiḥ //

	{ viśīṣṭā cā'viśiṣṭā ca pratipattiḥ }
antadvayavarjanapratipatter anantaraṃ viśiṣṭā cāviśiṣṭā ca pratipattir 
uddiṣṭetyato'ntadvayavarjanapratipattinirdeśānantaraṃ pṛcchati - viśiṣṭā 
cāviśiṣṭā ca pratipattiḥ kathameti /
ata āha -
	viśiṣṭā cāviśiṣṭā ca jñeyā daśasu bhūmiṣu //
tāsu daśasu bhūmiṣv ekaikaśo dviprakārāpi pratipattiḥ /
katham ityata āha - yasyāṃ bhūmau yā pāramitā'tiriktatarā bhavatīti 
vistaraḥ /
yasyāṃ bodhisattvabhūmau vyavasthito boshisattvo yasyāṃ pāramitāyām 
ādhikyena prayujyate sā tasyāṃ bodhisattvabhūmāv itarābhyaḥ 
pāramitā'bhyo'tiriktatarā bhavatīti viśiṣṭocyate /
tadyathā prathamāyāṃ bhūmau dānapāramitātiriktatarā bhavatīti /
tatra pratipattir viśiṣṭocyate /
evaṃ yāvad daśamyāṃ jñānapāramiteti /
aviśeṣeṇa śeṣāsu sarvāsu pāramitāsu tasyāṃ bhūmau samudāgacchatīti 
tāsu pratipattir aviśiṣṭocyate /
yathoktaṃ pramuditāyāṃ bhūmau dānapāramitā'tiriktatarā bhavatīti /
na ca śeṣāsu na samudāgacchati yathābalaṃ yathābhājamānam iti /
evaṃ śeṣāsu bhūmiṣu yathāsūtraṃ nirdeśo veditavyaḥ /
uktaṃ pratipattyānuttaryaṃ //5-27//
