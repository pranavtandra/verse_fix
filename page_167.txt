idānīṃ sāmānyalakṣaṇe'viparyāsapradarśanāyāha -
	dharmadhātuvinirmukto yasmād dharmo na vidyate /
iti vistaraḥ /
na hi dharmanairātmyena vinā kaścid rūpyarūpī vā dharmo vidyate /
sarva dharmāṇām anabhilāpyādvayaprakṛtatvād grāhagrāhakabhāvena 
rahitatvāt /
tasmād dharmadhātuḥ sarvadharmāṇāṃ sāmānyaṃ lakṣaṇam iti /
sarvadharmeṣv apṛthaktvena tat sat sāmānyalakṣaṇam ucyate /
yadevaṃ jñānamayaṃ sāmānyalakṣaṇo'viparyāsaḥ saṃvṛttyā na tu 
paramārthataḥ /
tasya nirvikalpatvāt /
tad anena cāviparyāsena bodhisattvo viśuddhyālambanakuśalo bhavati 
nirvikalpajñānaviṣayatvāt /
viśeṣaḥ sāmānyalakṣaṇasvalaksaṇayoḥ ko vidyate ?
abhilāpasvabhāvasamāropitatvasya pratipakṣeṇānabhilāpyasvabhāvatā 
paramārthalakṣaṇaṃ /
parikalpitasvabhāva-nairātmyaṃ punaḥ sarvadharmāṇāṃ prakṛtir iti 
sāmānyalakṣaṇaṃ /
evaṃ darśanabhedavaśād viśeṣo bhavati na tv arthād iti /
idānīṃ dharmadhātv aviśuddhiviśuddhāv aviparyāsapradarśanārtham āha -
	viparyasta manaskārāvihāniparihāṇitaḥ /
iti vistaraḥ /
viparyasta manaskāraprahāṇaṃ tasya dharmadhātor aviśuddhiḥ tat prahāṇaṃ 
viśuddhir iti /
tatra viparyasta manaskāra ālayavijñānahetur viparyayavikalpaḥ /
tasyāprahāṇaṃ dharmadhātor aviśuddhiḥ prahāṇaṃ viśuddhiḥ /
yadevaṃ jñānamayam aviśuddhau viśuddhau cāviparyāso yathākramam iti /
viparyasta manaskārāprahāṇaṃ tasya dharmadhātor aviśuddhir iti 
dharmadhātor aviśuddhāvaviparyāsaḥ /
tasya prahāṇaṃ viśuddhir iti viśuddhāv aviparyāsaḥ /
kiṃ kāraṇaṃ ?
dharmadhātor āgantukenopakleśenopakliṣṭatvāt /
ayam api saṃvṛttyaivāviparyāso na paramārthataḥ /
anena cāviparyāsena bodhisattvo'prahīṇāvaraṇaṃ dharmadhātoḥ 
prahāṇañca prajānāti /
idānīṃ dharmadhātor aviśuddher 
viśuddheścāgantukatve'viparyāsapradarśanārtham āha -
	dharmadhātor viśuddhatvāt prakṛtyā vyomavat punaḥ /
iti vistaraḥ /
dharmadhātoḥ punar ākāśavat prakṛtiviśuddhitatvād dvayam 
apyetadāgantukam aviśuddhir viśuddhiś ca paścād iti /
tatra dharmadhātorabhūtavikalpopakliṣṭatvād aśuddhir uktā /
paścāc ca pratipakṣodayena tat prahāṇād viśuddhiḥ /
etad dvayam api dharmadhātāv āgantukaṃ /
kiṃ kāraṇaṃ ?
ākāśavad dharmadhātoḥ prakṛtiviśuddhatvāt /
prakṛtiviśuddhasyāviśuddhir āgantukī bhaved viśuddhes tu nityaviśuddhitaḥ 
katham āgantukībhāva iti ?
yā malavigamatātmatayā viśuddhiḥ sātra vivakṣiteti nāyaṃ doṣaḥ /
ata evoktaṃ viśuddhiś ca paścād iti /
yadevaṃ dharmadhātor jñānam etad dharmadhātor viśuddhi viśuddhyor 
āgantukatve'viparyāsaḥ /
ayam api saṃvṛttyā na tu paramārthataḥ /
anena cāviparyāsena bodhisattvo dharmadhātoḥ 
sa:nkleśavyavadānāvasthayoḥ kṛtrimatvaṃ prajānātīti //
