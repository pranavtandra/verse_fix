samudāgamaḥ /
sa ca tato'nyaviśiṣṭasamudāgamābhāvād anuttara ity ucyate /
tato'tra tasmāt samudāgamād anuyaviśiṣṭābhāvavastutvam ānuttaryaṃ /
atra punaḥ pratipattir bodhisattvānāṃ sādhāra saphalā ca mahāyānam 
ityuktaṃ bhavati /
etac ca trividhenānuttaryeṇānantarokte nānuttaryam ity ucyate /
atha vā mahāyānaṃ sarvavikalpaprapañcavigataṃ jñānaṃ sasamprayogaṃ 
sasambhāraṃ saprayogaṃ sapṛṣṭhalabdhajñānaṃ saviśiṣṭaguṇaṃ sāalambanaṃ 
saphalañca /
tatrālambanaṃ tathatā /
phalam apratiṣṭhitaṃ nirvāṇam iti /
asya ca mahāyānasyābhidhāyikā yā sūtrādideśanāpi mahāyānaṃ 
bhavantīti grantho'pyadhikāra ivopacaryate /
yathā kaṃsavadhā maṇiharaṇañca /
tac ca saptavidhamahattvayogān mahat /
yāntyanena prāpnuvantyapratiṣṭhitaṃ nirvāṇam iti yānaṃ /
saptavidhaṃ punar mahattvam 
ālambanapratipattijñānavīryopāyaprāptikarmamahattvaṃ /
tatrālambanamahattvaṃ śatasāhasrikādyanantasūtradharmālambanād 
bodhisattvamārgasya /
pratipattimahattvaṃ sakalasvaparārthapratipattitaḥ /
jñānamahattvaṃ pudgaladharmanairātmyajñānāt /
vīryamahattvaṃ triṣu kalpasaṃkhyeyeṣv anekaduṣkaraśatasahasraprayogāt /
upāyakauśalyamahattvaṃ saṃsāranirvāṇapratiṣṭhānāt /
prāptimahattvaṃ 
balavaiśāradyāveṇika-uddhadharmādy-prameya-saṃkhyeyaguṇādhigamāt /
karmamahattvaṃ yāvat saṃsārastāvad bodhyabhisandarśanena 
buddhakāryānuṣṭhānāt /
etad eva mahāyānam anuttaraṃ tad anyasya viśiṣṭayānasyābhāvāt /
tadabhāva ānuttaryaṃ /
tat punaḥ pratipattyālambanasamudāgamāḥ /
tathāhi tadyogān mahāyānam anuttaram ucyate /
pratipattirevānuttaryaṃ pratipattyānuttaryaṃ /
evam ālambanānuttaryaṃ samudāgamānuttaryañca veditavyaṃ /
etad ānuttaryaṃ yathākramaṃ bodhisattvamārgaṃ tad adhiṣṭhānaṃ tat 
phalañcādhikṛtya vyavasthāpyate /
etāvac ca buddhavacanenocyate - yaduta pratipattir ālambanaṃ tat 
phalañceti //

	{ pratipattyānuttaryaṃ }
ālambanapūrvikā pratipattir ityālambanapūrvanirdeśanayena pratipattiḥ 
pūrvam uktā bodhisattvānāṃ pratipattisāratāsandarśanārthaṃ /
tatra kathaṃ pratipattyānuttaryam ityata āha - pratipattyānuttaryaṃ 
daśapāramitāpratipattito veditavyam iti /
pāramitāpratipattiḥ
