tāni balān nānātvena tat praveśatat pṛṣṭalabdhayor abhimukhībhavanti 
yāvat ṣaṣṭhī bhūmiḥ /
saptamyāntu bhūmau sarvanimittaikarasākāreṇa jñanena dharmadhātor 
nimitttābhāvatvaṃ pratividhyatīti tāni tāni nimittāni na 
samudācaranti /
tasmāt tasyāṃ bhūmāv upāyaparamitā'tiriktatarā bhavati yataḥ 
sarvanimittāny animittatvena pratividhyati nimittakṛtaṃ ca vyavahāraṃ na 
virodhayati /
tat prativedhāc ca tasyām eva vaśitvaṃ pratilabhate /
aṣṭamyā'hīnadhikārthaṃ /
dharmadhātoḥ pratividhyatīti sambadhyate /
dharmadhātoḥ prakṛtipariśuddhatvāt samalā'malā'vasthayor 
viśeṣā'bhāvāt /
ata evā'sau nityaṃ tathaiveti kṛtvā tathatetyucyate /
nimittā'bhāve kvacid api hīnatā nāsti /
sanimittatve kvacid apyadhikatvaṃ nāsti /
tathā hi nimittam evā'nimittam iti kiñcid 
anutpattikadharmakṣāntilābhād ityatraiva kāraṇaṃ /
aṣṭamyāṃ hi bhūmau bodhisattvo'nutpattikadharmakṣāntipratilambhād 
dharmadhātor abhyuccayā'bhāvatvam apacayā'bhāvatvaṃ caivaṃ 
pratividhyati /
sa:nkleśe vyavadāne vā kasyacid dharmasya 
hānivṛddhyadarśanādityapūrvadharmā'nutpādāt /
caturthā vaśitā - nirvikalpavaśitā kṣetrapariśuddhivaśitā jñānavaśitā 
karmavaśitā ca /
tatra nirvikalpavaśitāśrayatvaṃ dharmadhātor veditavyaṃ 
anābhogenasarvanimittasamudācārā'bhāvāt /
saptamyām api nimittasamudācārābhāvaḥ sambhavaty abhisaṃskāreṇa /
iha tv anabhisaṃskāreṇeti viśeṣaḥ /
kṣetrapariśuddhivaśitāśrayatvaṃ cā'ṣṭamyāpratividhyatīti prakṛtaṃ /
kṣetrapariśuddhivaśitāśrayatvaṃ dharmadhātor yatheṣṭaṃ 
buddhakṣetraparṣanmaṇḍalasandarśanaśaktipratilambhāt /
tasmāt tasyāṃ bhūmau praṇidhānapāramitā'tiriktatarā bhavati 
nirvikalpakṣetravaśitālābhena kuśalanirantaraprayogāt /
jñānavaśitāśrayatvaṃ navamyā /
dharmadhātoḥ pratividhyatīti prakṛutaṃ /
kathame tad vijñayate navamyā jñanavaṣitāśrayatvaṃ pratividhyatīti ?
ata āha - pratisaṃvil lābhāt /
asyāṃ hi bhūmau dharmārthaniruktipratibhānapratisaṃvido'nyair 
aparyāntāḥ pratilabhate /
ato'tra jñānavaśitāśrayatvaṃ dharmadhātoḥ pratividdham iti vijñāyate /
tasmāt tasyāṃ bhūmau balapāramitā'tiriktatarā bhavati 
prajñābalaviśeṣayogāt /
