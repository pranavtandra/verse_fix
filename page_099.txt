tad anantaraṃ tu viśuddhiviṣayaṃ vaktavyam ityata āha - 
viśuddhigocaratattvaṃ dvividham iti /
tattu yathā dvividhaṃ tat pradarśayann āha - 
kleśāvaraṇaviśuddhijñānagocaraṃ jñeyāvaraṇaviśuddhijñānagocarañceti 
/
kleśāvaraṇaviśuddhiḥ kleśāvaraṇaprahāṇaṃ /
kleśāvaraṇaviśuddhaye kleśāvaraṇaviśuddhau vā jñānaṃ 
kleśāvaraṇaviśuddhijñānaṃ /
kleśāvaraṇaviśuddhiprayojanaṃ tan nimittaṃ vetyarthaḥ /
tasya kleśāvaraṇaviśuddhijñānasya gocaram iti viṣaya ityarthaḥ /
evaṃ jñeyāvaraṇaviśuddhijñānasya gocaram api vaktavyaṃ /
tatra kleśāvaraṇaṃ sarve darśanabhāvanāmārgaprahātavyāḥ kleśopakleśā 
ityato darśanabhāvanātad upacārajñanaviṣayaṃ śrāvakapratyekabuddhānāṃ 
yat tattvaṃ duḥkhasamudayanirodhamārgasatyam ucyate tat 
kleśāvaraṇaviśuddhijñānasya gocaraṃ /
jñātavyaṃ jñeyaṃ /
tac ca sarvātmanā pañca vidyāsthānāni /
tāni punar adhyātmavyākaraṇahetucikitsāsarvaśilpakarmasthānākhyāni /
tatra jñānavibandhabhūtam akliṣṭam ajñānaṃ jñeyāvaraṇaṃ /
tad viśodhanaṃ lokottarajñānānulomādhimukticaryābhūmisa:ngṛhītaṃ tat 
pṛṣṭhalabdhañca jñānaṃ /
tat punar bodhisattvatathāgatasantānavartiṃ /
tasya ca tathatā sarvatragādyarthamukhenālambanam ityato 
jñeyāvaraṇaviśuddhijñānagocarasya tattvaṃ tathatā /
phalahetutattvoktyā sūkṣmatattvoktyā ca viśūddhigocaraṃ dvividham api 
nirdiṣṭam eveti punar asya vidhānaṃ vyarthaṃ /
na vyarthaṃ /
yatas tat phalahetutattvaṃ sūkṣmaṃ ca vastu vyavasthānata uktaṃ /
atra tvāvaraṇaṃ yathā viśuddhyate tathaitan nirdiṣṭaṃ /
idaṃ dvividhaṃ viśuddhigocaram ekasmāt pariniṣpannād eva svabhāvāt 
kīrtitaṃ /
atraiva kāraṇam āha - na hyanyaḥ svabhāvo viśuddhijñānagocaro 
bhavatīti /
kiṃ kāraṇaṃ ?
abhūtaparikalpasya dauṣṭhulyamaladvayena samalabhūtatvāt parikalpitasya 
cāsattvāt /
dvividhajñānaviṣayasya tu dharmatāyā viśuddhiviṣayatattvaṃ 
pariniṣpannād eva vyavasthāpyata iti /
uktaṃ viśuddhigocaratattvaṃ //3-12//
