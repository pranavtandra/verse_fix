vyavadānañcetyato'śuddhau viśuddhau cāviparyāsaḥ 
sa:nkleśavyavadānāntare sa:ngṛhyate /
dharmadhātoḥ sa:nkleśavyavadānayor 
āgantukatve'viparyāso'sāvapyākāśopamatāyāṃ yojayitavyaḥ /
sa:nkleśavyavadānāvasthayor dharmadhātoḥ kṣayavṛddhyabhāvād 
atrāse'nunnatau cāviparyāso vajrapadeṣv ahīnāviśiṣṭatāyāṃ 
yojayitavyaḥ /
evam ete daśāviparyāsā daśasu vajrapadeṣu yathākramaṃ yojayitavyāḥ /
vajrapadānāṃ śarīravyavasthānam iti yānārthānadhikṛtya vajrapadānāṃ 
pravṛttis te'rthās teṣāṃ śarīraṃ /
teṣāntu caturdhā vyavasthānaṃ svabhāvataḥ ālambanataḥ avikalpanataḥ 
codyaparihārataś ca /
tatra trayaḥ svabhāvāḥ pariniṣpannaparikalpitaparatantrākhyā ādyais 
tribhiḥ padair yathākramam iti /
tatrādau bhavānyādyāni tāni punaḥ sadasattā'viparyāsa āśrayaśceti /
tatra sadasattayā pariniṣpanna uktaḥ /
aviparyāsena parikalpita uktaḥ /
āśrayatvena paratantra uktaḥ /
katham iti ?
pariniṣpannasya sadasattāyā sadasattayā nirdeśa iti kecit /
kathaṃ sadasal lakṣaṇatvāt pariniṣpannasya sadasattayā'bhidhānam ityeke /
saṃyogāt saṃstavācākṣarāṇāṃ kalpitenārthena sārthakatvaṃ paśyan 
viparyayāc ca nirarthakatvaṃ pariniṣpannaḥ svabhāvo'nabhilāpya iti 
avagamyate /
atas tat sadasattā pariniṣpannaḥ svabhāva ity uktam ityanye /
mahāyānadeśanādharmaḥ suviśuddhadharmadhātuniṣyandatvād 
viparyāsapratipakṣālambanatvāc ca pariniṣpannaḥ svabhāva ity ucyate /
so'rthaḥ kalpitasvabhāvasya bhavati na tu parikalpitaḥ /
ataḥ sadasattā pariniṣpanna ityuktam ityapare /
anye tv āhuḥ - sadasattā'viparyastatvāt pariniṣpannaḥ svabhāva ukta iti 
/
aviparyāsena parikalpitas tasyāvidyamānena dvayarūpeṇa prakhyāti 
lakṣaṇāt /
āśrayeṇa paratas tasyābhidhānaṃ /
hetutvād āśraya ity ucyate phalatvāt paratantra iti /
etad eva svabhāvatrayam ālambanato jñeyaṃ /
avikalpanata ityatra vigrahabhedād āha - yena ca vikalpayati nirvikalpena 
jñānena yac ca na vikalpayati prakṛtiprabhāsvaratāṃ /
tatrārya vigrahabhedaḥ - na vikalpayatyanenetyavikalpanaṃ na vikalpyate 
tadityavikalpanaṃ /
tad bhāvo'vikalpanatā /
evam avikalpanatā dvābhyāṃ vajrapadābhyāṃ paridīpiteti darśayati /
tad anena jñeyajñānavyavasthānaṃ yathākramaṃ veditavyaṃ yad uta tribhiḥ 
svabhāvair avikalpanatayā ceti /
tatra tribhiḥ svabhāvair jñeyavyavasthānaṃ grāhyatvāt /
avikalpanatayā jñānavyavasthānaṃ grāhakatvāt /
codyaparihārata upayuktebhyaḥ śiṣṭāni vajrapadāni /
atra kiñcid vajrapradaṃ codyasya parihāra eva /
tadyathā māyopamatā ākāśopamatā ahīnāviśiṣṭatā ceti /
