tad anantaraṃ tu sa:ngrahatattvaṃ vaktavyam ityataḥ pṛcchati - kathaṃ 
trividhe mūlatattve sa:ngrahatattvaṃ veditavyam iti /
sa:ngrahatattvaṃ trividhe mūlatattve saṃgṛhītaṃ pradarśayann āha -
	nimittasya vikalpasya nāmnaśca dvayasaṃgrahaḥ /
iti vistaraḥ /
dvaye paratantre parikalpite ca sa:ngrahaḥ /
tatra sa:ngrahatattvaṃ pañca vastūnītyata āha - yathāyogaṃ pañca 
vastūny ārabhyeti /
yathāyogaṃ mūlatattve pañca vastūni sa:ngṛhītāni veditavyāni na tu 
yathākramaṃ /
pañca punar vastūni nimittaṃ nāma vikalpas tathatā samyag jñānaṃ ceti 
/
sarvasya jñeyasyāsmin  pañcavidhe vastuni sa:ngrahād etat pañcavidhaṃ 
vastu saṃgrahatattvam ity ucyate /
tatra nimittam ālayavijñānaṃ kliṣṭamanaḥpravṛttivijñānāni ca /
anyo'nyanimittabhāvān nimittaṃ /
nāma yat tasyaiva nimittasyā'nabhilāpyasyāpi sato'kṣinikocavad sūcakam 
abhidhānaṃ /
yathoktanimittasvabhāvaviśeṣavikalpakāstraidhātukapratisaṃyuktāścittacaittā 
vikalpaḥ /
śūnyatā tathatā /
samyag jñānaṃ lokottaraṃ tathatālambanaṃ /
kramaḥ punar eṣāṃ tatra sa:nkleśaprajñaptivastutvān nimittam ādau /
sa:nkleśaprajñaptitvād anantaraṃ nāma /
yathā nāma tathā sa:nkleśaprajñaptivastuvikalpatvāt tad anantaraṃ 
vikalpaḥ /
sa ca sa:nkleśas tathatāsamyag jñānena nirudhyata iti tad anantaraṃ 
tathatā samyag jñānañceti kecit /
	bhrāntinimittaṃ bhrāntidauṣṭhulyaṃ tan nimittam iti /
	tasmād bhrāntis tasya nimittaṃ prakṛtiśamas tatra jñānaṃ //
ity anayā gāthayā'sya pañcavidhasya vastuno'rtha ity anye /
tatrā'dhyātmikacakṣurādivijñānsaṃgṛhītāyā bhrānter nimittaṃ 
bāhyādhyātmikā rūpādicakṣurādisaṃgṛhītā bhrāntiḥ /
tasyā bāhyādhyātimikāyā bhrānter nimittaṃ dauṣṭhulyaṃ /
tat punar ālayavijñānavāsanā /
tasyā vāsanāyā nimittaṃ saiva bāhyādhyātmikabhrāntiḥ /
tasmāt tatrābhilāpapravṛttiḥ /
abhilāpavaśāc ca vāsanāyā avasthānam ālayavijñāne /
prakṛtiśama iti tathatā /
prakṛtyā kliṣṭatvād dharmadhātoḥ /
tatra ca jñānam iti tathatālambanaṃ mārgasaṃgṛhītaṃ yaj jñānaṃ /
tatra nimittavikalpayoḥ paratantreṇa sa:ngrahaḥ hetupratyayotpāditatvāt /
