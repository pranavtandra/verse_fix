tataḥ śrāvakādyatikramaṇaprayogeṇa yathopacitāni sarvāṇyetāni 
laukikakalokottarāṇi kuśalamūlāni bodhau pariṇāmayitavyāni /
tataś ca nāmavasthāmāgato mahābodhyāśayatvād buddhabodhisattvānāṃ 
gambhīrodāreṣu nirdeśeṣu na trasyati /
tato bahuśrutapratipattiyukta udāraṃ dharmaṃ nirdiśati 
sattvaparipācanārthaṃ /
tataḥ paripakvavineyeṣu tadvimocanārthaṃ vaśitvaṃ buddhatvam avāpnotīti /
atra hi triṃśadāvaraṇāny uktāni āvaraṇīyārthāḥ śubhādayo daśa 
cetyataḥ pṛcchati - eṣāṃ tu śubhādīnāṃ kasya katyāvaraṇāni 
vijñeyānīti /
ata āha -
	trīṇi trīṇi ca eteṣāṃ jñeyānyavaraṇāni hi /
iti /
kuśalasya trīṇyāvaraṇānīti /
aprayogo'nārambhaḥ /
kim atrāvaraṇaṃ ?
yena kleśena na prayujyate pramādena kausīdyena vā yasminn avasthāviśeṣe 
kleśabījānugate'vyākṛte kliṣṭe vā na prayujyate /
na prayujyate nityaṃ gauravād vā alapatvād vetyapratyoga ucyate na tu 
prayogābhāvāt tasminn āvaraṇā'nupapatteḥ /
anāyatanaprayogo bodher bodhyāvāhakānāñca kuśalamūlānāṃ yat 
pravacanam utpattidvāraṃ tadāyatanaṃ tasmād anyad anāyatanaṃ /
tasmin prayogo'nāyatane prayoga etasmād bāhyeṣvartheṣu /
tatra kimāvaraṇaṃ ?
yenā'jñānena dṛṣṭikleśena vā'nāyatane prayujyate sa eva ca prayoga 
āvaraṇaṃ /
ayoniśaḥ prayoga iti /
yonir upāyaḥ /
yasya ya upāyas tasmād anyathā prayogo'yoniśaḥ prayogaḥ /
ihaiva śāsane paṭhyate yad uta rāgacaraṇabhāvānāṃ hi na 
rāgaprahāṇopāyaḥ pratighacaraṇasyā'śubhabhāvaneti /
atrā'pyayoniśo manaskāras tannimitto vā prayoga āvaraṇaṃ /
bodhes trīṇi /
yathārtham avabodho bodhiḥ /
kuśalasyā'nutpattir iti pūrvoktam eva /
katham anutpattirāvaraṇaṃ ?
kuśale'sati bodhir na bhavet tataḥ kuśalasyā'nutpattir bodher āvaraṇaṃ /
yo vā kleśo 'vasthāviśeṣo vā kuśalotpatter āvaraṇaṃ sa eva bodher api /
yo hi yat sambhārasyāvaraṇaṃ niyatam asau tat phalasyāpīti /
utpannasyāpy amanasikaraṇaṃ abhāvanā'napekṣetyarthaḥ /
yena kausīdyādikleśena yasminn avasthāviśeṣa utpannam api kuśalaṃ paunaḥ 
punyena na manasi karoti tad atrāvaraṇaṃ /
aparipūrṇasaṃbhāratā ceti /
yāvatā sambhāreṇa bodhiḥ prāpyate tāvān
