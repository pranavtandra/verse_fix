rūpārūpyadhātāv asatyapi kavaḍīkārāhāre 
jīvitendriyādevā'vasthānasadbhāvāt /
na ca tatra sparśādīnām avasthāna ādhipatyaṃ yujyate /
nirodhā'saṃjñisamāpattyavasthāyām āsaṃjñike ca sparśādyabhāve'pi 
sattvānāṃ sthitisambhavād iti /
kulasandhāne strīpuruṣendriyayor ādhipatyam iti /
putrotpattau kulāvyuparatatā vartata iti putrotpattiḥ kulasandhānaṃ /
etad eva darśayann āha - prasavādhipatyād iti /
kāyendriyapradeśa eva kaścid ādhipatyaviśeṣāt strīpuruṣendriyākhyaṃ 
pratilabhate /
yatra stīndriyaṃ puruṣendriyañcas tastatra kulavyavasthānam asti /
tad vaśena kulo na vyuparamate /
kāmāvacareṣv api ca deveṣu strīndriye puruṣendriye ca sati 
kulavyavasthānaṃ nā'satīti /
nahi gandharvādīnāṃ tatrādhipatyaṃ satyapi gandharvādau paṇḍakādiṣu 
strīpuruṣendriyayor asadbhāve putraprasavasāmārthyābhāvāt /
āhārādīnāṃ punaḥ sattve'saṃkhyātatvāt satyapyādhipatye nendriyatvaṃ 
/
upabhoge vedanendriyāṇāṃ /
ādhipatyam ityadhikṛtaṃ /
katham ityata āha - kuśalākuśalakarmaphalopabhogād iti /
vedanendriyāṇi pañca - sukhaṃ duḥkhaṃ saumanasyaṃ daurmanasyam upekṣā ca 
/
tarhi kuśalākuśalakarmaphala upabhujyate anubhavātmakatvāt /
na nāmarūpasya tadādhipatyaṃ vedanāsambhārabhāvāt /
tat sambhārabhāvaś ca vedanāpradhānatvād vipākasyeti /
laukikaviśuddhiḥ prahāṇaṃ laukikakleśānāṃ /
tatra śraddhādīnām iti 
śraddhā-vīrya-smṛti-samādhi-prajñā-pañcendriyāṇāṃ /
ādhipatyam iti vartate /
teṣām aśraddhādipratipakṣetvena laukikamārgaprabhāvitatvāt /
mokṣabhāgīyaprabhāvitatvād ity apare /
lokottaraviśuddhāvanājñānamājñāsyāmīndriyādīnām ityādhipatyam 
iti vartate /
lokottaramārgeṇa kleśānāṃ prahāṇaṃ lokottaraviśuddhiḥ /
tatrānājñātam ājñāsyāmīndriyājñendriyājñātāvīndriyāṇāṃ 
trayāṇām ādhipatyaṃ /
teṣāṃ darśanabhāvanā'śaikṣamārgeṣu yathākramaṃ saṃgṛhītatvāt /
tatra śraddhādīnāṃ pañca manaḥsukhasaumanasyopekṣāṇi 
cānāgamyādyāśritaviśeṣeṇa sambhavam adhikṛtyaitāni navendriyāṇi 
darśanamārge'nājñātam ājñāsyāmīndriyaṃ /
bhāvanāmārga ājñendriyaṃ /
aśaikṣaṃārga ajñātāvīndriyam ity ucyate /
yogācārāṇāṃ tv anyathā vyavasthānaṃ /
