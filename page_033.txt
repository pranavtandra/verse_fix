dvayam api karmātmakatvāt /
ayantu viśeṣaḥ -
svarūpāvasthaṃ karma saṃskārāḥ bījāvasthaṃ bhava iti /
janmasaṃkleśaḥ śeṣāṇya:ngānīti 
vijñānanāmarūpaṣaḍāyatanasparśavedanājātijarāmaraṇāni eṣāṃ 
janmasa:ngṛhītatvāt /
dvedhā saṃkleśo hetuphalābhyāṃ prabhidyate /
hetusaṃkleśaḥ kleśakarmasvabhāvair a:ngaiḥ /
phalasaṃkleśaś ca śeṣair a:ngair iti /
tatra hetusaṃkleśatvaṃ karmakleśānāṃ vijñānādivedanāparyantaṃ 
jātyātmakaṃ jarāmaraṇādīnavajanma sampādayitum abhipravṛttatvāt /
phalasaṃkleśaś ca śeṣair vijñānādibhiḥ karmakleśaphalatvāt /
punaḥ sa eva pratītyasamutpādaḥ saptadhā saṃkleśa iti 
viparyāsākṣepopanaya-parigrahopabhogākarṣaṇodvegahetutvāt saptavidho 
hetuḥ /
tatra viparyāsahetur avidyeti /
avidyāgato hi tattvasammohād anityādīn nityādirūpeṇa viparyasyatīti /
ākṣepahetuḥ saṃskārā iti vijñāne catura:ngātmajanmabījasya 
pratiṣṭhāpanāt /
upanayahetur vijñānam itīha cyutasyopapattiviṣaYa upanayanāt /
parigrahahetur nāmarūpaṣaḍāyatana ity upanītasya 
nāmarūpaṣaḍāyatanābhyāṃ nikāyasabhāgaparigrahāt /
pūrvaṃ nāmarūpeṇaiva parigraha ukto jarāyujādikāṃ yonim abhipretya 
idānīm upapādukam adhikṛtya ṣaḍāyatanenā'pi parigraha uktaḥ /
atha vā nāmarūpeṇa tat prathamata ātmabhāvamātrasya sa:ngraho 
vivakṣitaḥ /
iha tv aparipūrṇasya paripūrṇasya ceti nāsti virodhaḥ /
upabhogahetuḥ sparśavedana iti /
sakāraṇenānubhavena kuśalākuśalakarmavipākasyopabhogāt /
ākarṣaṇahetus tṛṣṇopādānabhavā iti /
sa upabhuktakarmavipāko bālas tasyāṃ tasyāṃ vedanāyāṃ tṛṣyati /
vivṛddhatṛṣṇaś ca tat saṃyogatṛṣṇayā kāmādīn upādatte /
tad upādānāc cāsya pūrvabhūtaṃ paunarbhavikaṃ karma vāsanāvasthaṃ 
pariṇataṃ yathākṣiptavipākadānayā pravṛttiṃ labdhvā bhavo bhavati /
sa ca tṛṣṇopādānabhavaḥ saṃskārākṣiptaḥ punar bhavābhinirvṛttaya 
ākṛṣyate abhimukhīkriyata iti tṛṣṇopādānabhavā ākarṣaṇahetuḥ /
atha vā tṛṣṇāvaśena cchandarāgānurūpānyatarakarmavāsanābhimukhāt 
tṛṣṇopādānabhavā ākarṣaṇahetuḥ /
udvegahetur jātijarāmaraṇe ityevamatra jarmākarṣaṇāt tasmiṃstasmin 
sattvanikāye jātijarāmaraṇaduḥkham anyacca
