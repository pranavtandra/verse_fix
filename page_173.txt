sā'bhrāntiḥ /
sā punaḥ prakṛtiprabhāsvaratāyāṃ pariniṣpannasvabhāvalakṣaṇāyāṃ /
bhrāntyabhrāntiphale caiveti tatra bhrāntiphalaṃ hi sa:nkleśaḥ /
bhrāntiniṣyanda iti vākyaśeṣaḥ /
abhrāntiniṣyandānāyā bhrānter abhrāntiphalatvāt /
abhrāntiphalaṃ vyavadānaṃ sasamprayogam iti vākyaśeṣaḥ /
evakārakaraṇaṃ tat phale sa:nkleśo vyavadānam ityanenaivokte 
ākāśopamatayā tu paśyati dharmadhātor yasya bhrāntyabhrāntiphalaṃ tat 
sa:nkleśavyavadānātmakaṃ /
nedaṃ tat prakṛtikam iti pradarśanārthaṃ /
paryantaś ca tayor iti sa:nkleśavyavadānayoḥ paryanto nirvāṇaṃ /
taccahīnānadhikatvena paryantam uktam ityeke /
anye tvāhuḥ - saṃsārasyāprameyatayā'pacayābhāvād ahīnatāyuktaḥ 
sa:nkleśaparyantaḥ /
nirvāṇasyāprameyatayopacayābhāvād aviśiṣṭatāyukto vyavadānaparyanta 
iti /
etāni daśa vajrapadāny aviparyāsaprasa:ngadvāreṇoktāni /
na tu sūtritānītyatas tānyantaraślokābhyāṃ sa:ngṛhītāni -
	sadasattā'viparyāsa āśrayo māyayopamā /
	akalpanā prakṛtyā ca bhāsvaratvaṃ sadaiva hi //

	saṃkleśo vyavadānañca ākāśopamatā tathā /
	ahīnatvādhikatvañca daśa vajrapadāni hi //
atra ca pariniṣpannaśabdārthasambandhadarśanaṃ viparyāsaḥ /
yathā pratyavabhāsate tathā 'rthānāṃ svabhāvasattādarśanaṃ 
viparyāsaḥ /
tad arthapravṛttikāraṇadarśanaṃ viparyāsaḥ /
arthasamāropāpavādadarśanaṃ viparyāsaḥ /
yathānām arthasvabhāvadarśanaṃ viparyāsaḥ /
bhāvānāṃ svasāmānyalakṣaṇayor vastutvena darśanaṃ viparyāsaḥ /
svastyasvastisañcārād viśuddhir aviśuddhiśceti darśanaṃ viparyāsaḥ /
ekasyaivāśuddhiviśuddhidarśanaṃ viparyāsaḥ /
sa:nkleśavyavadānaprahāṇapratilambhayor hāniviśeṣadarśanāt 
trāsamānapravṛttir viparyāsaḥ /
ityete daśa viparyāsāḥ /
bodhisattvasya tad abhāvādaviparyāsāvasthitasya 
yathokteṣvartheṣvariṇatā vidarśanā bhāvanā veditavyā /
uktā'nudharmapratipattiḥ //

	{ antadvayavarjane pratipattiḥ }
