saṃsārasāgarottarakatvena sarvasahetukaduḥkhā'tyantaviyogātmakatvena tat 
pratipādakatvena ca dharmaratne parijñānaṃ /
śrāmaṇyapahalasthatvenānuttaradakṣiṇīyakṣetratvena cāsa:ngharatne 
parijñānaṃ /
satyadarśanāt prahīṇavicikitsaścā'vetya prasādalābhena 
ratnatrayaguṇe'bhiśraddatte nā'prahīṇavicikitsa iti /
ato vicikitsāsaṃyojanaṃ ratnatrayaparijñāna āvaraṇam uktaṃ /
īrṣyāsaṃyojanaṃ lābhasatkāraparijñāna āvaraṇaṃ /
atraiva kāraṇam āha - tad doṣā'darśanād iti /
īrṣyāsaṃYojanaṃ hi lābhasatkārarakto'nyasampatpratighāṃśikaścetaso 
vyāroṣaḥ /
sarvānarthāśrayatvā'vabodho lābhasatkāraparijñānaṃ /
īrṣyayā tu lābhasatkāraṃ sarvaguṇavipakṣabhūtatvena 
vividhā'narthamūlatvena ca na parijānātītyato lābhasatkāra 
parijñāna āvaraṇam irṣyāsaṃyojanaṃ /
mātsaryasaṃyojanaṃ saṃlekhaparijñāna āvaraṇam iti /
katham ityata āha -
pariṣkārā'dhyavasānād iti /
mātsaryaṃ pariṣkāre'ṣv avyavasito rāgāṃśikaścetasa upakaraṇeṣv āgrahaḥ 
/
saṃlekhaḥ pariṣkārair vivekaḥ /
tasya sarvaguṇāśrayavastutvenā'vabodhaḥ parijñānaṃ /
na tu mātsaryeṇa pariṣkārā'dhyavāsānāt sarvaguṇāśrayetvena 
parijānātītyataḥ saṃlekhaparijñāne mātsaryāvaraṇatvam uktaṃ /
tatrā'nunayapratighasaṃyojane tattvādhigamaprayogāvaraṇe /
śeṣāṇi tu prayuktasya tattvadarśanasyāvaraṇāni tasmād eva teṣām 
anukramaḥ /
tadyathā'vaśyaṃ prathamam eva mumukṣūṇāṃ saṃsārāc citttam 
udvejayitavyaṃ /
tataḥ sarvata upekṣeṇa bhavitavyaṃ /
tataḥ sarvā'narthamūlatvāt satkāyadṛṣṭiḥ parijñeyā /
tad anantaraṃ dukḥ samudayātmakatvena tadvastu parijñeaṃ/
tatastannirodho vijñeyaḥ/
tad anantaraṃ lābhamārgo vijñeyaḥ /
tataḥ satyadarśanād ratnatraye tad guṇā'vabodhād avetya prasādalābhaḥ /
satyadarśanabalenaiva ca lābhasatkārasaṃlekhayor doṣaguṇadarśī 
bhavatītyevam āvaraṇīyadharmakramahetunā tadāvaraṇakramo veditavyaḥ /
nanu sarvasyāpi sarvatrāvaraṇatvena saṃbhavād 
vastupratiniyamenāvaraṇanirdeśo na yuktaḥ /
naivā'tra vastupratiniyamaḥ kriyate kintarhi pratyāsattitaḥ prādhānyaiṣa 
nirdeśaḥ //

		{ bodhisattvāvaraṇaṃ }
