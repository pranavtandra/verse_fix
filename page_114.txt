pratītyasamutpādānantaraṃ sthānā'sthānārtho vaktavya ityata āha -
	aniṣṭeṣṭaviśuddhīnāṃ
iti vistaraḥ /
sthānāsthānaṃ saptavidhapāratantryārthena veditam iti /
tatra sthānaṃ hetuḥ asthānamahetuḥ /
yathoktaṃ gāthāyāṃ -
	satyaṃ vaden na ca kruddhed dadyād alpoo'pi yācitaḥ /
	ebhis tribhir iha shtānair devānāṃ samitiṃ vrajet //
hetubhir iti gamyate /
athavā sthānaṃ sambhavaḥ asthānaṃ punar asambhavaḥ /
tat kauśalyaṃ sthānāsthānakauśalyaṃ /
paratantrasya bhāvaḥ pāratantryaṃ /
parāyattatetyarthaḥ /
tac ca saptavidham api triṣvantarbhavati karmakleśajanmapāratantryeṣu /
tatrāniṣṭe pāratantryam iti /
aniṣṭā duargatayaḥ /
tatra gamanaṃ duścaritāyatattvaṃ pāratantryañceti pradarśayann āha - 
duśaritena prāṇātipātādinā kāyavā:nmanaḥkarmasaṇggṛhītena 
saprayogapṛṣṭhenā'nicchato'pi durgatigamād iti /
durgatayo narakāḥ pretāstiryañcaś ca /
teṣām ekāntena duḥkhatvān na kasyacit tatra gamanecchā bhavati /
karmaṇāṃ balavattvād durgatigetutvāc ca duścaritasyānicchato'pi tatra 
karmavaśād eva gamanaṃ bhavati /
iṣṭe pāratantryaṃ sucaritena sugatigamanād iti /
iṣṭaṃ sugatigamanaṃ /
sugatir devā manuśyāś ca /
sucaritaṃ saprayogapṛṣṭhebhyaḥ prāṇātipātādibhyo viratiḥ 
dānamānasatkārādikriyā śāstṛguruṇānvitādiṣu maitrīkaruṇādayaś ca 
guṇaviśeṣāḥ /
pāratantryaṃ sucaritamantareṇānyathā prārthayatām api 
sugatigamanābhāvāt /
sucaritapratibaddhataiva sugatyutpatteḥ /
viśūddhau pāratantryam iti vistaraḥ /
viśuddhiḥ kleśaprahāṇaṃ /
pañca nivaraṇāni kāmacchando vyāpādaḥ styānamiddhamauddhatyakaukṛtyaṃ 
vicikitsā ca /
styānamiddhamekīkṛtya auddhatyakaukṛtyañca pañca bhavati /
abhedena kuśalapakṣaṃ nivṛṇvantīti nivaraṇāni bhedena punar yathākramaṃ 
pravrajatyābhiratiṃ tad anurūpe kāyavākkarmāṇi sabrahmacāribhiḥ samyak 
coditasya samyak pratipattiṃ śamathakāle'līnatāṃ pragrahakāle'vikṣepaṃ 
śamathavipaśyanākāle svarasavāditāṃ chādayantīti nivaraṇāni /
tānyaprahāyātyaktvā yāvat sapta bodhya:ngānyabhāvayitveti /
yāvad iti paryantopadeśāt smṛtyupasthānasamyak 
prahāṇarddhipādendriyabalānām api grahaṇaṃ /
duḥkhasyāntā'karaṇād iti /
duḥkhasyānto
