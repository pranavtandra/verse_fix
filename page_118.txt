anājñātam ājñāsyāmīndriyājñendriye dvaye daśendriyāṇi /
pūrvokteṣu navasu daurmanasyam adhikaṃ prakṣipya daśa bhavanti /
ājñātāvendriyan tu navaiva /
tatra prayogamārge nirvedhabhāgīyātmake pañcadaśasu ca 
darśanamārgakṣaṇeṣu manaindriyaṃ pañca 
śraddhādīnyanāgamyādyāśritaviśeṣeṇa yathāsaṃbhavaṃ 
sukha-saumanasya-daurmanasyopekṣendriyāṇāṃ cānyatamād daurmanasyendriyaṃ 
punaḥ prayogakāle nirvedhabhāgīyapṛṣṭhenottaravimokṣaspṛhāsa:ngṛhītaṃ 
veditavyaṃ /
tadetat saṃbhavato daśavidham indriyaṃ /
anājñātam evājñātuṃ pravṛttatvād anājñātam ājñāsyāmīndriyam 
ucyate /
tadeva daśavidham indriyaṃ ṣoḍaśadarśanamārgakṣaṇādyāvad vajropamaḥ 
samādhir ityetasmin śaikṣamārgam ājñendriyam ity ucyate 
apūrvājñātājñeyābhāvāt /
aśaikṣamārge daurmanasyendriyavyatiriktāni 
śiṣṭānyājñātāvīndriyākhyāṃ pratilabhante /
ājñātāvīndriyam iti ājñātāvīndriyam ucyate /
evam indriyakauśalasya yaḥ puruṣanārāyaṇeśvarādiṣv adhipatigrāhaḥ sa 
vyāvartate //

	{ adhyārthaḥ }
indriyārthānantaram adhvatrayo vaktavya ityata āha -
	phalahetūpayogārthanopayogāt tathā'paraṃ //
iti /
tatra janmakāraṇaṃ hetuḥ /
phalaṃ janamasaṃgṛhītaṃ /
phaladānaṃ hetor upayogaḥ /
phalasyopāyogo'nubhūtasya parikṣayaḥ /
evam prakāraścātīto hetus tatphalaṃ cetyataḥ phalasya 
taddhetoścopayogārthenānīto'dhvā /
tenaiva prakāreṇa phalasya hetoścā'nupayogārthenā'nāgato'dhvā /
vartamānaḥ punar adhvā vartamānaṃ janma /
hetordattaphalatvād hetūpayogārthena tatphalañca vartamānaṃ 
janmā'nuvartata eveti phalā'nuyogārthena ca veditavyaḥ /
bhāvānāñca pratikṣaṇaṃ hetuphalabhāvatvena kṣaṇatvam 
adhikṛtyā'dhvavyavasthānaṃ veditavyaṃ /
tatra hetūpayogaḥ svaphalasyotpādānāt /
phalopayogaḥ phalasya janmānantaravināśāt /
evaṃ phalahetūpayogārthenātīto'dhvā phalahetvor 
anupayogārthenānāgato'rdhveti prakṛtaṃ /
kāryānutpādād anutpannasya ca vināśābhāvāt phalahetvor anupayogo 
veditavyaḥ /
