ataḥ samādānānantaraṃ dhīmatvam uktaṃ /
evañca prayujyamāno viparyāsaṃ prajahāti aviparyastaṃ ca jñānam 
utpādayati /
ato dhīmatvānantaram abhrāntir uktā sa punar darśanamārgaḥ /
tad abhyasanenāvaraṇaṃ vijahātītyatas tad anantaram āvaraṇaprahāṇam 
anāvaraṇam uktaṃ /
nirāvaraṇasya bhavabhogasaktyabhavād buddhatva eva mānasaṃ pariṇāmayati 
nānyatreti /
atas tad anantaraṃ natiruktā /
evam bhūtasya ca sugambhīreṣv api buddhadharmeṣu prāmodyotpatīratrāsa 
ityatas tad anantaramatrāsa uktaḥ /
evaṃ dharmapratyakṣeṇa yena yenopāyena sattvavinayaṃ paśyati 
tantamevopāyambhajata iti tasya kuto mātsaryambhaved ityatrāsānantaram 
amātsaryam uktaṃ /
evam prahīṇāvaraṇaḥ paripūrṇaśukladharmā ca bodhisattvaścittavaśitve 
pravartata ityatas tad anantaraṃ vaśitvam uktaṃ /
eṣo'nukrama iti /
asmin paricchede āvaraṇaprasa:nga ityataḥ śubhādīnām utpatīkāraṇāny 
āvaraṇavat sūtritāni /
prasa:ngāgatānām antaraślokena sa:ngrahaḥ kriyate /
	kāraṇaṃ daśadhotpattau sthitau dhṛtyāṃ prakāśane /
	vikāraviśleṣanatipratyayaprāyaṇāptiṣu //
eṣa dvitīyo'ntaraśloka udāharaṇasa:ngrahārthaṃ -
	cakṣurāhārabhūdīpa vahnyādis tad udāhṛtiḥ /
	dātraśilpajñatādhūmahetumārgādayo'pare //
iti //
