puṇyajñānasambhāro'pūrṇaḥ pūrvavad avasthākleśo'lpasambhāro vā /
anyaḥ punar āha - aviśeṣavacane'pi nirvedhabhāgīnānām 
evātrā'nutpattir abhipretā nānyeṣāṃ /
utpanānām api teṣām evā'manasikaraṇaṃ manasikurvato'py 
asaṃbhṛtasambhāra āvaraṇam ucyata iti /
samādānaṃ bodhicittotpāda iti /
niravaśeṣāḥ puṇyajñānasambhārās tat phalañca buddhatvaṃ sarvasattvāś 
ca niruttare nirupadhiśeṣe nirvāṇadhātau pratiṣṭhāpayitavyenā'nena iti 
samādānaṃ bodhicittotpādaḥ /
sa punaḥ svaparahitakriyāśayaścetanācchandasahitaḥ /
tasyāvaraṇeṣu triṣu gotravaidhuryaṃ /
agotrakaḥ śrāvakādigotrako vā /
kalyāṇamitravaidhuryaṃ /
satyapi gotre ye taṃ bodhicittotpāde samādāpayanti tair asamavadhānaṃ 
yān vāgamya bodhir vyāvartate kuśalair vā dharmair na vardhate /
satyapi kalyāṇamitrasamavadhāne saṃsāraduḥkhena parikhedacittatā 
vipratipannasattvānām udvagacittasyāśu parinirvāṇābhiprāyaḥ /
kim atrāvaraṇaṃ ?
hetuvaikalyaṃ pratyayavaikalyaṃ sattveṣu cā'kāruṇyaṃ kausīdyaṃ vā /
akāruṇiko hi sattvārthena khidyate kusīdo vā /
dhīmattvaṃ bodhisattvateti /
svabhāvena śrāvakādigotrakebhyamtīkṣṇendriyatvāt sarvathā 
sarvajñeyopaparīkṣādhikāritvāt gambhīrodāradharmā'dhimukteś ca 
bodhisattva eva dhīmān nā'nyaḥ /
bodhisattvatā'viparyastā parārthaparā pratipattiḥ /
pratipatter vaidhuryaṃ pāramitādiṣv apravṛttiḥ /
bodhisattvo hi pratipattivṛtto jñeyaḥ /
kujano mūrkhajana iti aparīkṣakajana ityarthaḥ /
kujano hi pratipattiyuktam api bodhisattvopamitaṃ na jānīte /
duṣṭajanaḥ pratihata iti bodhisattvadveṣiṇaḥ /
te hi randhrā'nvayitvād bodhisattvasya bhūtān guṇān na paśyanti /
asaddoṣadarśanāt pratigharaudracittād vā bhūtān guṇān 
pracchādyā'dhyāropyā'bhūtān doṣān prakāśayanti /
kim atrāvaraṇaṃ ?
dānapāramitādivipakṣo 'jñānam pradveṣaśceti /
abhrāntes trīṇi /
adhyāropāpavādād bhrāntiḥ tat pratipakṣas tu bhrātyabhāvo 
darśanamārgaḥ /
viparyāsadauṣṭhulyaṃ grāhyagrāhakā'bhiniveśavāsanā paripuṣṭir iti 
kecit /
sarveṣāṃ darśanaheyānām anuśayānāṃ 
sarvatragadharmadhātubodhapratibuddhasya
