{ pāramitāsvāvaraṇāni }
   tad anantaramparāramitāsu vaktavyam ityata āha - pāramitāsvāvaraṇaṃ 
/
	aiśvaryasyā'tha sugateḥ sattvā'tyāgasya cāvṛtiḥ /
iti vistaraḥ /
atra daśānāṃ pāramitānāṃ yasyāḥ pāramitāyā yat phalaṃ tadāvaraṇena 
tasyā āvaraṇam udbhāvitaṃ /
kimatra kāraṇaṃ yan mātsaryādayo vastuto dānādivibandhe vartamānas tat 
phalāvaraṇatvenocyante na punaḥ pāramitāvaraṇatveneti ?
tat phalārthena dānādau sarvaḥ pravartate /
tasmān mātsaryādiprahāṇa ādarotpādanārthaṃ phalāvaraṇam uktaṃ na 
pāramitāvaraṇam iti /
athavā mātsaryādikam āvaraṇam aiśvaryādivipakṣabhūtaṃ 
dāridryādiphalaṃ janayadaiśvaryādiphalapratibandhāt tadāvaraṇam ucyate 
/
kintāsāmphalaṃ ?
tatra dānapāramitāyāḥ phalaṃ prādhānyenaiśvaryādhipatyaṃ /
śīlapāramitāyāḥ sugatiḥ /
kṣāntipāramitāyāḥ sattvā'parityāgaḥ /
vīryapāramitāyā doṣaguṇahānivṛddhī /
dhyānapāramitāyāḥ śāsane sattvāvatāraṇaṃ /
prajñāpāramitāyā guruvimocanaṃ /
upāyakauśalyapāramitāyāṃ bodhipariṇāmanayā kuśalā'kṣayatvaṃ /
prāṇidhānapāramitayāḥ śubhanairantaryaṃ /
balapāramitāyāḥ kuśaladharmaniyatīkaraṇaṃ /
jñānapāramitāyā dharmopabhogaḥ sattvaparipācanañca /
aiśvaryādhipatyāvaraṇam iti /
tatraiśvaryaṃ dhanopakaraṇabahulatā /
ādhipatyaṃ
