{abhūtaparikalpapiṇḍārthaḥ}
piṇḍārthaḥ punar abhūtaparikalpasya navavidhaṃ lakṣaṇaṃ paridīpitaṃ 
bhavati /
sallakṣaṇaṃ asallākṣaṇaṃ svalakṣaṇaṃ sa:ngrahalakṣaṇaṃ 
asallakṣaṇā'nupraveśopāyalakṣaṇaṃ prabhedalakṣaṇaṃ paryāyalakṣaṇaṃ 
pravṛttilakṣaṇaṃ saṃkleśalakṣaṇañca /
piṇḍārthaḥ punar abhūtaparikalpasyeti vistareṇārthanirdeśaḥ 
sukhāvabodhārthaṃ piṇḍārthanirdeśo'vismaraṇārthañca /
tato'trārthadvayam uktaṃ /
sallakṣaṇaṃ -
	abhūtaparikalpo'sti
iti /
asallakṣaṇaṃ -
	dvayaṃ tatra na vidyate /
iti /
svalakṣaṇaṃ -
	arthasattvātmavijñaptipratibhāsaṃ prajāyate
	vijñānaṃ //
iti /
sa:ngrahalakṣaṇaṃ -
	kalpitaḥ paratantraś ca pariniṣpanna eva ca /
iti /
asallakṣaṇānupraveśopāyalakṣaṇaṃ -
	upalabdhiṃ samāśritya nopalabdhiḥ prajāyate /
ityādi /
prabhedalakṣaṇaṃ -
	abhūtaparikalpaś ca cittacaittās tridhātukāḥ /
iti /
paryāyalakṣaṇaṃ -
	tarthārthadṛṣṭir vijñānaṃ tad viśeṣe tu caitasāḥ /
iti /
pravṛttilakṣaṇaṃ -
	ekaṃ pratyayavijñānaṃ dvitīyaṃ aupabhogikaṃ /
iti /
saṃkleśalakṣaṇaṃ -
	chādanād ropaṇāc caiva nayanāt samparigrahāt
ityādi //

evam abhūtaparikalpaṃ navaprakāraṃ khyāpayitvā yathā śūnyatā 
vijñeyā tan nirdeśatīti ko'trābhisambandhaḥ ?
dvayam anena pratijñātam abhūtaparikalpaḥ śūnyatā ca /
