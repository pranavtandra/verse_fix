upasargeṇa dhātvartho balād anyaḥ pratīyate /
	ga:ngāsalilamādhuryaṃ sāgareṇa yathāmbhasā //
sugatātmajam iti /
suṣṭhu gataḥ savāsana kleśāvaraṇāj jñeyāvaraṇāc cāpratiṣṭhitaṃ 
virvāṇam iti sugataḥ /
sa ca sarvavāsanāvaraṇaprahīṇaḥ sarvathā sarvadharmāvabodhasvarūpaḥ 
sarvavibhūtyāśrayabhūta-ṇiratnavad acintyaprabhāvavigrahaḥ 
sarvasattvānāmanābhogena sarvārtha raṇasamartho 
nirvikalpakajñānaviśeṣātmakaḥ sugataḥ taj janitatvān nirvikalpasya 
jñānasya /
tasmāt tasmin vā jātaḥ sugatātmajaḥ /
athavā sugatātmanā jāta iti sugatātmajaḥ /
yathoktaṃ sūtrāntare -jāto bhavati tathāgatavaṃśe tadātmakavstu 
pratilābhād iti /
evaṃ sati bodhisattvasya daśamyāṃ bhūmau pratiṣṭhitasya sarvākāraṃ 
jñeyaṃ vastu karatalastham ivāmalakaṃ 
tattvaṃśukāvacchāditalocanasyevābhāsamāyāti /
bhagavataḥ punar apanītalocanāvaraṇasyevetyayaṃ viśeṣaḥ /
atra hi sugatātmajasya śāstrapraṇetur avabodhasampat pradarśitā 
lābhasatkāranirapekṣasya śāstrapraṇetṛtvena karuṇāsampat 
prajñāsampacca /
vaktāram iti vyākhyānasya kartāraṃ /
abhyarhyeti sambadhyate /
sugatātmajamityapītyapare /
sa punar āryā'sa:ngaḥ /
tasya hīdaṃ śāstram abhivyaktaṃ ākhyātaṃ cāryamaitreyādhiṣṭhānāt 
dharmasantānena /
ceti samuccaye pādapūraṇe'dhikavacane vā /
anyān api buddhabodhisattvānarcayitvā na kevalaṃ praṇetāraṃ 
vaktārañceti /
kebhyo vaktāraṃ /
asmad ādibhya iti /
vayam ādir yeṣāṃ te'samadādayas tebhyo'smadādibhyaḥ /
anenātmana upadeśasyāmṛṣātvaṃ vyajyate /
abhyarhyeti abhyarcya /
abhitaḥ purataḥ sākṣād iva sthitaṃ /
arhitvā arcayitvā kāyavā:nmanobhiḥ /
śāstrasya praṇetāraṃ vaktāraṃ cābhyarcya kiṃ kariṣyasītyāha -
	yatīṣye'rthavivecane //
iti /
yatnamārapsye arthavivecane arthavivareṇe pṛthagbhāvakaraṇe vā /
iyaṃ ca nimittārthā saptamī /
arthavivecananimittam ityarthaḥ //
