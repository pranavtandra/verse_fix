āśrayaparāvṛttilakṣaṇatvād bodher yāvat sattvās tāvad avatiṣṭhantīty 
ucyate na tv anutpattimātratvāc chrāvakādivat /
utpattiṃ vinā bhāvitatvāt tu sthiter utpattir apyatra bhavati /
tasmād bodhisthitikāraṇam evocyate notpattikāraṇaṃ /
na cā'nyat tad vyatirekeṇa sthitikāraṇam astītyata etad evoktaṃ /
viśuddhatathatā vā bodhiḥ /
na tathātāyāḥ svabhāvavuśuddhātvād utpattir asti 
kintarhyāgantukamlā'pagamād anyantam ebhiḥ kāraṇaiḥ sthitirevocyate 
notpattiḥ /
sthitikāraṇāni punar bodheḥ kuśalotpādo manasikāraḥ paripūrṇasambhāraś 
ca /
atra ca śubhānutpattyādayaḥ kuśalotpatyādivibandhe vartamānā bodheḥ 
sthitiṃ kurvantīti sthityāvaraṇam uktāḥ /
tadyathā catvāra āhārāḥ sattvānām ityatraiva nidarśanaṃ /
na hi kavaḍākārādibhir āhārair apūrvāḥ sattvā utpadyante 
purvotpannās tu vyavasthāpyanta eva /
dhṛtikāraṇam iti /
dhṛtir bodhicittaṃ sarvakuśaladharmādhārabhūtatvāt /
atraiva nidarśanaṃ yadyasyādhārabhūtan tatra tasya dhṛtiḥ 
dhāryate'neneti kṛtvā /
tadyathā bhājanalokaḥ sattvalokasyeti /
tat kāraṇaṃ gotraṃ kalyāṇamitrasampannatā cittā'parikhedatā ca /
tatra dhṛtikāraṇaviruddhatvād gotravaidhuryādikandhṛter apyāvaraṇam 
uktaṃ /
etad evāha - dhṛtyāvaraṇaṃ samādāna āvaraṇaṃ 
bodhicittasyādhārabhūtatvād iti /
abhivyaktikāraṇaṃ dhīmattve bodhisattvasya prakāśanīyatvāt /
tacca pratipattisampat sujanavāso duṣṭajanā'vāsaś ca /
atra dhīmattve pratipattivaidhuryād ayo'bhivyaktyāvaraṇatvenocyante /
ata evāha - abhivyaktyāvaraṇaṃ dhīmattve tasya prakāśanīyatvād iti /
tad yathā - āloko rūpasyetyatraiva nidarśanaṃ /
āloko hi rūpamātrasyā'bhivyaktikāraṇam eva notpattyādikāraṇaṃ /
vikāraṇam abhrāntau /
samāropā'pavādabhrāntiparivṛttyātmakatvād abhrāntir 
darśanamārgo'bhrānter vikāra ucyate /
tadyathā bhasmādayaḥ kāṣṭhādīnāṃ /
tasya trīṇi kāraṇāni - viparyāsadauṣṭhulyopaghātaḥ 
kleśādyāvaraṇānāṃ trayāṇām abhāvo
