kāraṇena svabhāvas trividho mūlatattvam ity ucyate tat pradarśayann āha 
- tatrā'nyatattvavyavasthāpanād iti /
tatrā'nyasya lakṣaṇāditattvasyā'ntargatatvam ityarthaḥ /
kim arthaṃ svabhāvas trividha upadhāryate ?
vyavahāraparamārthatad āśrayapradarśanārtham ity eke /
viparyāsa-tannimitta-tatpratipakṣālambana-pradarśanārtham ityanye /
bodhisattvānām āvaraṇavisaṃyogārthaṃ prahāṇaparijñāsākṣāt 
kriyāvastupradarśanārtham ityanye /
prakṛtyā gambhīrasya prajñāpāramitārūpasya 
trisvabhāvadvāreṇā'bhrāntipratipattyartham ityanye /
evaṃ hy uktam abhidharmasutragrāthādvaye -
	māyādideśanāṃ bhūte kalpitān nāsti deśanā /
	caturvidhaviśuddhes tu pariniṣpannadeśanā //

	śuddhiḥprakṛtivaimalyaṃ nārgatālambanan tathā /
	viśuddhānāṃ hi dharmāṇāṃ caturdhā grahaṇam mataṃ //
iti /
laukikalokottaratatpṛṣṭhalabdhajñānaviṣayatvapradarśanārthaṃ 
yathāsambhavaṃ trayopādānaṃ svabhāvatam ity anye /
kim atra svabhāvatraye tattvam iṣyata iti /
aviparītārtho hi tattvārthaḥ /
kimatra svabhāvatraye'viparītatvam iti yasmāt svabhāvatraye tattvam 
iṣyata iti pṛcchābhiprāyaḥ /
	svabhāvas trividho'sacca nityaṃ
ityādi vistaraḥ /
parikalpitalakṣaṇaṃ nityam asad iti yatra hi bālānāṃ 
grāhyagrāhakavācyavācakābhiniveśaḥ svapna iva pravartate /
asacca sasvarūpābhāvāt kalpitasvabhāvaḥ /
tasmād asadātmakatvād abhrāntitvāc ca parikalpitalakṣaṇam asattvaṃ /
etat parikalpitasvabhāve'sattvam eva tattvam ucyate /
aviparītatvād iti aviparītārtho'tra tattvārthaṃ iti darśayati /
paratantralakṣaṇam sacca na ca tattvata iti /
grāhyagrāhakādikalpitavyavahārāśrayatvāt sattvaṃ /
grāhyagrāhakādyātmakatvenāsattvān na tathā tattvata iti /
etad eva pratipādayann āha bhrāntimātratvād iti /
etad uktaṃ bhavati - yat khyāti māyopādānavat tad asti /
yathā khyāti grāhyagrāhakākāreṇa naitad eva māyāpuruṣādivat /
ato bhrāntir iti /
ityetat paratantrasvabhāve tattvaṃ anyathā vidyamānasya svātmany 
avidyamānenākāreṇa prakhyānaṃ /
atrāpyaviparītatvād iti prakṛtatvāt sambadhyate /
