pratipaditi vistareṇa yāvad dvitīyapradīpodāharaṇaparyantaḥ paṭhyate /
kasyāntasya varjanādasau veditavyā ?
katam asyāntasya varjanān madhyamā pratipatter jñātavyetyarthaḥ /
ata āha -
	pṛthaktvaikatvam antaś ca tīrthyaśrāvakayor api //
iti vistaraḥ /
ayaṃ pañcadaśavidho'ntadvayabhedaḥ /
asya varjanāt sā veditavyā /
atra pṛthagādidarśanamantaḥ /
ekaikapakṣābhiniviṣṭatvāt kārye kāraṇopacāraṃ kṛtvā /
tatra rūpādibhyaḥ pṛthaktvam ātmana ityantaḥ ekatvam ityantaḥ /
tatra ye rūpādīn sknadhānāyatanāni dhātūṃś ca nityātmana 
upabhogavastutvena kalpayanti tad rūpādibhyaḥ pṛthaktvamātmana ityantaḥ /
ye punar ādhyātmikān saṃskārān rūpādīnām ātmatvena pratipannā yaḥ 
paśyati śṛṇoti jānāti sa ātmeti bāhyāṃścātmīyatvena kalpayanti tad 
rūpādibhya ekatvam ātmana ityantaḥ yat sūtryate - sa jīvas taccharīram 
anyo jīvo'nyaccharīram iti darśanaṃ tadevātrāntaśabdenoktaṃ /
tat parivarjanārtham iti rūpādibhyas tattvānyatvāntaparivarjanārtham 
ātmanaḥ /
madhyamā pratipad yad dharmātraṃ viṣayabhūtaṃ jñānaṃ /
etad eva pradarśayann āha - yā nātmapratyavekṣā 
yāvannamānapratyavekṣeti /dharmamātradarśanāddhi vandhyāpurtravad 
ātmā nāstītyavagacchan nātmānaṃ pratyavekṣate nānusaratītyarthaḥ /
kuta eva tasya rūpādibhyas tattvam anyatvaṃ vā gṛhyata iti ?
ātmaśabdenaiva tadarthe'vagamyamāne kimarthaṃ 
sattvādiparyāyaviśeṣābhidhānaṃ ?
na sarvasya sarvaparyāyavyutpattir ityataḥ sarveṣāṃ tadarthāvabodhārthaṃ 
paryāyāntaranirdeśaḥ /
tat kālavikṣiptānāṃ vā paryāyāntareṇa tadarthaprāpaṇārthaḥ /
tenaivābhidhānenocyamāne'nyeṣāṃ prāpaṇāt /
nityaṃ rūpam iti tīrthikāntaḥ /
te hi parikalpitarūpān vastutvenābhiniveśya nityatvena kalpayanti 
nityaviparyāsopahatatvāt /
anityam iti śrāvakāntaḥ /
tair api kalpitasvabhāvānāṃ rūpādīnāṃ vasturūpeṇānityatvena 
samāropaṇādantatvaṃ /
atyantasaṃsārodvegakarān nirvāṇasya cāntapātanād 
anityadarśnasyāntatvaṃ /tad ubhayaparivarjanārtham iti 
nityānityadarśanaparivarjanārthaṃ madhyamā pratipad yā rūpādīnāṃ 
nanityapratyavekṣā nānityapratyavekṣeti /
tat punaḥ kiṃ /
nirvikalpajñānaṃ /
yan na rūpādīn rūpādiṣv eva nityatvenānityatvena cānugacchati /
rūpādīnām ityādiśabdena 
vedanā-saṃjñā-saṃskāra-vijñāna-skandhānāṃ 
pṛthivy-ap-tejo-vāyv-ākāśa-dhātūnāṃ 
cakṣuḥ-śrotra-ghrāṇa-jihvā-kāya-mana-āyatanānāṃ ca grahaṇaṃ /
