ekeṣām iti yeṣāṃ pāramitānāṃ nāmamātram api na vyutpannaṃ teṣāṃ tan 
nāmavyutpādanārtham ucyate -
	dānaṃ śīlaṃ kṣaṃā vīryaṃ dhyānaṃ prajñā upāyatā /
	praṇidānaṃ balaṃ jñānam etāḥ pāramitā daśa //
iti /
adhikāraparamatā sarvatattvārthakriyoddeśādityuktam ato'yaṃ praśnaḥ - 
kimāsāṃ pratyekaṃ karmeti /
ata āha -
	anugraho'vidhātaś ca karma tasya ca marṣaṇaṃ /
iti vistaraḥ /
dānena hi bodhisattvaḥ sattvānanugṛhṇātīti /
tatra pūjānugrahakā:nkṣayā bāhyātmikavastuni avadyatvāt 
pratyupakāravipākādinirapekṣo yayā cetanayā parityajati 
kāya-vāk-karmāṇi ca pratipādayatīdaṃ bodhisattvasyāmiṣadānaṃ /
anena kṣutpipāsādiduḥkhāpanayanāt sattvānānugṛhṇāti /
sattvānupaghātakānām ājīvaśāstrakalānipuṇānāṃ karuṇāpūrvakam 
upadeśaḥ sugatimokṣamārgopadeśaś ca dharmadānaṃ /
anenā'pyajīvikādurgatisaṃsāraduḥkhāpanayanāt sattvānanugṛhṇāti /
rājacauradāyādavyālādibhir āghrātānāṃ tebhyo vimokṣaṇam 
abhayadānaṃ /
anenāpi jīvabhogaputradārādivipadbhayāpanayanāt sattvānanugṛhṇātīti 
/
śīlenopaghātam eṣāṃ na karoti /
sarvakāyavāgduścaritaviratir lokānuvṛttyā ca pratiṣiddhād vihitasya ca 
kāya-vāk-karmaṇo viratir anuṣṭhānañca śīlaṃ /
asmin śīle vyavasthānāt sattveṣvapakāraṃ duṣṭopasaṃhāraṃ jīvitahetor 
api svayaṃ na karoti nāpi kārayatīti /
kṣāntyā paraiḥ kṛtam upaghātaṃ marṣayatīti /
karuṇāpuraḥsareṇa cetasā sarvākāra-duḥkhamarṣaṇādhivāsanātmikā 
kṣāntiḥ /
tenānvair hiṃsābandhanatāḍanādikṛtam upaghātaṃ marṣayati 
pratyapakārāsamīhanāt /
vīryeṇa guṇān vardhayati /
dānādīnadhikṛtyotsāho vīryaṃ /
tena dānādiguṇābhyāsād viśiṣṭaguṇaniṣpattir vṛddhiḥ /
dhyānena ṛddhyādibhir āvarjyāvatārayatīti /
ādiśabdena cittaparyāyajñānaṃ
